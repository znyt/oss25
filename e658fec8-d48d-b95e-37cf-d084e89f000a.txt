　　第113章
<br/>　　出来混要讲装备的，名将的装备是白马银枪，侠客的装备是铁剑长发，书生才子的造型当是白袍、折扇、玉佩、贴身铺纸研墨的小童一个都不能少。
<br/>　　当然，装备够了颜值也得在线，不然装备置办得再齐全，也是薛蟠而不是贾宝玉。
<br/>　　两方面来讲，张肥和高飞豹都不及格。
<br/>　　唐如才一眼就看出来，这两个是出苦力的下等人，也不知道他们怎么混进渔阳楼。肯定是听到二楼曲声响，所以上楼想偷看唱曲的姑娘。
<br/>　　意识到这点后，唐如此当时就想把这两个人赶出去。不过转念间，他又冒出个主意……
<br/>　　如今酒也喝过，曲也听罢，兴致差不多快要散了，倒不如在这两个人身上做做游戏。
<br/>　　“你们也会作诗？”
<br/>　　高飞豹当时便挺起胸膛：“略懂而已。”
<br/>　　咯咯！
<br/>　　那身着彩衣，刚才舞动腰肢的婉儿小姐捂嘴轻笑。唐如才强忍住笑，挥挥手道：“来来，笔墨伺候，让我们也膜拜膜拜二位的大作。”
<br/>　　周围七八人是与唐如才交好的同伴，都以戏谑的眼神盯着张肥和高飞豹。听口音便知是北国人，当今天下的文章一在京城，一在江南，京城是帝国首府自不必说，江南是繁华之地，诗书文章盛行。至于北国……那里向来是不通教化的蛮荒之地，据说那里的人，如今还茹毛饮血呢。
<br/>　　这唐如才也是机灵，怎么就想到拿这二人做戏，当真是为今日添了不少酒兴。
<br/>　　“来，我与二位磨墨，膜拜二位文采。”
<br/>　　“或是诗，或是词做上一首，由婉儿之口唱出，明日定轰动全城。”
<br/>　　如今的诗词传播，主要还是通过青楼歌姬之口，青楼承载很多社交作用。而文人才子雅集，既不可没有酒，也不可没有歌姬助兴。而今日，唐如才等人还是戏耍张肥二人的意思居多，一众人嬉嬉笑笑，那婉儿轻啐一口，香唇中吐出『讨厌』二字。
<br/>　　如此换来更多的笑声，那年轻人饮一口酒，要去吃婉儿口上的胭脂。
<br/>　　张肥却是冷汗哗哗落，他是楞，但不代表他傻。如今这宴会是什么滋味，他怎会品不出来。
<br/>　　这帮人分明就是将二人当猴耍！
<br/>　　此时就该笔走龙蛇，写出一篇震惊四座的诗，狠狠打一下他们的脸。
<br/>　　奈何，张三爷习武的本事不错，在文艺方面虽然有爱好，却委实不擅长。
<br/>　　“诸位，是我们莽撞了，打扰了诸位的雅兴，唐突之处还请诸位海涵。”
<br/>　　场面话一说，张肥就想走。谁知唐如才冷笑一声，道：“想走么……你们鬼鬼祟祟，我看是贼吧。”
<br/>　　“若是写不出来，就拿他见官。”
<br/>　　“唐公子往城主府递张条子，就能要你二人的姓名。”
<br/>　　唐如才是千舟城唐家的人，家里是开绸缎庄的，其叔父在城主府任职。相比幽州，繁华的扬州各样官职更齐全些，权力也更大。
<br/>　　张肥和高飞豹也没想到随便逛逛，就惹到城里的大人物，此刻他心中已有三分火气，换作其他人，早就钵子大的拳头擂上去。奈何今日张三爷还是以文人自居，不想动武，失了自己『文人』的体面。
<br/>　　“我们两个才疏学浅做不出来，不过我家公子要在的话，他肯定能做得出来。”
<br/>　　“你家公子……该不会是想溜吧？”唐如才。
<br/>　　“你们如果不信，就让我把我家公子请来……”
<br/>　　“去一个人，我倒是不相信，北蛮子里面还有会做诗的……”
<br/>　　此刻程大雷正在一楼大厅闲逛，高飞豹隔着楼梯张望一眼，立刻喊道：“大……公子，公子，快过来，出事了。”
<br/>　　“出什么事了？”程大雷和柳芷苏樱一起，信步走上二楼。
<br/>　　“他们不让我们走。”
<br/>　　“咦……谁们？”程大雷抬起头扫了一眼。
<br/>　　不知怎地，当程大雷的目光扫过身上的时候，唐如才有种被毒蛇盯上的感觉，激灵灵打个寒颤。
<br/>　　张肥简单把事情说了，指了指唐如才。
<br/>　　“喔，原来是作诗喔。”
<br/>　　不知怎地，张肥隐隐觉得，程大雷的语气有些兴奋的意思。
<br/>　　“今日爷就赏你们个脸，给你们个颜色瞧瞧。”
<br/>　　程大雷执笔在手，心潮澎湃，穿越者必做清单，有一条一定是抄诗，如果不能抄，唐诗宋词元曲……自己九年义务教育岂不是白背了。只是幽州荒僻，不通文墨，自己一肚子诗才也没有显摆的机会。
<br/>　　扬州可是诗词之乡呐，自己随便拿出来一首，不就震着这帮人瞠目结舌。
<br/>　　唐如才几人盯着程大雷，早已打探出来他的来历，北地来做生意的商人。怪不得一身铜臭味，穿着像个暴发户，凭这种人会有什么诗才。
<br/>　　“阁下倒是写不写？”见程大雷一直握着笔，唐如才不耐烦问道。
<br/>　　“容我想想，想想。”
<br/>　　“呵！不会和你的手下一样想不出来吧？”唐如才冷笑。
<br/>　　程大雷不是想不出来，而是不知道该抄哪一首，没办法，肚子里存货太多，偏偏程大雷还有些选择困难症。
<br/>　　罢了，就是它了。
<br/>　　程大雷饱蘸浓墨，笔走龙蛇，在纸上写下四个字。
<br/>　　唐如才一见，忍不住笑出声来。其他几人也捂着嘴，尤其是那婉儿，更是憋得满脸涨红。
<br/>　　无他，主要是因为程大雷字太丑了。他前生没用过毛笔，来到这个世界后也没有刻意练过，在程大雷的认知中，字够用就行。这在青牛山都算好的，青牛山不识字的大有人在，至少程大雷会写。当然，到扬州这种诗书之乡，程大雷这首字就成了被嘲笑的对象。
<br/>　　字写成这样，会有什么好诗才。
<br/>　　有人笑得前仰后合，唐如才强忍住笑：“阁下的字没有先生教过吧？”
<br/>　　“是啊，怎么了？”程大雷抬起头，困惑的望着他，手中笔不停，写下一串鬼画符般的字迹。
<br/>　　唐如才视线落在纸上，本想奚落对方几句，但在看清楚笔下的内容时，顿时睁大了眼睛。
<br/>　　歪三扭四，如同蛇在纸上胡乱爬过，出现一串墨字。
<br/>　　『北国风光，千里冰封，万里雪飘』
<br/>　　
<br/>(本章完)
