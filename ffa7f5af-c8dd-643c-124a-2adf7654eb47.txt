　　“老衲明白。”苦行大师微笑着打断他的话，“这宝剑在你手中，或能拯救苍生，你安心拿去便是。”
<br/>　　叶少阳拱手道谢，从苦行大师手中接过太阿剑，突然想起一个问题，试探问道：“大师，恕我直言啊，天台宗这些年在法术界没啥存在感，大师为什么没想过把太阿剑拿出来自己用，或者给门中弟子，这样说不定就有机会重振山门。”
<br/>　　“老衲不是没想过，甚至老衲的师父，和前面几位师祖，都想过这个，但是一来我山门远在边陲，没有争雄之心，二来……将太阿剑据为己有，一时间的确可以吸引各派的目光，但随之而来的，很可能就是厄运了，甚而祸患无穷。我天台宗而今虽然人少，但师徒诸人每日修佛，倒也自得其乐，放着太平日子不过，何苦去找麻烦呢？”
<br/>　　匹夫无罪怀璧其罪。
<br/>　　叶少阳想到了这句话，苦行大师说的不是没道理，但就算这样，但人的贪念是难以遏制的，放着太阿剑在山门中却用不上，不是所有人都能坚守得住。
<br/>　　光凭这一点，这位苦行大师就值得点赞。
<br/>　　“那个，大师，之前为了取剑，把药师佛弄坏了一部分，你看……”那可是千万级的文物，叶少阳说的小心翼翼。
<br/>　　苦行大师叫来一个和尚，让他去请谁谁过来，趁着晚上没人把药师佛修好，随即对叶少阳道：“不必担心，这干漆夹苎”的手艺传承，就在我们寺里，半个晚上足够修补了。
<br/>　　叶少阳简直感激涕零。
<br/>　　喝了一壶茶，道风坐不住了，起身告辞，苦行大师跟着站起来，问道：“你们既然拿到了五把剑，老衲想问，你们是否知道如何使用？”
<br/>　　叶少阳看看道风，有点傻眼，这问题……自己还真没想过，关于什么五把宝剑的说法，还是道风一而再地跟自己说的，本以为他都知道，但是看他现在的表情，才知道他好像并不知晓。
<br/>　　“大师知道？”道风问道。
<br/>　　苦行大师让他们稍等，自己起身出去，过了一会，拿来一本线装书，当着二人的面翻开，叶少阳大致扫了下内容，是佛门的金刚经，心中疑惑，只见苦行大师把经书翻到最后一页，只见上面用潦草的字迹写着一段话，开篇便是：五剑连璧，当以此法汇聚人间五行之气，可诛邪魔……
<br/>　　叶少阳心中一动，跟道风一起读了下去。
<br/>　　“这是当年那位祖师留下的手抄本《金刚经》，本来这一页纸是夹在书页中的，因是祖师遗物，由历代方丈保管，后为了保存，重新装裱，这一页纸长才被发现，后来一直夹在这经书里保存下来，今天总算把东西都交出去了。”
<br/>　　苦行大师轻松的语气中透着一丝落寞。
<br/>　　叶少阳朝道风看去，发现道风也在看着自己。
<br/>　　两人都怔怔无言。
<br/>　　苦行大师知道这张纸上写了什么，也能理解他们的性情，双手合十，宣了一声佛号，叹道：“五剑连璧，也未必是那太阴山鬼王的对手，办法在这里，如何取舍，看你们自己了。”
<br/>　　道风将这张纸取下来，塞到叶少阳背包里，望着苦行大师，十分认真地说道：“你想要什么？”
<br/>　　“要什么？”
<br/>　　“我不能白拿你的东西，你想要什么，只要能办到，我都可以给你。”
<br/>　　苦行大师有点吃惊地看着他，这个传说中的人物，他能给得起的东西，自然非同凡响。
<br/>　　九段光的法器？人间最好的功法？
<br/>　　苦行大师让他等等，自己去了隔壁，很块又回来，手里捧着个笔记本，递给道风。
<br/>　　“如果方便的话，签个名吧，老衲门下弟子，可有不少都是你的粉丝。”
<br/>　　道风愕然。
<br/>　　这么好的机会……他居然要签名。
<br/>　　“大师可想清楚，我这一走，你可就找不到我了。”
<br/>　　苦行大师笑了笑，“老衲土埋半截的人了，还缺什么，什么不缺，你来一次，好歹留个签名，让弟子们欢喜一场，也算我这个做师父的有面子。”
<br/>　　叶少阳心中震骇，他有点明白这苦行大师为啥能守着太阿剑而不动念头了，他是真的风轻云淡，这样的人不管实力怎样，都是真正的宗师。
<br/>　　“一个签名，是有点少……”
<br/>　　苦行大师摸了摸脑门，“你要是不怕累，给我签十个二十个吧，我分下去，哈哈，有劳有劳。”苦行大师似乎在装糊涂。
<br/>　　道风接过笔，在笔记本上签了二十个名字……然后把笔记本还给苦行大师，说道：“别的我也没本事做什么，将来天台山若有危难，只管找我。”
<br/>　　说完想到什么，补充道：“我要是死了，就找我师弟，我们要都死了，找他儿子也一样。”
<br/>　　“我哪来的儿子！”叶少阳反对。
<br/>　　苦行大师双手合十，庄重地施了一礼。
<br/>　　道风转身就走，叶少阳跟出去，刚出门就看到外面站着十数个僧人，正在讨论什么，看到他们出来，立刻笔直站好，一言不发了，不少都表现出激动的样子，跟粉丝见到偶像似的，一直追他们到山门外。
<br/>　　两人乘着夜色下了山。
<br/>　　“你是怎么想的？喂，你是不是已经决定了？”
<br/>　　叶少阳追上道风，挡在他前面，逼他回答。
<br/>　　“你呢？”道风淡淡问道。
<br/>　　“果断不可以啊！自爆元神，那连鬼都没得做，这算什么手段，简直就是自杀！”
<br/>　　道风沉默了一会，说出三个字：“我替你。”
<br/>　　叶少阳怔住。“得了吧，我还真不怕，但他们四个呢，让人家主动献身？”
<br/>　　“这是他们的使命。”
<br/>　　“那你问过他们本人没有？”叶少阳据理力争，“就算是为了拯救全世界而牺牲，那也得本人同意吧？”
<br/>　　道风淡淡一笑，“别管别人，先问你自己，你愿意吗？”
<br/>　　“我……”
<br/>　　叶少阳犹豫了，这还不是一般意义的死亡，自爆元神，连鬼都做不成……“我想，一定还有别的办法，未必要这么做。”
<br/>　　出乎意料的是，道风竟然点了点头，道：“这件事先别跟任何人说，我回去想想。”
