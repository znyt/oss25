　　叶少阳浑身一颤，包子在喉咙口噎住了。
<br/>　　南京……不会……真的这么巧合的！
<br/>　　没错！！
<br/>　　就是同一家店，一样的配方，一样的味道！自己不会吃错的，这家店，就是谢雨晴经常带自己去吃的那家店的祖先开的！百年老店，果然是真的跨越百年……
<br/>　　没想到自己在百年之前，居然吃到了一样的包子。
<br/>　　想起自己跟谢雨晴以往吃大肉包子的情景。没有任何预备的，叶少阳眼泪哗的一下就下来了，急忙把头低下去，趁着翠云没看见，把眼泪抹掉了。
<br/>　　他不是爱哭的人，甚至一点都不爱哭，但是在百年前吃到自己曾经吃过的同一家店的食物，还是经常跟谢雨晴一起吃的店……想着自己一个人孤零零的在这个世界，叶少阳思想情绪爆发，不免有了一种物是人非的感觉。
<br/>　　不过仔细一想，物是人非这个词也不合适，那一般是形容某些人失去了之后，而自己现在的情况，却是来到了百年之前，这时候谢雨晴的爷爷奶奶估计都还没出生呢。
<br/>　　这种感觉，叶少**本找不到词语来形容。
<br/>　　“老板，你一定要去南京开店，就冲你这包子的口感，大城市的人绝对喜欢吃。没准可以一直开下去，开上一百年也不成问题。”
<br/>　　老板收获这么高的评价，也是非常高兴，哈哈大笑起来，“小兄弟，冲你这句话，我一定好好考虑这件事！”
<br/>　　“还有，你应该给你家包子铺取个名字，响亮一点的，这叫品牌效应。”
<br/>　　“品……啥子？”
<br/>　　叶少阳这才想到百年前肯定没这个词，于是解释道：“就是说，别人吃着好吃，记着你家店的名字，之后一旦吃包子，就想起你家店来。”
<br/>　　老板恍然大悟，“哦对对对，你说的没错！不过……我一个大老粗，可取不好什么讲究的名字……”
<br/>　　叶少阳道：“这不简单吗，你这卖的就是大肉包子，我看就叫大肉包子好了，简单明了，别人一听就能想起你这包子的味道。”
<br/>　　“好啊！大肉包子！这名字好，嘿嘿！那就叫这个了，我明天就去做个招牌，多谢这位先生，您真是我贵人！”老板眉开眼笑，对叶少阳直拱手。多送了叶少阳两笼包子，也没收钱。
<br/>　　吃完之后，叶少阳感慨良深地离去，回头看了一眼包子铺，突然间，脑海中灵光一现地想到了什么，“大肉包子”这四个字，是自己取的，看老板开心的样子，应该不是敷衍自己，如果只是为了讨客人喜欢，也没必要白请自己一顿，也就是说，他是真的喜欢这四个人，一定会去用，那么百年之后，自己和谢雨晴去吃的包子铺，这名字其实是自己取的……
<br/>　　难道说，自己影响了历史？
<br/>　　换句话说，当初自己跟谢雨晴去吃包子的时候，自己并不知道这一切，但当时那家包子铺已经叫这个名字了……当时的名字，又是谁给取的？
<br/>　　类似的问题，叶少阳之前其实已经想过，只不过当时没有这么极端的例子来给自己思考。
<br/>　　叶少阳让翠云等一会，自己站在路边想了半天，结果越想越乱，最后也只能放弃了，不过他猛然又想到一件事，假如，这个包子铺的名字真的是自己取的，并且流传到了百年之后，对自己那个时代发生了影响，那么，别的东西是不是也可以流传下去，给百年后的人看？
<br/>　　甚至说，自己在这个世界做出的改变，自己原本的那个时代，也会受到影响和改变？
<br/>　　想到这里，叶少阳决定做一个尝试，他把背包翻出来，倒出来一些东西，找到了一个笔记本和圆珠笔，这是他一直装在背包里的东西，也跟着一起穿越了。
<br/>　　叶少阳拿出笔，想了想，开始在笔记本上写字。翠云在旁边看着，对圆珠笔感到十分惊奇，在一旁说道：“着东西哪里来的啊，在上海买的吗。”
<br/>　　“对啊。”叶少阳随便答应了一声。
<br/>　　翠云见他认真写字，也就不理他了。
<br/>　　叶少阳写了一会，想了想又觉得不合适，于是找出一张灵符，在上面随便画了个茅山内门的符文，然后在背面写字：我是少阳，我回到1922年了，我找不到徐福，我不知道怎么才能回去，我在努力找他，如果你们能看到这封信，说明两个世界是有关联的，但你们应该无法联系我。
<br/>　　道风，如果有办法，你去帮我救冷玉，如果没有机会就算了，我会想办法尽快回去，亲自救她。大家保重，做自己的事，不要牵挂我。但你们也联系不到我，我也不知道你们能否看到，所以不会再写更多。你们放心就好了。
<br/>　　我很好，我不会死，我一定会回去！
<br/>　　叶少阳心中有千言万语，但是想了想，只要报个平安就好了。就算是这些话，一张符纸也写不下，叶少阳用了五张灵符写完，然后卷起来，想了想，九十年的时间，可能纸张都风化了，于是找出了一个鸡血石的挂坠，里面是中空的，叶少阳把几张灵符卷在一起，然后塞了进去。
<br/>　　叶少阳让翠云在这里等着，自己又去了包子铺，找到老板，说道：“老板，我想请你帮个忙。”
<br/>　　“先生有什么请说。”
<br/>　　叶少阳把鸡血石挂坠交给他，说道：“我想让你帮我保存一个东西。”
<br/>　　老板接到手中，说道：“行，多久来拿？”
<br/>　　“九十多年之后。”
<br/>　　老板以为自己听错了，问道：“多久？”
<br/>　　叶少阳又重复了一遍：“九十多年之后，交给一个人。”
<br/>　　说着拿出张道长给的钱袋，摸了一把有十几个大洋给他。
<br/>　　老板一看，吓了一跳，急忙摆手说道：“这位先生，我不明白你的意思。”
<br/>　　“你不需要明白。”叶少阳望着他，神色十分恳切，“你放心，我不是疯子，这个东西相反也不会给你带去恶任何麻烦，只求你把他收起来，九十多年之后，把他叫给一个人就行，我把地址写给你。”
<br/>(本章完)
