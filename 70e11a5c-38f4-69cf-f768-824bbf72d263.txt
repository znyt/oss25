　　第180章
<br/>　　林冲灼灼目光盯着程大雷，令程大雷觉得有些不自然。无奈，他抬起手，摇摇指向远方。
<br/>　　“在那里。”
<br/>　　林冲顺着程大雷手指的方向看去，看的是青牛山最高峰。峰顶一棵杨树，却看不到有人影。
<br/>　　林冲面露困惑。
<br/>　　“他就在牛角峰上，以大树为信号，指明戎族的进军方向，这样戎族只要有风吹草动，我们就可以提前知晓。”
<br/>　　林冲怔了怔，忽然咆哮：“混蛋，你是要他去送死。”
<br/>　　程大雷吓了一跳：“也没有那么危险啦，你刚才不就是没发现他么。峰顶有大石挡着，很难看到他的，再说，也没人会成天往天上看。”
<br/>　　林冲道：“一人在外，不被发现还好，但凡被发现就是死路一条。混蛋，既然不危险，你怎么不去送死！”
<br/>　　程大雷愣了愣，看来这林教头不仅是个苦逼的英雄，而且脑袋还有点不清楚啊。所以说性格决定命运呐，他悲剧的一生，也是这悲催的性格注定。
<br/>　　“山河破碎，万民流离失所，这片大地每天都在死人。蛤蟆寨一旦被攻破，五千人命不过是人的刀下之鬼。我会死，林教头也会死，少羽……也会死。你我皆别无选择。”
<br/>　　林冲怔了很久，良久突出一句话：“我不是林教头啊。”
<br/>　　这个人为何一直唤自己林教头，该不会脑袋有什么问题吧。
<br/>　　林冲摇摇头，目光看向远处的牛角峰，忽然道：“动了，动了，杨树动了。”
<br/>　　“又来！”
<br/>　　程大雷一激灵，挥着剑大喊：“上城，上城，戎族又打过来了！”
<br/>　　趁着黄昏日落，戎族大举进攻，这次他们发动的力量比任何一次都要强大。多少戎族勇士视死如归，冒死冲锋，迎接他们的是弩箭，火油，圆石……几次城头险有失守的危险。
<br/>　　林教头展现了他顶级人物的实力，他、秦蛮、赵子龙各率一队人守在城头，三柄枪齐动，舞出一支战场杀戮之舞。
<br/>　　大战持续了一天一夜，戎族无功而返。
<br/>　　这次攻打，赤眉悲催的发现，久守之下，蛤蟆寨不仅没露出半点疲态，反而更勇更猛。眼看着投石机砸在城墙上，没过多时发现城墙竟然完好如初。而且他们的箭术也明显提升不少，以前蛤蟆寨在城头射箭，完全是拼运气，能不能射中全靠赌脸。可这次，他们的箭术不说百步穿杨，至少十箭有五六箭是能够射中的。
<br/>　　在接下来的日子，赤眉又发动了十数次攻城，牺牲无数，但仍没有破开城门。
<br/>　　蛤蟆寨就像一颗顽固的钉子，结结实实的扎在青牛山，任你用锤子、钳子，他都牢牢不可破。
<br/>　　屈指算来，程大雷已坚守十七日，城墙未破。
<br/>　　青牛山，除了赤眉和蛤蟆寨两家实力外，还有大大小小许多势力派来的斥候，他们隐蔽着身形，躲在草丛中、大树上、山石后，眼睛盯着青牛山的一举一动，又随时把情报汇报给自己背后的主人。
<br/>　　“蛤蟆寨被攻破没有？”
<br/>　　“没有。”
<br/>　　“再查再探。”
<br/>　　“报，戎族大军第十一次攻打蛤蟆寨！”
<br/>　　“情况如何，蛤蟆寨破了？”
<br/>　　“没有，他们守住了。”
<br/>　　“又守住了，他们究竟什么时候被攻破。”
<br/>　　在幽州，乃至临着幽州的青州冀州，这一幕在诸多个城池内发生。
<br/>　　……
<br/>　　青州，天一城。
<br/>　　“城主，据逃到码头的人所说，戎族尚未突破青牛山。”乔野。
<br/>　　“还没被打破，程大雷那帮山贼竟真的守住了。”方伯山。
<br/>　　最近逃到青州的人越来越少，这种情况只有两种可能，一是戎族已经大军杀来，开始屠城。程大雷直到现在还守在蛤蟆寨。
<br/>　　第一种情况可能性更大些，相比之下第二种就有些匪夷所思了。
<br/>　　“凭蛤蟆寨那点乌合之众，能挡得住一万精兵。”方伯山纳闷道。
<br/>　　在帝国，士兵强弱对比大概是这样的：京州保卫京城的禁卫兵强于各州牧兵，各州牧兵又强于各城城防军。
<br/>　　戎族骑兵，强过以上加起来。
<br/>　　以上这些，统统强过山贼。
<br/>　　山贼落草为寇，虽说田野埋麒麟，偶尔也会冒出几个猛人，但士兵整体素质完全不能和正规军相提并论。
<br/>　　而现在发生的情况是，在兵力极端劣势的情况下，最弱的山贼却挡住了最强的戎族铁骑
<br/>　　“城主，不若我们回去吧。”乔野道：“统领残兵，于蛤蟆寨里应外合攻击戎族，戎族腹背受敌，必定被我军击败，到时候城主可是大功一件。”
<br/>　　方伯山上三眼下三眼的打量着乔野，一副『你没疯吧』的表情。
<br/>　　“那可是戎族！”
<br/>　　乔野默然，垂着手退下。
<br/>　　心里却冒出一个想法：戎族怎么了。
<br/>　　戎族怎么了！
<br/>　　这个念头不仅在程大雷心中出现过，也在帝国无数少年将领的心底出现过。有人不信邪，要与戎族放手一战，问问戎族究竟怎么了。
<br/>　　结果往往是输多胜少，甚至说百战百败也并不夸张。
<br/>　　在无数场失败中，便有了戎族不可敌的说话。时间渐远，曾经的少年将领若没有死在战场上，便也该两鬓白发。
<br/>　　此刻再有少年要和戎族放手一战时，他都会愤怒的呵斥一句：那可是戎族！
<br/>　　而『戎族怎么了』这个问题，从来没有人给出答案。
<br/>　　乔野也曾年轻，他出身不好，军功也是一点点打出来的。当程大雷去落叶城借兵时，他体内的血也曾炙热过，但想起自己的小妾，想起自己刚出生不久的宝儿，最后那炙热的血又凉下来，选择和方伯山一起逃离落叶城。
<br/>　　今日，乔野比无数人都关注青牛山发生的事情，似乎是把某种自己无法完成，寄托在程大雷身上。
<br/>　　期待程大雷可以在那个问题下写出答案。
<br/>　　其实问题的答案所有人都知道，但自从三十年来，无人有资格在上面写下答案。
<br/>　　问题不复杂，答案也很简单，只需要寥寥几个字。
<br/>　　但却无人能在其下落笔。
<br/>　　因为这答案需要以精兵为笔，以鲜血为墨，以山河大地为纸张，以一场干净利落的胜利画上句号。
<br/>　　戎族不怎么。
