
<br/>　　<br/>　　    王惊蛰和茅小草对陈三岁的出手觉得非常惊艳，尸蟞是寻常人拿锤子都不一定能给凿开的，但却被他一脚一块石子就给崩死了，这人的身手有些太嚣张了点。<br/>　　<br/>　　    等陈三岁走了，王惊蛰蹲在地上用一根树枝扒拉了下被砸憋了的尸蟞，离得近了就能闻到那股尸味了。<br/>　　<br/>　　    “他既然都已经知道那女人尸体里连尸蟞都出现了，难道他还觉得人没死？这一点，明显有些说不通呢？”王惊蛰有点狐疑了，先前他一直以为对方的心里有些病态，或者潜意识里觉得女人没死，但现在看来明显不是这么回事了，尸蟞都从尸体里跑了出来还被他一脚给踢死了，他绝对不可能认为人是还活着的了。<br/>　　<br/>　　    茅小草低头看着地上的尸蟞，皱眉说道：“他是巫族的可能性非常大”<br/>　　<br/>　　    王惊蛰抬头看向了她，茅小草接着说道：“天道峰有记载，巫族中有一法术叫借尸还魂，难道他留着这女人的尸体是要·······”<br/>　　<br/>　　    王惊蛰顿时惊愕的站起来说道：“那不可能，这女人至少死了几年，魂早就进了阴曹地府，说不准都开始进入轮回投胎转世了，再说尸体虽然保存的还算完好，但她身上既然已经出现了尸蟞，那内脏肯定已经被啃食过了，就算是还了魂还能活得下去？”<br/>　　<br/>　　    “谁说魂魄进了阴间就一定会投胎的？阴曹地府里被压了多少亡魂，谁又能知道？”茅小草大眼睛里闪烁着精光说道：“谁又说，这男人一定非得要他老婆必须得像正常人一样生活的呢，如果他只是想要把女子的魂魄从阴曹地府弄出来，然后还魂到这具尸体上，并不是非得要人重新活过来，只需要在自己身边就可以了，这回你觉得可能么？”<br/>　　<br/>　　    王惊蛰张了张嘴，顿时也反应过来了，茅小草的意思是那女人的魂魄应该还在阴曹地府没有转世投胎，陈三岁一直留着这具尸体，就是想要等待什么机会将女人的魂魄从阴间带出来，然后封进这尸体里。<br/>　　<br/>　　    其实，这就跟鬼上身是一个道理，借尸还魂差不多也是一个意思。<br/>　　<br/>　　    就像东北的出马仙，就是最标准的鬼上身，出马的人从阴间借一道亡魂出来上自己的身，然后你想问什么，算什么，都由这阴魂来回答。<br/>　　<br/>　　    如果陈三岁的打算真是如此的话，那此人可就太跋扈了！<br/>　　<br/>　　    “这回好奇了吧？”茅小草瞅着王惊蛰问道。<br/>　　<br/>　　    “有点了”<br/>　　<br/>　　    “那我有个想法······”茅小草笑眯眯的说道。<br/>　　<br/>　　    王惊蛰道：“你笑的太贼了，不过我挺感兴趣的”<br/>　　<br/>　　    茅小草白了他一眼，说道：“一会他回来了，咱们以邻居的名义跟他打个招呼，从认识开始再到慢慢熟悉，然后逐渐瓦解掉他的提防心，不过如果他不搭理我们呢？”<br/>　　<br/>　　    王惊蛰叹了口气，指了指自己和茅小草：“首先你得搞清楚，你是什么人我是什么人，咱们都不是普通人，他未必看不出这一点来，所以他提防是正常的，不搭理也是正常的”<br/>　　<br/>　　    “姐们就善与人交际，沟通的事我来就行了·······”<br/>　　<br/>　　    两人在老城区里等了没多久，陈三岁推着轮椅就回来了，经过他俩身旁的时候眼睛依旧没有往旁边扫一眼，视若无睹。<br/>　　<br/>　　    茅小草靠着墙，忽然开口了：“听说良水成后山，有户人家的坟地被人给挖了，棺材也给撬开了，尸体都被人给偷走了，最近几天在良水这事传的到处都是，有人说是人贩子偷器官的，也有说是要结阴婚的，我的邻居啊你听说这件事了么？”<br/>　　<br/>　　    王惊蛰顿时一脸懵逼，捂着脸无奈的感叹了一句：“真他么的活见鬼了，这叫善于人交际？就这开场白，他一脚踢死咱俩都不多啊，这女人说话怎么不走心呢，你就不能循序渐进的套路他么？”<br/>　　<br/>　　    陈三岁推着轮椅顿时就停住了，就那么一瞬间王惊蛰和茅小草都感觉他身上蔓延出一股戾气出来，人虽然没有动，杀意肯定是有了，一点不夸张的说，搞不好在下一刻陈三岁就有杀人灭口的心思了。<br/>　　<br/>　　    茅小草靠着墙，抱着胳膊淡淡的说道：“千万不要着急杀人灭口，首先你也未必能动得了我们，俺们两个可不是那只蠢蠢的尸蟞，不知道反抗，你以一敌二的话胜算渺茫的很，我就是有一点挺好奇的·······良水城你再也偷不到尸体了的话，你拿什么来熬制尸油保证轮椅上那个女人的尸体长久不腐啊，我说的对么？”<br/>　　<br/>　　    陈三岁缓缓的转过身子，第一次用眼神正眼看着茅小草和王惊蛰，眼睛在他俩的身上盯了许久之后才开口：“你想要什么？”<br/>　　<br/>　　    茅小草说道：“我就是好奇，你留着尸体这么多年，也知道她体内生出了尸蟞，那就是你已经知道人确实死了，你还留着尸体到底是什么打算呢？莫不是，想要借尸还魂？”<br/>　　<br/>　　    陈三岁低着脑袋没有吭声，不知道在想些什么，但王惊蛰离他的距离有点近，角度和他斜对着，王惊蛰发现陈三岁握在轮椅上的手吐出了一道道的青筋，这是明显有要动手的心思了。<br/>　　<br/>　　    “想要为尸体防腐，并不一定要靠尸油也可以的······”王惊蛰忽然开口说道：“有很多比这稳妥的方式，比如据我所知，有两处养尸地就可以保证尸体常年不腐，甚至连尸蟞都不一定会生的出来，你这种法子有些老了”<br/>　　<br/>　　    陈三岁紧握着轮椅的手，悄然之间松懈了下来，他扭着脑袋说道：“你是说抚仙湖和闽南永安？据我所知，这两块地方现在已经不行了吧，永安养出来的尸体早些年都成僵尸了，抚仙湖水下的古城也都破败了，未必能够养尸了”<br/>　　<br/>　　    王惊蛰哦了一声，问道：“你了解过？”<br/>　　<br/>　　    陈三岁手摩挲着轮椅上的女尸，淡淡的说道：“不然我还会用这种老掉牙的法子？为了这件事，我已经东奔西走很多年了，最后我才来到了良水”
