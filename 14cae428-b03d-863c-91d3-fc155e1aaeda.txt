        歌唱得好，身材好，穿什么都好看，还有那么多男人愿意掏钱给她打赏，更是有两位大金主为了她“小谢”在“夜惑”里打群架。
<br/>
<br/>        啧啧嘴，这样的名气，“夜惑”营业这么多年，还从未有人出过她这样的彩头。
<br/>
<br/>        童筱颖在茶水间门口驻足，往里看了看。
<br/>
<br/>        里面有五个女服务员，一人端着一杯茶，站在柜台前谈笑风生。
<br/>
<br/>        其中四个女人看到她后，异口同声地唤了声：“小谢姐，好！”
<br/>
<br/>        童筱颖礼貌性地回礼微笑，正准备离开的时候，里面刚刚那个唤她的女孩，疾步走了出来，手里还端来着一杯姜茶。
<br/>
<br/>        “小谢姐，你刚唱完歌，来，喝一杯我自己调制的姜茶吧！”女孩双手捧着杯子，朝童筱颖递了过去。
<br/>
<br/>        另外四个女人也纷纷赞许地说道：“小梅调的姜茶，确实很好喝噢！”
<br/>
<br/>        “是啊！”
<br/>
<br/>        “小谢姐也尝尝吧！”
<br/>
<br/>        “嗯，真的很好喝！”
<br/>
<br/>        “谢谢！”童筱颖腼腆地笑了笑，接过杯子，发现水温不烫，便直接喝了一口，觉得味道不错后，还夸赞了小梅一番，“这姜茶好好喝，可不可以把配方告诉我？”
<br/>
<br/>        “当然可以，就是把最新鲜最嫩的姜切碎了，和芝麻混合在一起，装瓶子里保存，如果想喝的话，就舀一勺出来，在杯子里放点蜂蜜和红糖冲温水即可。”小梅一边耐着性子说，一边接着劝说童筱颖把姜茶全部喝完，“你快点儿喝完吧！喝完了，我好洗杯子。”
<br/>
<br/>        “好。”童筱颖捧着杯子一口气就把姜茶给全喝了。
<br/>
<br/>        小梅接过童筱颖手中的空杯子，转身回到了茶水间，开始收拾内务。
<br/>
<br/>        童筱颖没再和她们搭话，而是自顾自地去了自己的休息室。
<br/>
<br/>        回到休息室里，当她刚想换件衣服时，只觉自己脑袋昏沉。她扶着椅子坐下，摇了摇头，抬眸之际，眼前一阵天旋地转。
<br/>
<br/>        不知不觉中，童筱颖体力不支，晕倒在了梳妆台上。
<br/>
<br/>        再次醒过来的时候，童筱颖是被热醒的。
<br/>
<br/>        她不是觉得房间里热，而是觉得她自己的身体很热。
<br/>
<br/>        视野里全是热气腾腾的白雾，双目就像被烈火灼烧了一样，又热又痛。
<br/>
<br/>        童筱颖撑起身子坐起来，四下看了看，发现这里并不是她的休息室，而是一间类似于酒店里的单人套房。
<br/>
<br/>        “你醒了。”一个熟悉的男音飘忽入耳。
<br/>
<br/>        童筱颖循声望去，只见落地窗那边，有个男人背对着她。
<br/>
<br/>        那男人手里端着红酒，翘着二郎腿坐在了单人沙发上。
<br/>
<br/>        落地窗旁有台灯，但因为是逆光，童筱颖看不清那男人的脸，不过，但从身形上看，童筱颖觉得这个男人像季少——季运恒。
<br/>
<br/>        她脑海里刚这么一想，这男人再次开口说话的声音，印证了她的想法。
<br/>
<br/>        “小谢，跟我上床吧！”季运恒将手里高脚杯中的红酒一饮而尽，然后将空杯子放在了一旁的圆玻璃桌上，从单人沙发上站起身来。
<br/>
<br/>        童筱颖立即掀开被褥，想要从床上下去的时候，只觉脑袋昏沉，身体燥热到令双腿发软，连站都难以站起来。
<br/>
<br/>        “你是逃不掉的，因为，我给你下了双倍量的那种药，就算是贞女，也会变成豪放女。”季运恒缓慢地向这边踱步而来。
<br/>
<br/>        在这暖橘色暧昧的光线里，他的声线格外悠长。
<br/>
<br/>        童筱颖背靠着床头，额头上有豆大的汗珠，顺着发梢滑落到了她的锁骨上。
<br/>
<br/>        “我……我老板……不会放过你的！我、我卖艺不卖身！”童筱颖试图为自己争取一丝“活命”的机会。
<br/>
<br/>        季运恒却不以为然地笑了笑，一边拉松自己衣领上的领带，一边意味深长地对童筱颖说道：“小谢，我给你打赏了那么多钱，为了你，我还进了拘留所。所以，让我睡你一晚，难道不应该？”
<br/>
<br/>        童筱颖使出浑身解数，身子刚跳下床，去因为四肢无力而摔倒在了床边。
<br/>
<br/>        季运恒此时已经走到了她的面前，缓缓地蹲下身去，伸出手捏住了她的下巴。
<br/>
<br/>        “我都跟你说了，我给你下了双倍的量。你现在是不是觉得，自己四肢软绵无力？”季运恒勾唇一笑。
<br/>
<br/>        童筱颖愤恨地瞪着季运恒，气若游丝地谩骂：“你比我想象中的还要卑鄙！”
<br/>
<br/>        “我卑鄙又如何？你若是从了我，我犯得着用这种下三滥的手段吗？”季运恒手下的力度越来越重。
<br/>
<br/>        童筱颖只觉下巴被他硬生生地掐痛了。
<br/>
<br/>        “你的脸越来越烫了，看样子，你心里一定也迫不及待了吧！”季运恒说着将手放开后，突然轻而易举地将童筱颖的身子捞了起来。
<br/>
<br/>        童筱颖惊吓得脸色半红半紫，无力地抗拒道：“季运恒，你、你放开我！你若是敢碰我，我就咬舌自尽！”
<br/>
<br/>        “哟！小谢姑娘原来知道我的大名啊！看样子，也并不是不把我放心上嘛！”季运恒将童筱颖放到床上后，一手撑在了她脑袋旁的枕头上，一手暧昧地抚摸着她脸上的白羽假面眼镜，“你放心，我暂时不会碰你。我要等你的药效彻底发作后，让你自己主动过来求我！”
<br/>
<br/>        他的手很冰凉，从她的白羽假面眼镜上一路往下，滑到了她滚烫的粉唇上，就像是在撩拨着隐隐的火苗。
<br/>
<br/>        童筱颖的思绪渐渐地凌乱，理智也在慢慢地丧失。
<br/>
<br/>        她看着季运恒的脸，从清晰逐渐到模糊。
<br/>
<br/>        “小谢，快说，快说你要我啊！”季运恒嘴角微扬，唇瓣里挤出一句诱导的话语。
<br/>
<br/>        童筱颖双眼迷离涣散，唇瓣翕动，却一句话也没说。
<br/>
<br/>        季运恒见童筱颖还有一丝她自己的意识，便不急不慢地拉扯着她这件留仙裙腰封上腰带。
<br/>
<br/>        “小谢，你真的是一个唯一让我心里痒痒的女人！”季运恒解开童筱颖身上的腰带后，温柔地褪去了她身上的外纱，这才发现她贴身穿的白色诃子都被汗水给打湿了，“是不是觉得自己很热？”
<br/>
<br/>        童筱颖的眼珠子转动得很慢，找不到目光的焦点在哪里。
