        落以柔回到秘书室里后，独自坐在办公桌前暗自掉眼泪。
<br/>
<br/>        她是真的不想离开总部。
<br/>
<br/>        在这滨江城里有师父郑忠威，还有帮她的连华生，以及她所建立起来的人脉关系网。
<br/>
<br/>        若是被派到云城分公司那边去了，她岂不是又得重新来。
<br/>
<br/>        大哥的心，怎么那么狠呐！
<br/>
<br/>        让她二选一，不接受外派，就辞职……
<br/>
<br/>        郑忠威站在秘书室门外，见落以柔满脸的委屈，忽然想到了一个人。
<br/>
<br/>        大少爷谁的话都可以不听，但是那个女人的话，说不定还能起到点作用。
<br/>
<br/>        所以，童筱颖在接到郑忠威的这个电话时正匍匐在书桌前做自己的设计，当郑忠威提到落以柔后，她手中的铅笔一顿，笔尖断了。
<br/>
<br/>        “大少爷要将小落调去云城的分公司，对我和大少爷来说都是一种损失。小落的辅佐能力很强，在一定程度上辅助了我和大少爷提高了工作效率。大少奶奶，您看看能不能替小落说个情，让她继续留在总部？就当是卖我这二代忠臣一个面子。”郑忠威十分认真又诚恳地说。
<br/>
<br/>        郑忠威自称是“二代忠臣”确实不为过，他辅佐过傅老太爷，现在又辅佐傅夜沉。
<br/>
<br/>        童筱颖虽然不知道他郑忠威是怎么知道她的手机号码的，但是作为“老忠臣”都这么和颜悦色地跟她开口替落以柔求情了，她不答应他，岂不是不近人情？
<br/>
<br/>        该给的面子，她还是的给。
<br/>
<br/>        “好，等大哥回来后，我跟他说。”童筱颖犹豫了一下，才应许了这件事情，顿了顿后，又补充道，“大哥决定的事情，我不一定能左右。我只能说尽量。”
<br/>
<br/>        “能不能不要告诉大少爷，这事是我向您提起的。”郑忠威有所顾虑道。
<br/>
<br/>        童筱颖自然懂他意思，微微一笑：“嗯，您放心吧！我不会说是您主动提起来的。”
<br/>
<br/>        “那谢谢大少奶奶了，改天，我和小落一起请您吃饭。”郑忠威客套道。
<br/>
<br/>        童筱颖也很客气地回答：“请吃饭倒不用了，那郑助理，您先忙。”
<br/>
<br/>        “好，再次多谢大少奶奶，不打扰您了。”郑忠威毕恭毕敬地应道。
<br/>
<br/>        童筱颖和郑忠威通完电话后，就在纳闷，大哥为什么突然要将落以柔调走？
<br/>
<br/>        难道，他俩之间做了什么不可见光的事？
<br/>
<br/>        呃——
<br/>
<br/>        她怎么就总是往坏的方面想？
<br/>
<br/>        或许是因为落以柔给她的印象似乎不太好。
<br/>
<br/>        既然答应了郑助理，要给落以柔说情，童筱颖觉得自己得想个法子。
<br/>
<br/>        要不……先主动约傅夜沉去外面吃晚饭？
<br/>
<br/>        想到这里，童筱颖放下了手中的铅笔，收拾了一下桌上的设计图草稿，起身离开了书房。
<br/>
<br/>        置物篮里的车钥匙，她每次都是随手拿。
<br/>
<br/>        这次，她到了车库后，才发现自己竟然意外抽中了傅夜沉的那辆商务越野车。
<br/>
<br/>        童筱颖嘴角微扬，走过去拉开了车门，却发现副驾驶座上居然放着一束几近枯萎的粉玫瑰，不由地身心一怔。
<br/>
<br/>        她侧过身去，将那束粉玫瑰抱在了怀里，看了看。
<br/>
<br/>        上面有张卡片，写着：I  love  you  all  the  time.（我一直爱你。）
<br/>
<br/>        购花的日期，是她和昼景见面的那天。
<br/>
<br/>        粉玫瑰的话语是“爱的宣言”。
<br/>
<br/>        这花是大哥送给谁的？
<br/>
<br/>        又或者说，是哪个爱慕者送给大哥的？
<br/>
<br/>        难道是落以柔吗？
<br/>
<br/>        童筱颖忍不住想入非非起来。
<br/>
<br/>        落以柔送给了大哥一束粉玫瑰表白，大哥和落以柔发生了一夜情；接着，大哥事后翻脸不认账，想着要把落以柔调走，就此了事；然后，落以柔向郑忠威哭诉，郑忠威心软，于是打电话来跟她童筱颖求情……
<br/>
<br/>        这事儿，童筱颖越想，脸色越黑。
<br/>
<br/>        她这需要太大的脑洞，才能装得下这些狗血想法。
<br/>
<br/>        除此之外，她还得有一颗强大的心来抗压，以及超大容积的肚量，来接受这总裁与秘书之间的潜规则。
<br/>
<br/>        忽然间，静下心来的时候，童筱颖又在想，希望以上都是她的胡思乱想，不是真的，大哥不可能和落以柔有一腿。
<br/>
<br/>        心里什么时候起，如此矛盾呢？
<br/>
<br/>        童筱颖深吸了口，赶走自己脑海里那些乱七八糟的想法后，发动了车子。
<br/>
<br/>        一天之内，来傅氏集团大厦两次，终于他这次不再是在开会了。
<br/>
<br/>        来之前，她有给他发短信，确定他在办公室里后，她才赶过来。
<br/>
<br/>        童筱颖准时出现在总裁办公室门外。
<br/>
<br/>        他的办公室，满室都是超现代化黑白灰的色调，不仅搭配出赏心悦目的简洁明快，而且还彰显了主人的品味。
<br/>
<br/>        这是她三年前，按照他那整整几十页纸的要求，给他设计的。
