        就连阿景和季家合作，所有的往来资料里，都是季运恒的名字，一个“季思妍”这三个字眼都找不着。
<br/>
<br/>        而且，刚刚爷爷傅炎廷在会见他们的时候，提及“远东城军机部项目”时，那位季董事长可是把自己撇得一干二净，说自己完全不知情，全是自己的儿子季运恒的所作所为。
<br/>
<br/>        姜确实是老的辣，关键时刻，连自己的儿子都能“卖”。
<br/>
<br/>        这件事情，虽然是阿景犯了错，爷爷表面上看起来冷酷无情，但打心底里，他还是想办法保住了阿景。
<br/>
<br/>        “那位季大小姐亲口说的，我都听到了，这都还不算证据吗？”连华生条条是道地回答道。
<br/>
<br/>        “她说她和阿景有染，现在又怂恿她父母过来跟我爷爷谈联姻的事情，想要嫁给我。你说，她季思妍所说的话，能信吗？”傅夜沉黑眸微眯，不以为然地反驳。
<br/>
<br/>        就在连华生替他感到无言以对的时候，傅夜沉又接着说道：“当然，这种事情无风不起浪。”
<br/>
<br/>        “你想通了！”连华生不禁喜上眉梢。
<br/>
<br/>        傅夜沉苦笑：“想通了又如何？”
<br/>
<br/>        “想通了，那你就得有所行动啊！自己爱的女人，得靠自己去争取。”连华生兴致勃勃地说。
<br/>
<br/>        傅夜沉不以为然地回答道：“她不爱我，心不在我这，我抢来躯壳有什么用？”
<br/>
<br/>        “所以，你打算无动于衷？”连华生诧异。
<br/>
<br/>        傅夜沉意味不明地笑了笑，不再吭声。
<br/>
<br/>        其实，不管阿景有没有出轨，但是现在阿景已经下定决心要和筱颖在一起了，而且，筱颖一心向着阿景，他傅夜沉再强行去“插”一足，岂不是成了“小三”。
<br/>
<br/>        现在，对他最好的决定，就是视而不见。
<br/>
<br/>        傅夜沉回到自己的偏院时，发现主屋里的灯未亮，便打了通电话询问管家二少爷和二少奶奶是否在傅家大庄园。
<br/>
<br/>        管家这才告知，二少爷傅昼景和二少奶奶童筱颖，已经带着行李向傅老太爷傅炎廷辞行，说他们决定去机场附近的酒店住，明日好直接启程飞往澳洲。
<br/>
<br/>        所以……
<br/>
<br/>        起初在会客厅里，是他见她的最后一面了，是吗？
<br/>
<br/>        想到这里，傅夜沉下意识地摸了摸自己手腕上的这条青丝手链，这条手链，自从戴在了他的手腕上后，他就不曾取下来过。
<br/>
<br/>        现在，是不是该取下来呢？
<br/>
<br/>        可是……
<br/>
<br/>        他发现，他终究是拗不过自己那颗执着的心。
<br/>
<br/>        就这样守着它过这一辈子好了……
<br/>
<br/>        “筱颍，临海城的苏记桃酥很好吃。我去买一些回来，明日我们带去澳洲吃。”刚到酒店的套房里，童筱颖正准备清理衣物洗澡的时候，傅昼景突然神色匆匆地说。
<br/>
<br/>        童筱颖并未想太多，而是拿起衣物一边往浴室门口走，一边微笑着应了声：“好！”
<br/>
<br/>        而那只小博美犬小爱，“嗖”地一下，窜进了浴室里，似乎是想和自己的女主人童筱颖一起洗澡。
<br/>
<br/>        傅昼景见童筱颖关上了浴室门，这才拿起钱包和手机离开了套房。
<br/>
<br/>        他确实去了临海城城东那边的苏记买了桃酥，同时还去见了一个人。
<br/>
<br/>        阿景，如果九点半你不准时来见我，我就割腕自杀，死在临海大酒店808号房里。——季思妍。
<br/>
<br/>        苏记桃酥店的对面，就是临海酒店。
<br/>
<br/>        傅昼景没想到季思妍会对自己如此“纠缠不休”，而他明明已经把她的手机号码给拉黑了，却没料到她会换个手机号码给他发短信。
<br/>
<br/>        808号房。
<br/>
<br/>        傅昼景准时准点地按响了门铃。
<br/>
<br/>        不一会儿，门便开了。
<br/>
<br/>        未等傅昼景缓过神来，一只纤纤玉手，猝不及防地便将他拉进了屋内。
<br/>
<br/>        “乓”地一声，房门自动关上。
<br/>
<br/>        傅昼景定睛一看，只见季思妍身上只裹着一条洁白的浴巾。
<br/>
<br/>        上遮胸，下遮臀，其余白花花的一片，在他眼前晃荡。
<br/>
<br/>        傅昼景顿时一脸不悦地皱起了剑眉。
<br/>
<br/>        其实，他心里明明知道她是在欺骗他，可不知道为什么，他还是选择过来见她最后一面。
<br/>
<br/>        季思妍踮起脚尖，伸出双手，凑上前去勾住了傅昼景的脖子，娇柔道：“阿景，我知道，你心里还有我。我可以成全你和筱颖去澳洲双宿双飞，但是，你能否给我个孩子？”
<br/>
<br/>        “你疯了！”傅昼景毫不犹豫地拉开季思妍的双手，下意识地后退了一步。
<br/>
<br/>        季思妍踱步上前，双眸含泪，举起手来按在胸前，条条是道地哭诉道：“我季思妍这一辈子，只爱你这么一个男人。我什么都给你了，我只想你给我一个孩子，这要求难道很过分吗？”
