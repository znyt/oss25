　　“老大你别开玩笑啊，就算是相赫打上路，遇到这两个也得跪下来啊。”matin一脸郁闷的说道。
<br/>　　“所以这才是考验你真正技术的时候，只要你能怂住，你就是咱们对内的最佳mvp。”金在贤继续劝说道。

<br/>　　“没错，matin啊，到时候我就不去上路了，我主抓中下，你只要稳住他们这个上野联动就炸了。”新来的打野小花生跟着给matin加油。
<br/>　　下路的coco还有newlife两人也都是如此，听到自家队友如此厚颜无耻的话语，这一刻matin的心里真的有一句话不知当讲不当讲了。
<br/>　　而在他们议论着的时候，十分钟的休息时间已经转眼即过，伴随着‘砰’的一声金属碰撞声，场上双方再次进入到了ban选界面。

<br/>　　“皇族这边的ban人很有问题啊，他们为什么就不ban香蕉的英雄呢，香蕉这个点确实值得针对一下的啊。”米粒一脸的不解。

<br/>　　双方轮流选人，很快各自的阵容便是确定了下来。
<br/>　　这一局中香蕉选出了之前版本更新出来，但是在赛场上却是很少有人用过的纳尔这个英雄。

<br/>　　“纳尔！！！”看到相继哦啊的这个选人，米粒激动的再次大声喊了起来。
<br/>　　“这个英雄自从更新出来后，在赛场上出现的次数并不多，只不过为数不多的那两次出场，纳尔的表现却是都不好，主要还是纳尔的怒气很难控制，不过根据各个战队的教练所说，这个英雄确实是可以登场的。”林雅楠跟着开口。
<br/>　　“还有一个消息就是，纳尔这个英雄在荣耀蓝白两队的比赛从来没有出现过，要知道对于版本新英雄这两个战队可是非常热衷的，而这个英雄被无数教练说能够出现在赛场上，他们却都不使用，那么就只有一种可能了，他们认为之前还不是这个英雄登场的时候。”米粒继续说道。
<br/>　　“看来这一次香蕉倒是可以让我们真正认识一下纳尔这个英雄了，我感觉从今天以后纳尔可能要火了。”林雅楠满怀期待。

<br/>　　而白队这边则是选择了：上单纳尔、打野螳螂、中路狐狸、下路则是圣枪游侠和扇子妈这对组合。
<br/>　　“皇族这边这一局的选人很有意思啊，前期有妖姬和蜘蛛这两个英雄，而到了后期他们有老鼠和泰坦这两个英雄，可以说他们这种阵容无论是前期还是后期都可以打。”米粒开口分析着皇族的阵容。
<br/>　　“如果把皇族这个阵容放在和其他的战队对战，我可能会说他们的阵容很好，但是和白队对战，我感觉就是有些不伦不类了，他们这样又想打前期、又想打后期，其实已经丢掉了很多东西了。”
<br/>　　“在和强队对战的时候，最忌讳的就是这种，你想打前期就就选个前期的阵容，这样队友都能配合上，你要打后期，那就全部怂下去，全部选后期阵容。”
<br/>　　“这一局我感觉白队的阵容选的就非常的好，他们这边的英雄无论是狐狸还是圣枪游侠，亦或者是螳螂这些都是前期就很暴力的英雄了。”
