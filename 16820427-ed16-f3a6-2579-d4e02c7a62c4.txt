　　“噗嗤——”
<br/>　　剑锋所指，所向披靡。熊熊之火，凶凶之光，以一种不可阻挡之势，划破虚空，插在了红云的胸膛之上。本来就因为自爆善尸的红云而处于虚弱之中，面对东皇太一这绝杀一剑，他哪里挡得住？
<br/>　　“红云！”
<br/>　　镇元子惊声大叫，见自己老友为自己挡住一剑，他眼中闪过狠厉凶猛，挥动手中的地书，不要命地砸向帝俊。
<br/>　　“去死！”
<br/>　　乙木之光闪烁，道则纵横，镇元子发起疯来，谁也挡不住，帝俊一个不慎，变得镇元子击中，镇元子也来不及去管帝俊，而是转过身来飞向红云。
<br/>　　“咳咳！”
<br/>　　红云胸膛被洞穿，鲜血染红他的衣裳，本来就十分红艳的道袍此刻被鲜血这么一染，却变得更加妖异了。他的嘴角鲜血直流，他却不去阻止，因为他知道，那是阻止不了的，东皇太一灌注在混元剑中的道则，已经在破灭他的身体，破灭他的神魂了。
<br/>　　“鸿蒙紫气，圣道之争，红云老祖，一生逍遥。有人说我傻，有人说我痴，但我并不后悔，人生在世，本就图个潇潇洒洒，自由自在，何必去看别人的想法！”
<br/>　　突然，红云笑了起来，他的笑容掺杂着嘴上的鲜血，怎么看都有些诡异。
<br/>　　“鸿蒙紫气，从一开始就是一个骗局。道祖、圣人、天道！哈哈！想要鸿蒙紫气，咳咳，就算我死，也不会让你们得呈！给我爆！”
<br/>　　“不好！”
<br/>　　东皇太一暗叫一声，看着此刻的红云，他只觉得自己是在面对一个疯子，疯子发起疯来，连自爆这种事都干得出来。他想要躲开，可惜已经来不及了，因为红云距离他太近了，而且红云还死死的抓着太一的混元剑，不让他走。
<br/>　　“轰隆隆——”
<br/>　　一股剧烈的毁灭神光惊天动地，一声雷霆般的轰鸣声毁天灭地，这一刻，空间破碎，大地碎裂，虚空空洞，方圆千丈之内，生灵俱灭，天地寂灭。
<br/>　　周遭黑洞衍生，时空混乱，寰宇破碎，空间紊乱不堪。在这爆炸之中，红云元神之中的鸿蒙紫气直接被卷入到这黑洞之中，消失不见。而红云的荡魂钟，却化为一道流光，飞出爆炸的区域，向着镇元子飞去。
<br/>　　可谁又曾想到，就在此时，异变再起！
<br/>　　天空之中，赫见一道血光乍现，连着一片血海，卷起红云的荡魂钟就走，瞬息之间，便已消失在天际。
<br/>　　“太一！我饶不了你！冥河，你在找死！”
<br/>　　镇元子知道自己的老友红云道人已经死了，连道身都自爆了，还能活着吗？亲眼见到红云为了保护自己而死在东皇太一剑下，那种心情，他比谁都难受。他在此刻恨不得将帝俊与东皇太一碎尸万段，亦不能解他心头之恨。
<br/>　　但红云的灵宝被冥河卷走了，他一样恨冥河，来不及多想，他连忙向着冥河消失方向追去，不是他要追回那鸿蒙紫气，而是他想看看，红云在荡魂钟中是否留下什么，哪怕是一丝真灵也好。
<br/>　　帝俊与鲲鹏也想去追，毕竟那可是关乎到成圣之基——鸿蒙紫气啊。他们可不想如此轻易就便宜了冥河。此时，他们并不知道，鸿蒙紫气已经消失在黑洞中了。
<br/>　　“咳咳……”
<br/>　　就在此时，那爆炸中心，传出一阵轻微的咳嗽声，却见东皇太一拖着半个身体走了出来，他的下半身已经被炸残了，手臂也丢失了一只，他此刻头发披散，血迹斑斑，气息萎靡，简直惨不忍睹。
<br/>　　显然，此次红云自爆，即便他没有死亡，也被重伤了，不用数千上万年时间的修养恢复，休想养好身体。
<br/>　　“二弟！”
<br/>　　帝俊见到东皇太一如此，连忙上前将他扶起，有些担忧地看着东皇太一。
<br/>　　“咳咳……我…还死不了！”东皇太一有些虚弱的说道。
<br/>　　“可恶，竟然便宜了冥河那厮！”同样一身是伤的要是鲲鹏此时有些恨声道。
<br/>　　“咳咳！我等此时都受伤了，小心那巫族前来报复，此地不宜久留，赶紧回天庭，此事以后再算！”
<br/>　　帝俊之前与镇元子大战，同样有些受伤，他想起了巫族，顿时就想要离开。要知道，巫族若是知道他们如今这种状况，前来阻击他们，到时他们哪有精力去抵挡巫族的攻击。
<br/>　　“不错，此事以后再说，冥河想来个渔翁得利，哪有那么好的事？”
<br/>　　“走！”
<br/>　　……
<br/>　　玉虚宫中，元始正在闭目演算，心中万念齐发，因果旋绕，金章神算。在他的周身，散发着一种鬼神难进的气息，道则缠绕，道韵流转，玄之又玄！
<br/>　　良久，他蓦然睁开眼睛，突然伸手向着虚空一抓，顿时，他的大手便伸进无尽虚空之中。不到一会儿，便再次伸了回来，而此刻，他的手中却多了一缕闪烁着紫金光芒的道韵神气，在它的周围，散发着大道韵味，道则闪烁，璀璨无比。
<br/>　　这正是鸿蒙紫气！
<br/>　　显然，红云临死自己送给虚空乱流之中的鸿蒙紫气被他得到了。
<br/>　　“鸿蒙紫气，本座正好用来解析鸿蒙之道，之前那一道不够，希望加上这一道，能够让本座这三大本源之道之一的鸿蒙道有些突破吧。”
<br/>　　元始眼眸深处闪烁着一丝紫金光芒，淡淡道。
<br/>　　接着，他不再说话，来到有鸿蒙元始印中，准备借助元始印参悟鸿蒙之道。
<br/>　　谁知，当他刚踏入元始印中，异变陡生。他手中的鸿蒙紫气来到鸿蒙元始印中，顿时便化为一道紫金流光，脱离他的掌控，飞入元始印核心之处。
<br/>　　紫光闪烁，如同一轮耀眼的紫色太阳，照亮了整片本源空间。光源融化，化为道道玄机，出现在空间之中，道则闪烁，紫光迸发，一股玄妙的感觉，袭上元始心头。
<br/>　　道韵流转，法则奥妙，玄之又玄，元始仿佛来到道则海洋之中，徜徉在紫色海洋之中，对道的领悟在不断上升。
<br/>　　“鸿蒙，乃最古、最老之道，乃灵魂本源之道，灵魂，代表着秩序，鸿蒙，掌控大道秩序。”
<br/>　　“鸿蒙，乃万灵之母，天地万灵，规则道韵，尽在其中。象征着秩序本源之道！”
<br/>　　“混沌之气，乃万气之母，天地万气，五行生克，光暗交替，风雨雷电，雨雪冰霜，尽在其中。”
<br/>　　“混沌，乃物质本源之道！”
<br/>　　“太初玄黄之气，乃万物之母。天地初开，太初临凡，玄者为天，黄者为地，故而，太初玄黄即为天地！”
<br/>　　“太初，乃能量本源之道！”
<br/>　　“鸿蒙、混沌、太初；灵魂、物质、能量，三大本源之道，构建一切大道之源。”
<br/>　　“鸿蒙、混沌，本座已有，至于太初么？”
<br/>　　元始沉吟片刻，将一柄紫色玄天大尺取了出来，紫光流转，玄黄生灭。
<br/>　　这正是，鸿蒙量天尺！
