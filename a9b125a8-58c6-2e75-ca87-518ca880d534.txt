　　叶少阳白了他一眼，这货说的倒是潇洒。
<br/>　　“行了就这样，你们处理好山门的事，就回石城去，在那边等着我。”
<br/>　　叶少阳要关视频，突然想到一个重要的事，嘱咐他们走的时候，一定把本门最核心的功法秘籍带着，给自己参详。
<br/>　　他说的大言不惭，要是别人听到，定会大吃一惊：一个门派的核心秘籍，怎能随便借给别人？不过四宝和吴嘉伟除了好奇他要秘籍干什么，对叶少阳的要求，一句话就答应下来。
<br/>　　这就是兄弟，不管叶少阳要做什么，他们都会无条件支持。
<br/>　　叶少阳又给腾永清打电话，互相祝贺了一下，叶少阳约他去石城见面，让他带着珞珈山的核心秘籍……腾永清有些为难，毕竟他是门派三巨头之一，不是一人毒打的掌教，擅自拿走秘籍恐怕做不到，在问了叶少阳要秘籍的原因之后，腾永清想了个办法，自己用手机去本门两本核心秘籍一页页拍下来，微信上发给叶少阳，看过后就删，这样效果一样，也免除了一切麻烦。
<br/>　　叶少阳爽快答应，于是腾永清立刻着手去干了。
<br/>　　“还有谁能弄到门派秘籍呢……”叶少阳拿着曲波给自己的名单，在上面一个个名字看过去。
<br/>　　“老大你上瘾了啊！”瓜瓜在一边吐槽道。
<br/>　　“尝到甜头了呗。”叶少阳随口敷衍，锁定了一个名字：李媛媛。虽然不熟，但从前听她说话，好像跟道风有过什么来往，倒是可以试探一下……于是立刻给曲波打电话，让他找李媛媛的联系方式，十分钟后，曲波把李媛媛的手机号发过来，叶少阳犹豫了一下，打过去。
<br/>　　李媛媛还记得他，叶少阳先跟她客套了几句，接着三言两语达成了一致……
<br/>　　后半夜，叶少阳仍然没睡，他在龙虎山诸多典籍之中找到了一本没名字的书，线装，看上去很老，但又不是特别老的那种古书，里面记载了人间法术各种修行门类，说白了就是一本讲法术的技能树。
<br/>　　道童，方士，真人，天师，地仙，灵仙，上仙。
<br/>　　这是道家主流修炼方式最标准的晋升等级，一般认为上仙就是最高境界了（大部分人只知道天师），叶少阳一直挺好奇上仙之上还有什么境界，虽然不甚了解，但身为上仙的他，觉得自己的实力既然连一流强者都算不上，那就显然上仙并不是最终的境界。
<br/>　　可惜青云子、广宗天师、道渊真人、徐福等几个有见识的宗师都不在人间了，真是想找人问都问不到，道风也不甚清楚。
<br/>　　如今在这本书里，总算找到了答案。
<br/>　　上仙之上，居然还有两重境界，分别是造化和无极。
<br/>　　对这两重境界，书上只有简单的介绍：道之大成，功参造化，领阴司四方巡游勾陈紫府大将军牌位，与诸王判官同列；
<br/>　　无极，洞虚阴阳，参悟天人，可与神明争高低。
<br/>　　造化和无极……叶少阳反复看了几遍，觉得造化境界很牛逼了啊，一旦修成，就跟几个王爷和崔府君同级别了。那无极境界的描述虽然只有一两句话，但听上去更牛逼啊，能跟神明一争高低？
<br/>　　看来，自己这实力还是不行啊，叶少阳顿时就对修行之路又充满了向往。
<br/>　　再翻到后面去，还有不少有关修行的介绍，叶少阳通篇看完，对修行有了更深的了解，原来世上有这么多修行之道。
<br/>　　光是道佛二门，除了常规修行之路外，就有不少非主流的路子。道门有清静道、全真道、无为道，等等，都是不同的修行之路，上限也都不一样。
<br/>　　想起青云子修的是清静道，怪不得他境界只是地仙，但实力那么强……
<br/>　　佛门的修行之法，比道门的还要多……
<br/>　　叶少阳又前后看了半天，书没名字，也不知道谁写的，他猜测有可能是道渊真人的手笔，只有他这个活了一百多岁的活化石，才会对这些知识有这么全面的了解。
<br/>　　叶少阳把这本笔记放在自己背包里，看看时间不早了，也就直接睡觉了。
<br/>　　第二天中午，他从龙虎山出发，带着瓜瓜和小马一起，坐飞机到台州。
<br/>　　这是他第一次到台州，坐在出租车里，从市区的高楼大厦经过，叶少阳一路看着街景，感觉跟石城也没啥差别，就是路边树木多一点，但人没有石城多，看样子都比较闲。
<br/>　　在市中心开了个快捷酒店，进房间后，小马一个劲吐槽叶少阳小气，“你看你，好歹也是两大门派的掌教，要地位有地位，要钱有钱，住这样的酒店，简直对不起你的身价啊！”
<br/>　　“我自己都不在乎，你一个鬼，管这么宽干什么！”叶少阳翻着白眼，这一路上跟小马在一起，感觉熟悉的时光又回来了，就是……无时无刻都好像有一群苍蝇围着你嗡嗡叫，简直烦死。
<br/>　　叶少阳不服辩道：“我是鬼怎么了，鬼也有尊严的嘛！”
<br/>　　叶少阳懒得理他，激活雪花玛瑙戒，将位置传递给道风。
<br/>　　“今天的行动，你们不要参加了，瓜瓜，你去空界找下军师他们，看看他们那边的情况，回来告诉我。”
<br/>　　“好吧。”瓜瓜本来也早就想去了，等叶少阳打开通往青丘山的虚空通道，便钻了进去。
<br/>　　“我不去。”小马见叶少阳望着自己，连忙摆手，“我对青冥界没兴趣，除非真有架打，否则我不去！”
<br/>　　叶少阳道：“我是要去天台山，山上可以有禁制，你是鬼，万一触发禁制，比较麻烦。”
<br/>　　“那我不去，我自己溜达去。听说台州小吃不错，我找个地方尝尝。你有啥事就叫我啊。”小马说完愉快地出门去了。
<br/>　　叶少阳在房间等了不到半小时，道风来了。
<br/>　　“怎么样，那佛像还在天台山吗？”一见面叶少阳就紧张地问道。
<br/>　　“走吧。”叶少阳算是从侧面回答了，于是两人出门，叶少阳滴滴叫了一辆车，直奔天台山风景区，道风坐在他身边，因为司机看不到他，叶少阳也不好跟他说话，免得司机以为是活见鬼了。
