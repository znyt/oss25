　　这老者的举动，就好像是在扔一个垃圾一般。
<br/>　　这更让众人觉得，这古籍绝对不是什么宝贝。
<br/>　　“这古籍是好东西，自己参悟吧。”
<br/>　　说着，这老者便是想要再度躺下睡觉。
<br/>　　“十万皇品灵石，给我多好，给我十万皇品灵石，我就有冲击天命之境的机会了。”
<br/>　　一名生死台阶第六重炼心之境的武者，嫉妒的说道。
<br/>　　他的境界距离天命之境，只差一步之遥，如果给他这么多的灵石，绝对是足以踏入天命之境，成就皇者。
<br/>　　一旦是成为皇者，在如今的九域都算是一位强者了，地位显赫。
<br/>　　可这么多的灵石，都被楚风眠买了一本这样无用的古籍。
<br/>　　“等着后悔吧。”
<br/>　　不少人都是幸灾乐祸的看着楚风眠。
<br/>　　等着看楚风眠发现这古籍没有任何用处之后的表情。
<br/>　　不过楚风眠接过古籍之后，却是没有打开的想法，反而是又看向了老者的方向，走了过去。
<br/>　　“这小子还打算做什么？”
<br/>　　看到楚风眠的举动众人都有些不明白。
<br/>　　按理说买到这样一本古籍，就应该赶紧离开去参悟了，可现在楚风眠居然还不打算走？
<br/>　　一道道目光盯着楚风眠，围观众人也没有看去，也想要看看楚风眠还打算做什么。
<br/>　　“还有什么要卖的东西吗？”
<br/>　　楚风眠看着老者，缓缓开口道。
<br/>　　此话一出，又是震惊众人。
<br/>　　所有人的嘴巴都是长得大大的，许久合不上，在场鸦雀无声。
<br/>　　“还要买？”
<br/>　　“这小子是疯了？还是灵石太多没处花了？”
<br/>　　从这老骗子手里买东西，就更挥霍灵石没有多少的区别。
<br/>　　楚风眠当一次冤大头就算了，现在居然还打算当第二次冤大头？
<br/>　　这跟在一棵树上撞死两次有什么区别？
<br/>　　他们在归墟城待了如此之久的时间，都从来没有碰到过这样疯狂的事。
<br/>　　“这小子莫不是这老骗子的托？”
<br/>　　一道道目光看向楚风眠，这得是多傻的人，才会真正信了这老骗子的话，从他这里花天价买东西。
<br/>　　“这位公子，我这里也有些好东西，比这老骗子的便宜多了，不然你也来看看？”
<br/>　　“对，我这里可有一件数十年前的古灵器，还是天级灵器的，公子不打算来看看？”
<br/>　　人群之中几个人更是向着楚风眠喊了起来。
<br/>　　既然是来了一个冤大头，那不坑白不坑。
<br/>　　不过楚风眠却是丝毫没有理会过这些人，而是看着一直盯着老者。
<br/>　　“你还要买？”
<br/>　　老者缓缓的起身，仔细的打量了楚风眠一眼，开口道。
<br/>　　“东西有，就怕你买不起。”
<br/>　　“先拿出来吧。”
<br/>　　楚风眠平静的开口道。
<br/>　　这口气，简直是灵石都是天下掉下来的一样。
<br/>　　让众人听见，都是一阵无语，看向楚风眠的目光都是无比嫉妒。
<br/>　　“哼，不知道是哪个宗门出来的公子哥一个。”
<br/>　　“有些灵石就乱花。”
<br/>　　“老骗子这一次可发财了。”
<br/>　　在众人一道道议论声中，老者打开了空戒，在他的手心之中，又出现了一个玉佩。
<br/>　　这一枚玉佩，看起来也是无比古老，至少有着数十万年的历史了，比起那古籍，还要破旧。
<br/>　　从上面，已经是几乎感觉不到灵力了，上面刻画的阵法禁制，也已经全部破碎，不成形状了。
<br/>　　跟一件废灵器，都没有多少的区别了。
<br/>　　“老骗子，这就是你说的宝物，一件废灵器？”
<br/>　　人群中有人起哄讥讽道。
<br/>　　老者没有理会人群的话，看向楚风眠，不咸不淡的开口道。
<br/>　　“买不买。”
<br/>　　“这玉佩什么价格？”
<br/>　　楚风眠盯着老者，同时看向了那玉佩。
<br/>　　这玉佩被拿出来的一刻，楚风眠感觉到，放入到了空戒之中的那一本古籍，居然都有着一阵灵力波动出来。
<br/>　　显然是这玉佩，跟古籍是一起之物。
<br/>　　这一本古籍对楚风眠有着如此巨大的吸引力，这一枚玉佩，楚风眠也不可能放过。
<br/>　　“一千万皇品灵石，或者十万圣品灵石。”
<br/>　　老者的回答一如既往的冷淡。
<br/>　　漠不关心一般。
<br/>　　但是他口中开出的，又是一个新的天文数字。
<br/>　　一千万皇品灵石，比起刚刚那古籍的价格，还要翻上一百倍。
<br/>　　这话一出，在场的众人都是吸了一口凉气，一千万皇品灵石，只怕是就连真帝都未必拿得出来吧。
<br/>　　如此之多的皇品灵石，拿出来都要堆积成山了，一些小一些的宗门宝库之中的财富，都没有这么多吧。
<br/>　　这简直是在漫天要价。
<br/>　　“我买了。”
<br/>　　就在众人的震惊之中，楚风眠率先开口。
<br/>　　他的手心之中，流光闪动，一枚枚晶莹通透的灵石飞了出去。
<br/>　　精粹的灵气，顿时从这灵石上散发了出来，周围的灵气都浓郁了无数倍。
<br/>　　“圣品灵石！”
<br/>　　“这是圣品灵石！”
<br/>　　“好精粹的灵力，给我几块圣品灵石，我就可以立刻突破了！”
<br/>　　“这小子到底是什么人？圣品灵石都拿得出来？据说在七大宗门之中，圣品灵石都是难得一见的宝物，只有真正的大人物才可以拿来修炼。”
<br/>　　一道道目光死死的盯着楚风眠手心之中的圣品灵石，心中各有想法，不少看向楚风眠的目光都带有着几分杀意。
<br/>　　如此之多的财富，足以让人起杀心了。
<br/>　　不过楚风眠却是全然没有理会，一直是拿出了十万圣品灵石，才手心一捏，化为了一座小山，扔给了老者。
<br/>　　不少的武者，甚至都有些按奈不住，要出手抢夺了，可看着周围巡视的武者，还是压抑了下来。
<br/>　　老者接过灵石，数了一下，再度放入了空戒之中，他的脸色从头到尾都没有任何的变化。
<br/>　　要是让一名寻常武者，甚至是让一名准帝到来，看到如此之多的财富，都要脸色大变。
<br/>　　可这老者的表情，却是从没有过任何的波动，看向这些灵石的目光，就好像是看向一堆破石头一样，随意的就给放入到了空戒之中。
<br/>　　随后他便是拿起玉佩，再度扔向了楚风眠，好像是丝毫不在乎这一枚玉佩一样。
<br/>　　楚风眠接过玉佩，打量了一眼，楚风眠便是放入到了空戒之中。
