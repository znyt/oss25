　　半晌，任务处的那位猎妖殿人员回来，并递还萧逸的令牌。
<br/>　　“易霄先生，这是你的令牌，已为你做好登记，从现在起你是三级猎妖师了。”猎妖殿人员客气地说道。
<br/>　　“我现在可以接三级任务了吧？”萧逸问道。
<br/>　　“自然是可以的。”猎妖殿人员说道，“不单止三级任务，四级、五级任何一个级别都可以接。”
<br/>　　“不是说有限制，不能越级接任务吗？”萧逸疑惑道。
<br/>　　猎妖殿人员笑笑，道，“那只是对于一级猎妖师而已，他们都是些菜鸟，实力不强。我们猎妖殿怕他们胡乱接任务，反倒让自己丢了性命，所以才有这个限制。”
<br/>　　“你已经完成了一个三级招募任务，又有先天境修为，如今已是三级猎妖师，自然不再有这些限制。”
<br/>　　“原来如此。”萧逸点点头，接过令牌，然后在任务处接了些任务。
<br/>　　顺带一提，他自从离开洞府后，便一直戴着幽魂面具。
<br/>　　曾经是杀手之王的他，很清楚，许多时候多一重身份，很可能就是多一条命。
<br/>　　既然如今以易霄的名字当着猎妖师，便暂时戴个面具吧。
<br/>　　在外行走的独行武者，有不少都会戴个面具，原因很多，或隐藏身份，或躲避仇家，或本身面容有问题。
<br/>　　总之，戴个面具对于武者是很平常的事，倒也不会引起别人格外注意。
<br/>　　就萧逸自己在破玄城中走一趟，就能遇到不少戴面具的武者。
<br/>　　再者，幽魂面具的样子也是平平无奇，和其它面具看起来差不多。
<br/>　　离开猎妖殿，萧逸准备前往裂天剑派了。
<br/>　　正当他踏出猎妖殿大门时，却迎面走来几个武者，竟是铁达大叔他们。
<br/>　　但他们都没留意萧逸，径直走了过去。
<br/>　　猎妖殿中每日武者来来往往，每人会特别去留意别人。
<br/>　　“队长，你说这次我们接什么难度的任务好？”
<br/>　　“要不要再发个招募任务招些人手呢？”
<br/>　　林飞和王源边走边问道。
<br/>　　铁达大叔叹了口气，“唉，易霄兄弟死了，浅沫也被家里人带走了，我们可没那个运气招募到那么好的队友了。算了吧，反正这次任务的难度不大。”
<br/>　　“诶对了林飞。”铁达大叔继续说道，“待会儿去买些香烛，过几天去雷鸣荒泽出任务时，顺便拜祭拜祭易霄兄弟吧。”
<br/>　　“好。”林飞点点头。
<br/>　　几人边聊着，就在萧逸身边擦肩而过，进了猎妖殿。
<br/>　　一旁的萧逸不禁满头黑线。
<br/>　　不过没有表露身份，反正就要离开破玄城了，日后也不会有交集，知道大伙儿没事就好了。
<br/>　　再者自己和北山慕容家还有过节，还是不要和其他人有关系吧，免得牵连他们。
<br/>　　想罢，萧逸回头看了眼铁达大叔等人，而后离开，准备前往裂天剑派。
<br/>　　.....
<br/>　　破玄城内，某间客栈的一个奢华房间内。
<br/>　　浅沫不停地摇着一个老者的手臂。
<br/>　　老者满脸苦涩，“小姐，你不要再摇我了，不然我这身老骨头要散架了。”
<br/>　　浅沫停下摇曳，哀求道，“那木爷爷你带我去那狂血玄君的洞府好不好，就一次。”
<br/>　　老者摇摇头，无奈道，“小姐，我知道你想救那易霄。可是，狂血玄君那厮的洞府，不，那根本不能说是洞府，本来就是个大型封印之地。”
<br/>　　“那可是狂血玄君以自己的命创造出来的，起码要强他数倍的武者才能强行打破禁制，岂是说进去就进去的，起码我是没那个本事的。”
<br/>　　“你骗人。”浅沫不依不挠。
<br/>　　老者苦笑道，“小姐阿，我真没骗你。别说是我，就算整个北山郡，怕也找不出能比当年的狂血玄君强数倍以上的武者。”
<br/>　　“以那个狂人的实力，等闲地元境都奈何不了他，起码要地元中期甚至是后期的武者才行。你说吧，你木爷爷我有何本事去破他的禁制。”
<br/>　　老者继续说道，“小姐，你想阿，像破玄城这种大城，有好几个大家族的家主都是洞玄境呢，以他们的势力，连普通猎妖师都知道的消息，他们能不知道？”
<br/>　　“可他们一个武者都不派去，摆明知道那里有猫腻嘛，派人去也是送死，根本捞不到宝贝的。”
<br/>　　“也就是那些笨蛋才会去，那易霄死了也是白死，他也是蠢...”
<br/>　　老者见浅沫脸色难看，闭上了嘴巴。
<br/>　　“你...你欺负人。”浅沫急得猛跺脚，一把扯过老者的胡子。
<br/>　　“诶哟...别扯我胡子...疼疼疼...”老者连忙说道，“好了，算木爷爷说错话了，你先放手。”
<br/>　　浅沫放开老者的胡子，急道，“北山郡没人能破开禁制，我回去找爷爷，爷爷一定能帮我。”
<br/>　　老者说道，“以谷主的实力，当然是能破开那禁制的。只是，我们回药王谷一趟，再回这北山郡，还来得及吗？”
<br/>　　“依你之前所说，我猜测那天星雷鳄必然还在封印之中，只是凝聚出了分身，还在禁制上做了手脚，那些普通的猎妖师才进得去洞府。”
<br/>　　老者肯定非泛泛之辈，竟没有亲自到过洞府，却已经猜到了许多事情。
<br/>　　老者继续到，“你不知道那天星雷鳄是何等凶兽，他凝聚的分身，足以碾压洞玄境以下一切武者。小姐你说吧，这么长时间，那小子不早就死了？”
<br/>　　“再者，你说的那场大爆炸，怕也已经将那小子炸死了。”
<br/>　　“我...”浅沫忽然脸色煞白，无力地坐倒。
<br/>　　老者语重心长地说道，“小姐，听木爷爷的。你这次溜出来，可把谷主和你父亲担心坏了，再不回去，他们要急死了，我们还是先回去吧。”
<br/>　　浅沫微微点头，却不言不语。
<br/>　　“唉。”老者叹了口气，暗道，小姐打出生后就受尽万般宠爱，未曾离开过家人，不谙世事，心地善良，不知这世间险恶。那易霄也不知怎么回事，竟让得小姐失魂落魄，一门心思都放在他心上。
<br/>　　罢了，反正那小子都死了，多想也是没用。
<br/>　　“走吧。”老者扶起浅沫，认真道，“小姐，我们可说好了，你要再在我眼皮子底下溜走，谷主非掐死我不可，你乖乖地，别捣蛋了阿。”
<br/>　　浅沫仍旧是不言不语，瘦弱的身躯显得有些可怜。
<br/>　　“唉。”老者再次叹了口气。
<br/>　　两人离开了破玄城。
<br/>　　.....
<br/>　　破玄城，城门外，萧逸看了眼那古朴的三个大字，笑笑，转身离开。
<br/>　　恰在他走后没多久，一个老者带着一个少女出现。
<br/>　　少女一身青衣，小小年纪便长得沉鱼落雁，不难想象长大后必是个绝色美女。
<br/>　　可是，不知为何，少女此刻脸色煞白，目光游离，也不知在想什么伤心事。
<br/>　　临行前，少女转身看了眼城门上的三个古朴大字，惨然地笑笑，转身离去。
<br/>　　......
<br/>　　萧逸离开破玄城后，立刻赶路，目的地裂天剑派。
<br/>　　当然了，顺带着，他还接了不少任务，全都是能在路上顺手完成的。
<br/>　　破玄城百里之外，某个荒郊野林里。
<br/>　　萧逸站于树上，瞥了眼下方，自语道，“这就是毒龙山贼团的老巢了吧，啧啧，还真是个大型山贼团，人数不下数百，为首的还是个先天九重武者，先天境也有数人，难怪那么多猎妖队都奈何不了他们，甚至死在他们手上。”
<br/>(本章完)
