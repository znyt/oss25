<br/>    太初古林外。
<br/>    
<br/>        陈明找到了陈灵鱼，关于生死簿和判官笔，到底给不给陈灵鱼，陈明思考了很久，最后还是决定给陈灵鱼，虽然这件仙器效果有些诡异，威力很强，陈鱼这小丫头可能一时兴起就用用，但是还是保障陈灵鱼的安全要紧。
<br/>    
<br/>        陈灵鱼又没有像是卓青瑶拥有，凌仙的，李素衣的，有这两件仙器防身，陈明才放心。
<br/>    
<br/>        陈鱼对于陈明找到她，还以为陈明有什么事安排呢，“师父，还有什么事吗？”
<br/>    
<br/>        陈明道:“我这一次找你，是想给你两件仙器，不过这两件仙器威力莫测，效果诡异，你要谨慎使用，明白了吗？”
<br/>    
<br/>        陈鱼笑道:“师父，我你还不放心吗，我又不是那种喜欢打打杀杀的人！”
<br/>    
<br/>        陈明看了一眼外面的天色，不对啊，是晚上啊，陈鱼这小妮子，也是转了性了？
<br/>    
<br/>        陈鱼对于仙器可是已经向往已久，大师姐，二师兄，三师兄，可是都有着仙器的，终于轮到自己了吗？而且听师父的口气，这两件仙器似乎比其他人的都厉害！
<br/>    
<br/>        陈鱼亲昵地抱着陈明的胳膊摇晃，轻声问道:“师父，师父，是啥好东西啊，给弟子瞧瞧嘛！”
<br/>    
<br/>        陈明终究是拗不过陈鱼的撒娇，对于两个女弟子的撒娇，陈明没有任何免疫力，都没用！
<br/>    
<br/>        陈明张开手，判官笔和生死簿落入手中，“你拿去吧，记住了，这两件仙器，一定要慎用，威力莫测，有可能误伤自己人。”
<br/>    
<br/>        陈鱼高高兴兴地接过了判官笔和生死簿，只是第一眼，就喜欢上了这两件仙器，拿了过去，认主倒是没有什么难度，轻松认主。
<br/>    
<br/>        陈明心说，得，这一波抽奖又是全便宜了自己的弟子，自己倒是啥也没捞到。
<br/>    
<br/>        哎，就当施肥了！
<br/>    
<br/>        陈灵鱼有了判官笔和生死簿，陈明也是放下心来，现在世道太乱了，还是多点保障的好。
<br/>    
<br/>        如今太初古林的后续也处理的差不多了，陈明开始准备另外一件事，因为弄到了十八对子母空石，陈明在燕山布置了一座传送阵，剩下的一座传送阵，陈明带在了身上的储物戒之中，随时可以回到燕山。
<br/>    
<br/>        现在的话，陈明准备将其他的传送阵也布置完毕，然后全部带在自己的身上，甚至于，陈明有些想将自己的十二柄国器之上，都绘制一个传送阵，这样一来，自己就可以在十二柄国器之间传送，有点像是瞬移的效果。
<br/>    
<br/>        以陈明如今对于阵法和炼器的造诣，这倒也不是什么难事，只是需要时间。
<br/>    
<br/>        有了这一招，对于战斗的帮助可是不小，这可是瞬移啊！
<br/>    
<br/>        对方一招打来，瞬间移动，根本打不着自己，简直无敌了！
<br/>    
<br/>        当然，大君也不都是傻子，有机会能够有看得出来自己的移动规律，看出十二柄国器之间互相传送的方式。
<br/>    
<br/>        但是大君的战斗，等他完全分析出规律，已经占了巨大的优势了！
<br/>    
<br/>        作为一个修了假仙的人，战斗方式肯定不能用常规的战斗方式，必须彰显我们的职业特色，三千大道，在下都略懂略懂！
<br/>    
<br/>        关上营地的房门，陈明在房中敲敲打打，开始改造这十二柄国器，一柄剑传送到另外一柄剑，倒是无需太多子母空石，一对子母空石就可以完成两柄国器，不过陈明做事，向来喜欢给别人意想不到的惊喜，所以用了七对子母空石，破空剑可以在两柄国器上互相传送。
<br/>    
<br/>        等到对方发现了传送的规律之后，以为破空只会传送到一柄国器上，单独防御这一柄国器的时候，突然传送到另外一柄国器上，给对方一个意向不到的惊喜，绝对的刺激！
<br/>    
<br/>        除了国器之上，还要有一对普通的灵剑，也可以互相传送，就要那种和其他灵剑样子差不多的，等到对手觉得已经完全掌握了国器之间的传送规律，额，我普通的灵剑也可以互相传送，绝对是一个重磅惊喜！
<br/>    
<br/>        抡动着铁锤，绘刻符印，陈明在三天之中都把自己关在了房间中，不断地炼制着自己的新兵器，一个个符印，一座座阵法，加持在一柄柄国器之上，陈明的眼中逐渐泛起一抹抹血丝，看向周围的国器，以及一对对子母空石，开始将一块块空石镶嵌在国器之上，镶嵌之后，开始绘制符印，沟通子母空石，以便可以用法力激发传送阵。
<br/>    
<br/>        子母空石的原理倒是很简单，就是两处可以互相传送，不存在一个传送阵传送到另外两个地方的可能。
<br/>    
<br/>        一次接着一次，陈明的手下不停，开始不断地改进自己的国器。
<br/>    
<br/>        在三天之后，陈明推开门，满头乱发，陈灵鱼看到笑了笑，打了一盆水来，准备给陈明打理一番。
<br/>    
<br/>        陈明现在累了三天，浑浑噩噩的，木木的走到桌子前坐下，“对了，先给我弄点吃的来，有些饿了！”
<br/>    
<br/>        陈灵鱼烧了几个菜，端到陈明的面前，陈明吃了几口，却是没有感受到自己上一次这么饿的时候吃到的那个味道，嗯，上次是什么东西特别好吃来着。
<br/>    
<br/>        对了，是卓青瑶的面，好久没有吃到卓青瑶的面，忽然有些想念。
<br/>    
<br/>        难不成，自己喜欢上了卓青瑶做出的味道，也没什么不对，卓青瑶的面，现在做的是好吃啊！
<br/>    
<br/>        只是，若是卓青瑶嫁人了，自己上哪儿吃面去？
<br/>    
<br/>        都说女大不中留啊，现在卓青瑶可是已经十六岁了，正是春心萌动的时候，很是危险啊，不过还好自己曾经上了一课，应该问题不大。
<br/>    
<br/>        这个时候，天色忽然黑了下来，陈鱼上线了，一脸冷酷地看向陈明，“师父，你是去偷鸡去了吗？怎么头发这么乱？”
<br/>    
<br/>        陈明看了陈鱼一眼，话说我为啥要担心这两个小妮子嫁人啊，以这两个小妮子的脾气，能够嫁的出去才有鬼了！她们能够嫁出去，我陈明的名字倒着写！
<br/>    
<br/>        陈明白了陈鱼一眼，道:“既然看到师父的头发乱糟糟，你就不知道过来帮师父梳一下，眼力见呢，没看到你师父累成狗了？”
<br/>    
<br/>        陈鱼很是认真，然后道:“师父，不要说自己累成狗了，实际上，狗并没有你累啊！”
<br/>    
<br/>        陈明:
<br/>    
<br/>        你的话竟然该死的甜美，让我无言以对。所以，我到底在担心啥？
