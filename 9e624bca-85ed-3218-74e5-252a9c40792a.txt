　　噗嗤噗嗤。
<br/>　　两道清脆的声音响起。
<br/>　　一刀下去，两颗头颅腾空而起。
<br/>　　“太轻松了吧。这把刀这么锋利？”
<br/>　　林峰大喜过望，他刚才都没有察觉到任何的阻力，就好像是在切纸一样，一刀下去，两颗头颅飞出。
<br/>　　林峰此时此刻，对鲜血已经不在恐惧了，可能是神经处于高度紧张状态下，也可能是之前已经见过血了，心中有了一定防备。
<br/>　　“不知道这两个丧尸脑袋里有没有那个东西，若是这两个丧尸也有就说明这晶石一样的东西绝对有用。”
<br/>　　林峰直接像劈西瓜一样劈开那两个头颅，果然也都各自找到了一颗。
<br/>　　林峰把他们都收拾好，回到空投箱子内。
<br/>　　箱子内除了一把武士刀之外，还有一些东西。
<br/>　　三包薯片两瓶水以及一个黑色背包。
<br/>　　林峰拿起背包。
<br/>　　叮！
<br/>　　开启物品栏！
<br/>　　顿时手中的背包消失不见。
<br/>　　取而代之的就是脑海之中的多出的物品栏，目前可以储存二十个东西。
<br/>　　“还有这种操作？还真是方便。”
<br/>　　林峰立即把三包薯片和两瓶水都放入其中，占据了五个物品格。
<br/>　　想了想把武士刀也放进去了，毕竟这东西出现的太诡异，林峰自己都解释不清楚，等要用的时候在拿出来，反正只要他念头一动，这东西随时可以出现，很方便。
<br/>　　林峰拿着三角尺，就朝下奔去。
<br/>　　“现在去救狗子！”
<br/>　　林峰原路返回，回到办公室，苏云依然还躺着。
<br/>　　林峰把她扛起来回到教室内。
<br/>　　此时此刻班级内已经死人了。
<br/>　　能够战斗的就那么几个，大部分女学生都已经吓瘫痪了，坐在地上要么哭要么发呆，一群男生也好不到哪里去，唯一能打的也就是两百斤的狗子和一个叫做孙畅的高个子。
<br/>　　此时此刻，陈勾已经被三个丧尸给包围住。
<br/>　　“妈的，想胖爷我之前给你们写情书，你们爱理不理的，现在三个人要和胖爷我群战。”
<br/>　　忽然，他看到了什么，顿时眼睛一亮。
<br/>　　林峰把苏云放在教室外。
<br/>　　目前整个楼道是安全的，其余几个班级，一个去多媒体教室听报告去了，两个去上体育课去了，就剩下他们这一个班级。
<br/>　　所以只要把教室内的丧尸解决掉就算暂且安全。
<br/>　　林峰对陈勾做了个嘘的手势，然后偷偷的靠近其中一个女丧尸，这女的林峰知道，是吴倩倩的一个死党，特爱穿超短裙，现在穿的就是超短裙。没事找事就喜欢张开腿故意吸引男人来看他。
<br/>　　林峰拿着三角尺就是靠近过去，而那丧尸没有任何的注意。
<br/>　　杀！
<br/>　　噗嗤！
<br/>　　三角尺直接洞穿了其中一个丧尸的脑袋，另外两丧尸顿时察觉，纷纷朝林峰怒吼。
<br/>　　“啊啊啊！”
<br/>　　林峰也张开嘴大声怒吼，然后一脚把靠的最近的一个给踹飞了出去。
<br/>　　剩下的一个丧尸就是要扑过来。
<br/>　　砰！
<br/>　　但下一刻，就被陈勾给扑倒在地。两百斤的肥肉压在这个丧尸身上，这丧尸顿时动弹不得。
<br/>　　“妈的，胸还不小，还敢动，在你胖爷身下还敢调皮。”
<br/>　　林峰立即拔出三角尺，然后给这个丧尸也来了那么一下。
<br/>　　“林峰你来得太是时候了。”
<br/>　　陈勾说道。
<br/>　　“你说，这算不算杀人啊？”
<br/>　　“算个毛，外面世界都变了，估计是打仗了，还管毛的杀人，再说就算杀人，我们这也叫正当防卫！”
<br/>　　林峰立即拉起胖子，此时此刻，教室内还有最起码七八个丧尸正在围攻吴倩倩他们，不过那个叫孙畅的事先用桌子翻过来组成了一条防线，勉强挡住了。
<br/>　　“你们看，林峰居然杀了丧尸！”
<br/>　　“林峰，快来救我们啊。”
<br/>　　几个男生看到林峰顿时大喊道就好像看到了希望。
<br/>　　这几个男生和林峰关系不好不坏，算是正常关系，林峰也不能见死不救，就和陈勾一起冲了过去，陈勾拿着个课桌当武器看到丧尸就猛砸。
<br/>　　顿时砸翻了两个。
<br/>　　林峰发现这丧尸似乎不打脑袋不会死，顿时道：“狗子，砸他的脑袋，这玩意脑袋非常脆！”
<br/>　　“是吗？”
<br/>　　陈勾愣了一下，旋即猛地一砸。
<br/>　　噗嗤。
<br/>　　“哎呦我去，还真是，旺旺小小酥啊，这么脆。”
<br/>　　而林峰这边也偷袭干掉了一个丧尸。
<br/>　　孙畅看到林峰和陈勾这么勇猛，也立即道：“大家一起出手！”
<br/>　　说着他也是第一个反抗。
<br/>　　在众人努力下。
<br/>　　剩下丧尸全部被解决，但可惜的是一个女学生被抓伤了。
<br/>　　“这怎么办。”
<br/>　　孙畅看着林峰道。
<br/>　　对于林峰，他之前是不理不睬的，但是现在，他把林峰当做主心骨了。
<br/>　　“我看电影，被丧尸抓伤了的人都会变成丧尸。”
<br/>　　一个男生害怕道。
<br/>　　“我不会变成丧尸的，求求你们救救我吧，林峰，我们是同学啊。”
<br/>　　女生哀求道。
<br/>　　林峰皱起眉头，“电影里是不是真的我不知道，你会不会变成丧尸我也不知道，但是你对于我们来说很危险，必须要把你隔离开来。”
<br/>　　林峰对陈勾示意。
<br/>　　后者点了点头立即把那女生给拖到隔壁办公室内，然后把门反锁起来。
<br/>　　“搞定了。”
<br/>　　陈勾拍了拍手道。
<br/>　　“林峰，你赶快护送我回家，我家里有保镖，我爸手里还有枪，他可以保护我的，到时候我让我爸给你钱。”
<br/>　　吴倩倩忽然从地上站起来道。
<br/>　　“护送你回家？”
<br/>　　林峰冷笑了一下。
<br/>　　“你知不知道外面发生什么了，自己都看一看外面的景象吧。”
<br/>　　一群人这才有空去看外面发生了什么，这一眼扫过去，每个人都彻底陷入了沉默。
<br/>　　“我不管！林峰，我给你钱！你一个无父无母的孤儿肯定需要钱吧？只要你护送我回家！”吴倩倩道。
<br/>　　“没空！要回家你自己回家！我还护送你？想太多了吧你。还是说你真的把自己当公主了，要每个人都围着你转。”
<br/>　　“林峰，你不怕曹豹找你麻烦吗？”
<br/>　　吴倩倩看林峰不答应居然威胁起来了。
<br/>　　啪！
<br/>　　下一刻。
<br/>　　一巴掌林峰就给她抽翻在地了。
<br/>　　“你敢打我？”
<br/>　　吴倩倩瞪大了眼睛万万没有想到林峰敢忽然对她出手。
