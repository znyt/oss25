　　“紫炎，那东西你不能拿。”慕容克着急地吼道。
<br/>　　“哦，是吗？”萧逸冷冷一笑，一把将乾坤戒握入手中。
<br/>　　所谓乾坤戒，其用途和乾坤袋一样，不过它的空间更大。
<br/>　　低级乾坤袋，空间只有数立方米。
<br/>　　中级乾坤袋，空间十数立方米。
<br/>　　高级乾坤袋，空间一百立方米。
<br/>　　乾坤戒，空间达一千立方米。
<br/>　　低级乾坤袋，是大部分武者的标配；中级乾坤袋，一般要家族和势力才会拥有。
<br/>　　高级乾坤袋，已经是稀有货了。
<br/>　　哪怕是萧逸，也不过拥有两个高级乾坤袋。
<br/>　　一个是易老赠送的，一个是之前在盛宝商行抢的。
<br/>　　他手头上大批的天材地宝、妖兽精血、妖兽内丹，单凭两个高级乾坤袋，根本放不完。
<br/>　　所以，他腰间还别着十几个低级和中级的乾坤袋。
<br/>　　乾坤袋，虽然只有巴掌大小。
<br/>　　但十几个都挂在腰间，即使有衣服外套遮掩，不至于不雅，但还是会感觉非常不舒服。
<br/>　　现在有着乾坤戒，一个，就能顶十个高级乾坤袋。
<br/>　　一个乾坤戒，绝对能把身上大批的东西全部装完，加之还有剩余的空间。
<br/>　　萧逸淡淡一笑，将乾坤戒戴于手上，而后转身离开。
<br/>　　“紫炎，你不能拿走乾坤戒。”慕容克急了。
<br/>　　乾坤戒，传说乃是远古时期留下来的东西。
<br/>　　如何制造，已经无人知晓。
<br/>　　现今，哪怕是高品炼药师，手段再过人，也无法炼制出。
<br/>　　所以，它珍贵无比。
<br/>　　只有那些真正的大势力才拥有。
<br/>　　整个北山郡，也就裂天剑派、暗影楼、猎妖殿，还有北山郡王拥有。
<br/>　　连白家这样的势力都没有。
<br/>　　据说，北山郡王也不过拥有一枚。
<br/>　　而北山慕容家，本身也只有一枚，但由于是商业家族，不知从哪又得到了一枚。
<br/>　　其中一枚，放在郡都盛宝商行总行；另一枚，便是东荒十八城盛宝商行分行。
<br/>　　本来，萧逸只想来将商物都拿走，教训慕容克一顿。
<br/>　　但现在，知道乾坤戒的存在，他便必拿不可。
<br/>　　若他拿走了乾坤戒，整个北山慕容家都不会过他。
<br/>　　他，正是想要这个结果。
<br/>　　下一秒，他的身影已经消失在原地。
<br/>　　盛宝商行的商物，被他悉数取走；但商行内的银两，他却分毫未拿。
<br/>　　他有着自己的打算。
<br/>　　萧逸向来心思缜密，做事，从不会无的放矢。
<br/>　　……
<br/>　　离开盛宝商行，萧逸化作紫炎火翼。
<br/>　　虽说如今已能御空飞行，但有紫炎火翼的增幅，他的速度会更快。
<br/>　　他正往第二道防线那些大城的郊外而去。
<br/>　　上一次，他便是在那里突破修为。
<br/>　　一个时辰后，他回到之前那个山洞。
<br/>　　在洞口做好掩饰，他立刻开始拿出材料，炼丹。
<br/>　　在这之前，他先检查了一遍这次的收获。
<br/>　　天材地宝，不计其数。
<br/>　　四级内丹、精血，各一千。
<br/>　　三级内丹、精血，过万之数。
<br/>　　亚灵器，两个。
<br/>　　还有一些杂七杂八的东西。
<br/>　　东荒十八城，盛宝商行分行，拥有的商物，远比其他分行要多。
<br/>　　洞内，离火金炉现，紫炎凝，炼丹正式开始。
<br/>　　离开萧家以来，他通过各种途径得到的天材地宝，太多了，根本不必担心不够。
<br/>　　他率先炼制了三千粒完美级别的四级丹药。
<br/>　　都是些巨力丹、生灵丹，等可以作为硬通货的丹药。
<br/>　　顺便，他还炼制了数十粒五品丹药，回天丹。
<br/>　　回天丹，乃是疗伤圣药；加之位列五品，又是完美级别，绝对价值不菲。
<br/>　　之后，他才正式开始炼制他自己的无杂质丹药。
<br/>　　七天后。
<br/>　　两千颗四级妖兽内丹，炼出了一千粒无杂质洞玄丹。
<br/>　　二十多万颗三级妖兽内丹，炼出了十一万粒无杂质先天丹。
<br/>　　还有两粒无杂质破玄丹。
<br/>　　虽说他炼药的速度快，但现在丹药的基数太大了。
<br/>　　自然消耗了他不少时间。
<br/>　　一粒粒丹药服下。
<br/>　　冰鸾剑、控火兽，双武魂齐开。
<br/>　　药力，几乎顷刻间就化为真气，全部储存于气泉。
<br/>　　一天后，当他服下三万粒无杂质先天丹时。
<br/>　　气泉，填满至两成。
<br/>　　修为，瞬间突破洞玄二重。
<br/>　　两天后…
<br/>　　三天后…
<br/>　　每当他服下三万粒无杂质先天丹，气泉便会填满一成，修为突破一重。
<br/>　　体内气泉的真气高度，上升三百三十三丈。
<br/>　　三天时间，消耗了9万粒先天丹，恰好突破了三重。
<br/>　　第四天，他将剩下的两万粒无杂质先天丹，一千粒无杂质洞玄丹，通通服下。
<br/>　　修为再升一重，达到洞玄五重。
<br/>　　体内气泉的真气高度，达到一千六百多丈。
<br/>　　之后，将两粒无杂质破玄丹服下。
<br/>　　这一次，他并未能突破修为。
<br/>　　破玄丹，蕴含的力量比洞玄丹多得多。
<br/>　　但只有两粒，还不足以突破。
<br/>　　“洞玄五重。”萧逸握了握自己强有力的拳头，一股自信的豪迈气息，渐渐涌现。
<br/>　　他的修为，虽然突破得快。
<br/>　　但，天知道他所耗费的丹药加起来，价格恐怖到什么地步。
<br/>　　也就是他萧逸。
<br/>　　要是换了别人，就算有大势力支持，也休想得到这么多丹药。
<br/>　　先不说大势力有没有这么多材料和妖兽内丹。
<br/>　　就算有，也没有炼药师能有这个本事炼丹速度如此快。
<br/>　　无杂质丹药，就算是懂炼药五行录的高品炼药师，一天能炼出个百来粒都算了不起的了。
<br/>　　他们，往往要在材料和内丹属性的组合上，思索很久。
<br/>　　洞内，萧逸修为突破洞玄五重，实力大增。
<br/>　　却并未离去。
<br/>　　他还有四级妖兽精血两千。
<br/>　　三级妖兽精血二十多万。
<br/>　　接下来，才是他最期待的事，突破修罗战体。
<br/>　　他现在的肉体力量，达到半步破玄。
<br/>　　也就是修罗战体第二层圆满。
<br/>　　接下来，要突破到第三层。
<br/>　　直觉告诉他，这需要极多的妖兽精血。
<br/>　　上一次，他还留有很多妖兽精血，也未服用。
<br/>　　就是在等今天，一举突破。
<br/>　　这一次，他率先服用四级妖兽精血。
<br/>　　待得所有四级妖兽精血服用完。
<br/>　　他明显感觉到，修罗战体，出现突破的征兆了。
<br/>　　同一时间，他的体内，心脏位置，出现了一个诡异的图案。
<br/>　　图案看不懂是什么，但充满了玄奥。
<br/>　　当他开始大量吞服三级妖兽精血时。
<br/>　　修罗战体，迟迟不见突破。
<br/>　　反倒是这个诡异图案，蒙上了一层诡异的血色。
<br/>　　妖兽精血的力量，仿佛在慢慢填充这个图案。
<br/>　　待得近十万三级妖兽精血吞服完。
<br/>　　图案，才填满了一半。
<br/>　　“好诡异的图案。”萧逸眉头一皱，“果然没有错，修罗战体第三层，有古怪。”
<br/>　　据狂血玄君的记录显示，第三层，一旦突破。
<br/>　　实力，将会有一个质的飞跃。
<br/>　　待得将剩下的十多万瓶三级妖兽精血一一服完。
<br/>　　图案，才算完全填满，充斥着一股骇人的血色之光。
<br/>　　只不过，现在，第三层仍旧没有突破。
<br/>　　萧逸对自己肉体的感觉很清楚，距离突破，只剩下最后一层屏障。
<br/>　　屏障，如纸般脆弱，一捅便破。
<br/>　　两瓶五级妖兽精血入腹。
<br/>　　屏障瞬间被破。
<br/>　　修罗战体，第三层，终成。
<br/>　　肉体力量，达到破玄一重。
<br/>　　同一时间，图案上，传来一阵莫名的信息。
<br/>　　“修罗之力？”萧逸的脸色，陡然间变得凝重无比。
<br/>　　但下一秒，又变为了狂喜。
<br/>(本章完)
