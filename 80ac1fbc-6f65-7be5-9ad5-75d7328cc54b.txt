　　泰山之上，伏羲禅位于神农，证道天皇，无尽功德落下，他借此机会，一举融合三道神轮，欲突破极道帝境。
<br/>　　浩大的八卦虚影惊动天地，无尽道韵流转不休，大道气息弥漫，天机显现，气势惊人。
<br/>　　“嗯？伏羲果然在冲击半步混元之境？”诸圣一个个目光有些惊诧，看着半空中的伏羲，想要出手阻止，却又无法插手。
<br/>　　人族天皇证道，乃是天地大势，如若出手阻止，便是逆天而行，圣人乃是天道的维护着，哪敢逆天行事啊？圣人中唯有元始与女娲两人面色不变。元始是毫不在意，就算伏羲突破混元，依旧不够他看，他倒是希望洪荒之中多出现几个逆天之人，那样洪荒便热闹了。
<br/>　　更何况，洪荒之中，强者越多，天道的力量便会被无形之中削弱，那样对元始绝对有好处，但对于圣人却有了限制，因为圣人能够借助天道作战，天道之力被削弱了，他们能够借到的力量便会随之削弱，这是圣人不愿意看到的。
<br/>　　而女娲则是只有兴奋，伏羲能够突破，对她来说也是极好的。
<br/>　　至于其他圣人，便没有元始的淡然与女娲的兴奋了，他们脸色纷纷变得阴沉下来，不希望伏羲能够突破。只可惜，结果却让众人失望不已，不久，伏羲便将三道神轮融合，融合成一道白金色的浩大神轮，有八卦的道韵扩散开来，也有绝妙的琴音从里面若有若无的传了出来。
<br/>　　轰隆！
<br/>　　一阵轰响，只见那伏羲悬浮于空，背后神轮浮现，全身气势滔滔，修为攀升，让风云倒卷，日月失色，苍穹巨变，惊天动地。
<br/>　　“半步混元，伏羲竟然真的……突破了？”大神通纷纷惊骇。
<br/>　　“竟然真的成功了，此子真是好运！”
<br/>　　伏羲还想将三大洞天合一，可惜，他身上剩余的功德之力已经不够了。伏羲无奈，只好将剩余的功德打入背后的神轮之中。
<br/>　　嗡！
<br/>　　一阵声响，在吸收了剩余的功德金光后，那道神轮立即变得金光闪闪，由之前的虚幻道轮变为如今真实的神轮，一道神轮悬浮，无尽金光绽放，诸邪辟易，，神魔退避，万法不侵！
<br/>　　“咔擦！”
<br/>　　伏羲全身气势大盛，突然，在他的脑海深处，一阵清脆的响声响起，在他的命格深处，好似有什么被打开了，一道虚幻的身影出现在他脑海深处，那道身影人首蛇身，面目与他竟有七分相似。
<br/>　　“你终于来了！”就在此时，那道虚影开口说道。
<br/>　　“你也是伏羲？！”伏羲开口问道，虽是疑问，但声音之中，却带着一股肯定，其实他心中早有猜测，如今见到这一幕，不至于那般慌乱。
<br/>　　“不错，我也是伏羲，你比我想象的要镇定许多。”只听那虚影开口说道。
<br/>　　“你一直在我身体之中？”伏羲问道。
<br/>　　“不，应该说，是你一直在我的身体之中才是！”只听那虚影开口笑道。
<br/>　　“我不就是你吗？”
<br/>　　伏羲听了此话，脸上没有丝毫的慌乱，他面色淡然，对自己极为自信，当即开口反问道。
<br/>　　“呃？不错，哈哈！我就是你！哈哈！”
<br/>　　那虚影突然一愣，紧接着便大笑起来：
<br/>　　“我是你，你也是我，你我都是伏羲！但同时，我也不是你，你也不是我，你我只是诸天万界之中的两朵相似的花而已。”
<br/>　　“是啊，你我只是命运轨迹相似而已。但终究是要合一的。”伏羲深深的感叹道。
<br/>　　“是啊，终究是要合一。相传，当修士达到一定境界之时，便要诸天合一，万我归一，唯我唯一，天地不留痕，大道不能录！”只听那虚影如此说道。
<br/>　　“是的，诸天万界，有无数个我，也有无数个你！你我他都是大道之下无数相似的花而已，而且彼此之间，都有因果联系，想要真正超脱大道彼岸，便需要万我唯一，你我他合而为一，诸天唯一，万界唯一，唯我唯一！”
<br/>　　“而你我，只不过是因为一个意外早一步合一了。”伏羲说道。
<br/>　　“不错，只是，合一之后，我还是我？你还是你吗？”虚影感慨了一声，语气之中，带着深深的无奈，那是对大道无涯的感叹。
<br/>　　闻言，伏羲也沉默了。一会儿后，伏羲再次开口说道：
<br/>　　“我便是我，不会因你而散，不会因他而逝，不论怎么融合，我只会是我，也只能是我！”说道这里，伏羲眼中流露出无比坚定的神色，那是对大道的执着，对自己的自信。
<br/>　　虚影看了一会伏羲，叹息一声，说道：
<br/>　　“唉，也罢，既然如此，那你我便合一吧，这是迟早的事情，只是，希望你能够成功超脱大道彼岸！”
<br/>　　“融合之后，你不就是我了吗？”伏羲突然开口说道。
<br/>　　“呃？哈哈，不错，我即是你，你即是我，原来我终究没有看透啊！你登临彼岸，不就是我登临彼岸吗？哈哈！”虚影突然大笑起来。
<br/>　　突然，他身上光华大盛，全身上下，顿时荡起一股无形的波动，他的身影，像是被风吹散了一般，无尽光点飞出，融入到伏羲身上，他的身影也渐渐开始变得暗淡起来。
<br/>　　“希望你好好照顾女娲！”
<br/>　　虚影突然一笑，紧接着他全身化为金色光点融入到伏羲身上。伏羲脑海之中，顿时多了无数信息，那是洪荒伏羲的一生。从出生，到死亡，一点一滴，对道的领悟，对女娲的感情，对妖族之情，对万物的见解，对……
<br/>　　那虚影的一切，全部融入到伏羲的命格之中，化为他的一部分，那些事情，就仿佛是他亲自经历过的一样，虚影对道的领悟，也仿佛是他自己领悟的一般，不分彼此。
<br/>　　这一刻，虚影便是伏羲，伏羲也是虚影，两者合而为一了。
<br/>　　神奇！
<br/>　　真是神奇！
<br/>　　轰！
<br/>　　伏羲全身一震，顿时，一道更强的气势从他身上散发出来，伏羲体内，天、地、人三大洞天，一瞬间便合而为一，化为一方浩大的人道世界。
<br/>　　他脑后那吸收了功德金光的神轮，也在这一刻瞬间没入新世界之中，随着一同没入的，还有他的天地法相。
<br/>　　一瞬之间，他便突破的世界境，成为除了武祖玄天之外的另一个武道帝境之修，这一境，可称之为“武帝”！
<br/>　　立于虚空之中的伏羲，突然睁开眼睛，眼中闪过一丝精芒，他全身的气势一下子便被他收拢起来，不再释放出任何威严，让人看上去，就仿佛是看到一个身披麻衣的凡人。
<br/>　　那是达到返璞归真的境界的表现，全身气势能够收放自如，对自己的掌控，达到了一个惊人的程度。
<br/>　　伏羲是证道人道天皇，就如同证道天道圣人一般，是没有天劫的，因为天劫还管不了同等级的事物，如，天道圣人，人道大帝，地道神皇等。是故，此时天上并没有降下灾劫。
<br/>　　伏羲突破极道帝境，一步一步地向着地面走来，同时，一道浩大而恢宏的歌声，传遍洪荒三界，惊动九霄：
<br/>　　“一线因果命中定，两世伏羲梦里醒！
<br/>　　三生三世人情故，破天见我我非我！
<br/>　　过去种种皆云散，未来重重道可期！
<br/>　　一朝证道圣天皇，两我合一帝伏羲！”
<br/>　　……
