        连华生顾不上那么多，径直离开了实验室基地，打算去找夏阳枭将童筱颖要回来。
<br/>
<br/>        可是当他马不停蹄地赶到夏阳枭的办公室里，找夏阳枭对峙的时候，夏阳枭却在一边凝眉打电话，一边发号施令：“一定要将童筱颖安全带回来！”
<br/>
<br/>        夏阳枭挂断电话后，看到连华生一点都不意外，反而调侃道：“这不知道的人，还真会误以为这童筱颖是你连华生的女人。”
<br/>
<br/>        “夏局，你答应过我和阿沉，不动童筱颖的！”连华生紧握双拳，语气里带着一丝怒火。
<br/>
<br/>        夏阳枭凝望着面前的连华生，严肃道：“现在不是我动不动她的问题，而是，我手下的人在接她回来的途中，她所坐的车，被一群黑衣人劫持了。”
<br/>
<br/>        “你说什么？”连华生惊愕，“被谁劫持了？你们查到了吗？”
<br/>
<br/>        “冒险者集团的人。”夏阳枭皱起了眉头，语重心长道，“看样子，他们也在找她。”
<br/>
<br/>        “你为什么要带走童筱颖？她压根就什么都不知道！你们带走她，就算对她用刑也问不出什么来！”连华生一边质问着，一边替维护童筱颖。
<br/>
<br/>        夏阳枭深沉地回答道：“我派人带走她，不是想从她身上知道什么，而是她……”
<br/>
<br/>        “她什么？”连华生追问。
<br/>
<br/>        但夏阳枭欲言又止，不得不顾左右而言他：“现在还没有答案。如果，我的人早些把她带回了，说不定，就能有答案了。但是，她却被冒险者集团的人劫走了！”
<br/>
<br/>        连华生却很恼火地将双手捶在夏阳枭的办公桌上：“你若是不派人将她接走，或许她就不会被冒险者集团的人劫走。她被劫走，也恰恰说明了，我们局里有他们冒险者集团的卧底。”
<br/>
<br/>        “我已经派人一路跟着被劫持的车了，人是从我这里被劫走的，我定会将她救回来。”夏阳枭允诺。
<br/>
<br/>        连华生接着说道：“希望夏局言而有信！”
<br/>
<br/>        语毕，连华生站直了身子，恼火地转身离去。
<br/>
<br/>        夏阳枭看着连华生离去的背影，只是皱着眉头，紧抿着嘴唇。
<br/>
<br/>        看样子，接下来必定会有一场暗战即将打响。
<br/>
<br/>        另一边。
<br/>
<br/>        一辆黑色的轿车被两辆越野车逼停在了隧道里，童筱颖被两个黑衣人从轿车里拽了下来，被押进了另一辆银色的保姆车内。
<br/>
<br/>        当她在保姆车内看到同样被绑架的落以柔时，瞬间便明白是怎么一回事了。
<br/>
<br/>        起初，她在那个实验室基地里抽血做检查，检查结果出来时，那名女医生喜上眉梢地跟他们上级汇报说：“我们终于找到她了！”
<br/>
<br/>        然后，她就被带到了一辆豪华轿车内，陪同她的女医生说是要带她去见他们的领导。
<br/>
<br/>        只是在这半路，就遇到了这帮持枪的土匪，劫持了他们的车子。
<br/>
<br/>        很显然，他们是冲着大哥傅夜沉来的。
<br/>
<br/>        落以柔的手脚都被白色的警用塑料手铐束缚，嘴上还贴着一块黑色的塑料胶布，束成马尾的头发有些凌乱，两眼泛红像是刚哭过。
<br/>
<br/>        她看到童筱颖时，一直“呜呜”直叫。
<br/>
<br/>        童筱颖刚想凑过去替落以柔撕掉嘴上的那块黑色胶布时，一把手枪直接抵在了她的太阳穴上。
<br/>
<br/>        “婴宁的妹妹，别来无恙。”
<br/>
<br/>        一个阴冷的声音，在脑后响起。
<br/>
<br/>        童筱颖循声微微偏头。
<br/>
<br/>        只见八爷的脸上仍旧戴着那副黑色的假面眼镜，而假面眼镜的两个眼洞里，那双深遂的瞳仁透着一股戾气。
<br/>
<br/>        “你知不知道，出卖我，是什么下场？”八爷咧嘴道。
<br/>
<br/>        童筱颖不寒而栗，心里因为紧张，而额冒冷汗。
<br/>
<br/>        “要不是我暗中替换了何明旭手里的东西，我还真不知道，你是我们古堡里的内鬼。”八爷冷冷地说道。
<br/>
<br/>        童筱颖忽然间暗自松了口气。
<br/>
<br/>        幸好，他们没有怀疑到大哥身上。
<br/>
<br/>        “哼，你一个装病，就引来安防局的人灭了我们的实验基地，又查封了我的古堡，害得雅珊被抓。我还真没想到，你这个女人，什么也不会，却有这么大的杀伤力。”八爷恼火道。
<br/>
<br/>        童筱颖深吸了口气，淡定地说道：“如果是一个什么都会的女人，肯定会让你发现破绽，刚好我这个什么都不会的女人，才会让你毫无戒备。”
<br/>
<br/>        “雅珊拿你当朋友，你却背叛雅珊。你这个女人，真恶毒！”八爷义愤填膺道，“如果不是看在阿十以及你是婴宁的妹妹的份上，我现在就一枪毙了你！”
<br/>
<br/>        随后，八爷将枪收了起来。
<br/>
<br/>        童筱颖再次瞥向落以柔，明知故问，试探性地问道：“这是八爷的新欢吗？”
