<br/>    经过雪山宗的传送阵，陈明和紫霞仙子等人来到了群星仙门，经过一层层的楼阁，陈明等人终于是来到了群星仙门的议事殿之外，雪山宗长老不过进去片刻，便又出来了，“诸位老君请诸位速速进去议事。”
<br/>    
<br/>        雪山宗长老毕竟是大君，已经看得出来，紫霞仙子说话之间很顾忌陈明，看来陈明的地位也不低，所以说的是议事，而不是禀报。
<br/>    
<br/>        议事是平等的，禀报则是下见上。
<br/>    
<br/>        陈明等人，踏步进入其中，只见其中星辰璀璨，楼顶之上有着一片璀璨的星空，一缕缕星光落入其中。
<br/>    
<br/>        陈明看了紫霞仙子一眼，紫霞仙子心领神会，直接上前一步，拱手道：“启禀诸位老君，此次我等进入九幽寒泉之中，有着魔宗布置传送阵，魔宗大君解千愁带着二十多王级军团前来伏杀！”
<br/>    
<br/>        雪山老君当时就怒了，“什么，魔宗竟然敢如此行事，他是要灭掉我仙门年轻一代不成！”
<br/>    
<br/>        群星老君也是怒道：“魔宗，胆敢！”
<br/>    
<br/>        铁衣老君却是道：“若是如此，你们既然出来禀报消息了，我们速速发兵，我们会派遣三位大君前去斩杀魔宗之人！”
<br/>    
<br/>        紫霞仙子道：“不必了，在九幽寒泉之中，当时情况危急，索性燕山宗师正在突破边缘，杀人证道，杀了两百余魔王，已经成功突破到了大君的领域，有了宗师对抗魔宗魔君解千愁，我等才得以保全，之后，宗师为了让我等能够尽快突破到大君，击败解千愁，以还魂水为材料，炼制了阴阳和合丹，我早在之前就已经完成了杀人证道那一步，可以突破到大君，在我渡劫之时，解千愁进入天劫干扰我渡劫，想要将我斩杀在天劫之中，有赖宗师不顾生死，在天劫之中出手，斩杀了解千愁，我才得以突破到大君，如今，九幽寒泉之中已经安全了。”
<br/>    
<br/>        四位老君都是看向陈明，群星老君道：“这位道友，就是和我们合作的燕山之主吧？”
<br/>    
<br/>        陈明扫了一眼四位老君，全是问号，算了，不看了，拱手道：“不错，正是在下。”
<br/>    
<br/>        群星老君继续问道：“怪不得能够教出卓青瑶这般天才，原来道友竟然有着如此实力，那么，如今悟道榜上的第一，可是道友？”
<br/>    
<br/>        陈明道：“若是那名字是陈明，那么就是在下了，在下修行的功法有些奇特，所以实力有所不同，不能用境界论，就如同小徒卓青瑶一般。”
<br/>    
<br/>        四位老君也是见多识广，对于这一点并没有什么疑惑，雪山老君道：“这有何妨，道友也是大君的实力，道友又和我们古仙门交好，封君以及封号之事，就交给我们古仙门了。”
<br/>    
<br/>        群星老君道：“现在魔宗既然已经对我们出手，那么就不能按照之前的战略继续下去了。魔宗介入，那么现在就是一场席卷整个大陆的战争。不过现在，我们首先要谢过陈明道友，若不是陈明道友，恐怕我仙门正道，年青一代杰出之辈，都要死伤殆尽，三位道友，我等一起谢过陈明道友力挽狂澜。”
<br/>    
<br/>        四位老君起身，朝着陈明弯腰拱手道：“多谢陈明道友力挽狂澜！”
<br/>    
<br/>        其他的悟道榜上的天才，看到四位老君对陈明行如此大礼，都是心头一震，这可是四大古仙门的四位老君啊！
<br/>    
<br/>        陈明现在简直骂娘的心都有了，这四个老家伙也太不要脸了吧，我为了这群小兔崽子，炼丹不分昼夜，厮杀不畏生死，紫霞仙子的身上，还砸了两千万灵石的丹药呢，你们一句谢谢就完了？
<br/>    
<br/>        以后要是问报酬，他们不就是一句，不是已经谢过了吗？四位老君给你弯腰行礼，还不够？
<br/>    
<br/>        果然人老成精，陈明也不好看着四个老家伙给他行礼，连忙将四位老君扶起来，“四位老君，折煞在下了。”
<br/>    
<br/>        就在这时，陈明的脑海中忽然想起了一个声音，“叮，李素衣已经突破到神府境，你获得了5000师德。”
<br/>    
<br/>        嗯，这是怎么回事？
<br/>    
<br/>        李素衣的实力，怎么不受限制的增长了？
<br/>    
<br/>        燕山，绝对是出了大事了。
<br/>    
<br/>        陈明道：“不知群星老君，可否将悟道榜给在下一观。”
<br/>    
<br/>        群星老君笑了笑，挥手道：“小事而已，三个榜单都给你一观吧。”
<br/>    
<br/>        头顶的星空之中，出现三个榜单，陈明分明地看到，李素衣的名字，已经是出现在了悟道榜之上，众多老君也是抬头看向悟道榜，陈明名列第一，而一个新的名字，已经是出现在了悟道榜上，李素衣。
<br/>    
<br/>        群星老君道：“这李素衣到底是何人，以前便是入道榜第二，不过短短几日之间，此刻已经是登临到了悟道榜上，修为增长之快，简直闻所未闻，此子，妖孽之极。”
<br/>    
<br/>        陈明道：“这是我那三弟子。”
<br/>    
<br/>        四位老君，各大仙门，宗门的天才都是看向陈明，这是你三弟子？
<br/>    
<br/>        燕山还真是个个都是怪物！
<br/>    
<br/>        群星老君忽然有了一个大胆的猜测，“那不知这如今的入道榜第一，陈灵鱼，还有那入道榜最后一名，凌仙，道友可知道又是何人？”
<br/>    
<br/>        现在反正燕山都扬名整个大陆了，陈明也不怕了，“陈灵鱼，乃是我门下四弟子，至于那凌仙，乃是我门下二弟子，凌仙修行的是《道经》，所以修为提升缓慢，若是有朝一日他突破到悟道境，若是我不在榜单之上，那么他应该就是悟道榜第一了。”
<br/>    
<br/>        周围所有人此刻浑然色变，这到底是一个什么样的妖孽门派啊，个个都是天赋惊才艳艳之人！几乎是血洗了整个榜单。
<br/>    
<br/>        不过现在陈明的脸上，却是没有任何血色，李素衣的修为提升如此之快，绝对是彻底入魔了。
<br/>    
<br/>        群星老君问道：“道友，李素衣乃是你的门下，修为提升乃是好事，为何道友愁眉苦脸的？”
<br/>    
<br/>        陈明道：“我那三弟子，恐怕会造就一场波及整个大陆的血祸，本来我还能留在此处一段时间，现在看来，燕山已经出了大事了，恕在下，先告辞了，我必须回到燕山，料理完十万大山的事。”
<br/>    
<br/>        陈明直接带着卓青瑶离开了此处，赶回燕山去了。
