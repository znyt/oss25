<br/>				经过曹坤的一番解释，以及其余众人的补充，叶尘很快便明白了事情的来龙去脉。<br/>　　<br/>　　原来，就在今天早上，云州的各方势力的巨头大佬们，在一觉醒来之后，现床头出现一张请柬和一把匕！<br/>　　<br/>　　请柬上写道，邀请众人今天晚上，在云州与林州交界位置的望江楼一聚，一起商讨今后的合作事宜，如若不到，后果自负！<br/>　　<br/>　　而落款，赫然是云州大佬何进！<br/>　　<br/>　　很显然，何进故意搞出这么一出，显然是在威慑云州众人，而吞并云州之心，已经昭然若揭。<br/>　　<br/>　　“我本以为这家伙，至少要等到武林大会上才会动手，没想到他如此心急，而且胃口如此之大，居然要将云州所有黑白两道的势力，全都一口吞下！”<br/>　　<br/>　　曹坤将事情的经过描述了一遍之后，又忍不住一脸恨恨道。<br/>　　<br/>　　唐业在一旁则幽幽叹道：<br/>　　<br/>　　“据说何进这次，是从海外招揽过来的高手，名字叫于成雄！此人在海外华人武林界，名气相当之大，传言已经达到化劲大成境界，而且为人狂傲霸道，做事肆无忌惮，只怕未必会被何进左右，因而这件事多半是此人的主意！”<br/>　　<br/>　　唐业说完，一名老者从人群站了出来，插口道：<br/>　　<br/>　　“不管是谁的主意，如今咱们已经成了一条绳上的蚂蚱，必须要共进退才行！”<br/>　　<br/>　　“不错！对方既然能神不知鬼不觉的，在我们的床头放下匕，想要取我们的性命，简直易如反掌！”<br/>　　<br/>　　“是呀，此人若是一日不除，咱们必定寝食难安！”<br/>　　<br/>　　……<br/>　　<br/>　　众人一阵议论纷纷之后，忽然一名中年男子站了出来，冲着叶尘拱手行了一礼道：<br/>　　<br/>　　“叶先生，听曹四爷和唐老爷子说，您也是化劲宗师，不知道和于成雄相比，孰强孰弱？”<br/>　　<br/>　　这中年男子虽然嘴上说的恭恭敬敬，但是眼眸之中，已经闪过一丝失望之色。<br/>　　<br/>　　这也难怪！<br/>　　<br/>　　在场的众人中，唯独曹坤、曹文、唐业等寥寥数人，曾经见过叶尘出手，哪怕是他们几人，也只是惊鸿一瞥，对叶尘的实力并不怎么了解。<br/>　　<br/>　　即便在他们几人想来，叶尘毕竟太过年轻，纵然再天才，只怕也不过是刚刚踏入化劲不久，根本无法和于成雄这样，成名已久的化劲大成强者相提并论！<br/>　　<br/>　　因而，就连曹坤和唐业他们几人，都不怎么看好叶尘，更不要说其他人了，自然更加不看好。<br/>　　<br/>　　若非他们实在找不到可以抵挡于成雄之人，只怕也未必会来到叶尘的门前苦等。<br/>　　<br/>　　叶尘将众人的神情看在眼中，心中不由得一阵冷笑，若非和曹坤、唐业等人交情还算不浅，此刻早就拂袖而去了。<br/>　　<br/>　　既然对我没信心，那还来找我做什么？<br/>　　<br/>　　“我如果说，我比那个于成雄要强，你们信吗？”<br/>　　<br/>　　叶尘似笑非笑道。<br/>　　<br/>　　众人一阵默然，唯独吴磊想要站出来，却被吴伯雄一把拉住。<br/>　　<br/>　　见众人不吭声，叶尘不由得笑了，又道：<br/>　　<br/>　　“那如果我说，我不如于成雄，那你们又准备作何打算？”<br/>　　<br/>　　众人再次沉默了片刻，最终有人重重一叹道：<br/>　　<br/>　　“那我们也只好把家业，拱手让给那何进了！毕竟，保命要紧嘛！”<br/>　　<br/>　　叶尘听到这话，脸上的笑容顿时更浓，不过却是冷笑，<br/>　　<br/>　　“合着你们来这里找我，只是把我当做你们的挡箭牌呀！我若是帮你们打败于成雄，你们便能保住家业，我若是败在于成雄的手上，你们就投靠对方，那于我而言，又有什么好处？换句话来说，你们凭什么让我冒着生命危险，去帮你们保住家业？”<br/>　　<br/>　　“这……”<br/>　　<br/>　　众人被叶尘一通质问，顿时全都满脸尴尬的说不出话来。<br/>　　<br/>　　他们本以为，叶尘虽然武功高强，但毕竟年轻、阅历浅，应该没有多少城府，却没有想到他居然看的如此透彻！<br/>　　<br/>　　最终，还是唐业率先开口道：<br/>　　<br/>　　“叶先生若是能保住我唐家家业，我唐家以后就以先生为尊！”<br/>　　<br/>　　曹坤见状，也连忙跟着道：<br/>　　<br/>　　“我曹家今后，唯先生之命是从！”<br/>　　<br/>　　其余众人见唐业和曹坤两大巨头都开了口，顿时也跟着纷纷表态，<br/>　　<br/>　　“我张家今后也以先生为尊！”<br/>　　<br/>　　“我钱家……”<br/>　　<br/>　　……<br/>　　<br/>　　等到众人表完态，叶尘这才满意的点了点头，<br/>　　<br/>　　“好了！既然你们愿意奉我为尊，那我自然有义务保护你们，晚上你们只管前去赴宴，到时候我自有计较！”<br/>　　<br/>　　众人这话，顿时一阵愕然，想开口询问，但又没有那个胆子，只好纷纷告退。<br/>　　<br/>　　不一会，众人全都散去，唯独唐业、曹坤、古纯阳师徒和吴磊父子，被叶尘留了下来。<br/>　　<br/>　　“叶师，那于成雄可是化劲大成的高手，您当真有把握对付他吗？”<br/>　　<br/>　　说话的是古纯阳，他是道门中人，并不在何进的那个受邀名单之列，因而算是局外人，所以说话不必顾忌太多。<br/>　　<br/>　　唐业、曹坤等人，听到古纯阳问出了他们心中的疑问，顿时全都一脸期待的望向叶尘。<br/>　　<br/>　　不料，叶尘淡淡一笑，身上升起一股强大的自信，缓缓道：<br/>　　<br/>　　“区区化劲大成而已，我杀他如屠狗！”<br/>　　<br/>　　众人听到这话，顿时全都变得激动起来，<br/>　　<br/>　　他们对叶尘的性格都还算了解，知道眼前这个少年，虽然有的时候狂傲不羁，但是在这种事情上，还不至于故意夸大其词，毕竟这可是关系他自己的身家性命。<br/>　　<br/>　　叶尘说到这里，微微一顿，接着又望向唐业和曹坤，道：<br/>　　<br/>　　“唐老爷子，曹四爷，我打算趁这次机会，将云州的势力整合一下，顺便把林州何家的地盘也夺过来，你们以为如何？”<br/>　　<br/>　　唐业和曹坤听到这话，先是一愣，继而纷纷露出狂喜之色。<br/>　　<br/>　　云州这些年来，一直是天南省十个州市里面，势力最为混乱的一个，唐、曹、林三家鼎足而立，相互牵制，下面还有一些小势力各自为政，也正因为如此，所以才经常遭到其他州市势力的觊觎。<br/>　　<br/>　　若是叶尘真的能将这些势力整合在一处，然后带领他们冲出云州，那么他们的家族势力，将会取得爆式的展，甚至一跃成为整个天南省的一流家族！<br/>　　<br/>　　一想到这里，唐业和曹坤的眼神，顿时变得越炙热起来。<br/>　　<br/>　　第三更！<br/>　　<br/>　　(本章完)<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/>
