　　第136章
<br/>　　“开门！”
<br/>　　“开门！”
<br/>　　小蝶卖力敲门，粉拳重重捶在门上。
<br/>　　“我把你个没人管的小娘皮，落在爷的手中，爷打得你屁股开花。”
<br/>　　门内传来苏忠粗鲁的声音，他将门拉开一条缝，吼道：“找死嘛！”
<br/>　　程大雷抓着一块石头，直接闷在他头上，苏忠身子晕倒在地上。
<br/>　　程大雷和小蝶踩着他的身体走进大门。
<br/>　　三四个门房已经被惊动，看到晕倒在地，满脸血污的苏忠，顿时吓了一跳。
<br/>　　程大雷操起门口的顶门杠，一下一个全部撂倒。
<br/>　　“带我去地窖！”
<br/>　　小蝶也没料到程大雷会直接闯进来，她抓紧时间带着程大雷往地窖走。过程中有护院家丁冲过来，被程大雷干脆放倒。
<br/>　　苏家很大，如果不是小蝶带着，程大雷很容易在其中迷路，更莫提找人。
<br/>　　当小蝶带着程大雷往地窖去的时候，围上来的人越来越多。程大雷出手稳准狠，大棒底下不知砸倒多少人。
<br/>　　“滚！”程大雷回头冲围过来的人吼道。
<br/>　　“你，你是什么人？”有人问道。
<br/>　　而此时已经有人将程大雷认了出来：“他……他是蛤蟆寨的山贼！”
<br/>　　“听说这人无恶不作，上到九十九，下到八岁小孩，什么人都杀。”
<br/>　　嘟，收到恐惧值688
<br/>　　嘟，收到恐惧值255
<br/>　　……
<br/>　　程大雷分明看到他们脸上呈现出恐惧的表情，虽然有人还大喊大叫：冲呀，杀呀。但他们的脚步分明在后退。
<br/>　　程大雷一扯小蝶胳膊：“走，我们先去找人。”
<br/>　　苏家的地窖就在厨房院子里，地窖是先在院子里挖出地洞，再在里面撑上柱子，过冬时用来存储一些蔬菜和鲜肉。苏家很大，人也很多，地窖就挖得很深很大。
<br/>　　这样的天气，地窖口盖着一块石板，上面还结着白霜。
<br/>　　“小姐，小姐！”
<br/>　　小蝶趴在地窖口大喊，程大雷用力将石板扳开，迈步冲了进去。
<br/>　　“小姐，小姐，我们来救你了。”
<br/>　　小蝶紧随其后，冲进地窖，地窖内黑漆漆一片，可就算凭着仅存的光线，也能分辨出地窖内除了几颗大白菜外，其他什么都没有。
<br/>　　苏樱并不在地窖。
<br/>　　那她会在什么地方？
<br/>　　“程寨主！”
<br/>　　程大雷和小蝶刚走出地窖，就看到院子里站着一个身穿孝衣的温婉女人。
<br/>　　“奴家苏李氏，见过程寨主。”
<br/>　　“你把小姐藏在哪里，快点把小姐放了。”小蝶立刻道。
<br/>　　“程寨主，樱儿失踪我们也很着急，但樱儿确实不在府内，她黄昏就走了，在场许多人都看到了。如果程寨主不信的话，尽可以在府中搜。苏家任何一扇门都为程寨主敞开，包括我的屋子。”
<br/>　　“你敢站在我面前，不怕我杀了你么？”
<br/>　　“想程寨主英雄人物，定不会为难我一个弱女子。”
<br/>　　程大雷打量着李如，她虽然穿着孝衣，却不减半点风韵，更衬得高挑的身材亭亭玉立。
<br/>　　果然，后妈皇后要比白雪公主更有吸引力。
<br/>　　小蝶带着程大雷在苏家能藏人的地方找了一遍，结果却是一无所获，最后回到苏家的灵堂前。
<br/>　　苏家这么大，想藏一个人太容易了。李如究竟把苏樱藏到哪里去了，或者，真如李如所说，苏樱根本不在苏家。
<br/>　　情况复杂了。
<br/>　　“我们现在怎么办？”小蝶现在也没了主意。
<br/>　　“程寨主，我已说过樱儿不在府内，如果程寨主知道樱儿的消息，一定要第一时间告诉我，我心里很担心她的。”
<br/>　　程大雷总感觉李如的眼里闪过一丝笑意，这完全是赤果果的嘲讽。
<br/>　　程大雷真有一棒开了她的意思，可她仍会觉得在智商上碾压了自己。
<br/>　　“我们走！”程大雷一扯小蝶。
<br/>　　“可小姐……”
<br/>　　程大雷抓着小蝶往外走，临走前，他看了一眼李如。
<br/>　　李如眼中的笑意一闪而逝，目光往灵堂上扫了一眼。
<br/>　　程大雷迈出去的脚步顿住，随李如的视线看向灵堂。
<br/>　　灵堂上摆着一口棺材。
<br/>　　“把棺材打开！”
<br/>　　程大雷提着杠子冲向灵堂。
<br/>　　“不行，亡夫尸骨未寒，怎么能开棺，快拦住他！”
<br/>　　程大雷用手中的大棒横扫，一棍敲在棺材盖上，棺盖应声而开。
<br/>　　棺材内，苏樱手脚被绑住，嘴巴勒着白布。
<br/>　　程大雷将苏樱抱出来，扯开她身上的绳索，苏樱此刻已经吓得浑身颤抖，紧紧抓住程大雷不放。
<br/>　　一定是李如知道程大雷进府后，紧急把苏樱转移到棺材里藏住。无法想象，在刚才过去的那段时间里，苏樱在黑暗的棺材里是如何度过的。
<br/>　　程大雷抱着苏樱不放，目光落到李如身上，李如吓得面色苍白，下意识往后倒退一步。
<br/>　　“今日的账，我改天再找你算！”
<br/>　　程大雷抱着苏樱离开，没有在苏家停留，苏樱现在的状况不是很好。
<br/>　　一直等回到小院，喂苏樱喝了一些稀粥，她才渐渐恢复些精神，猛地哭出声来。
<br/>　　能哭出来便是好事，碰到这种事，精神直接崩溃的情况也不是没可能。
<br/>　　“好了，没事了，没事了。”
<br/>　　程大雷没有问苏樱事情的经过，既然已发生这种事，就没有必要让苏樱再回忆一遍痛苦。
<br/>　　苏樱抓着程大雷的衣角，抬起头看着他的眼睛道：“你是个好人。”
<br/>　　“呃……好人卡能不能别乱发。”程大雷有些无语，虽然我是个好人的事情，也是事实啊。
<br/>　　忽然，苏樱抬起头来，堵住了程大雷说话的嘴，一瞬间程大雷如通电般颤栗，脑袋一片空白，如坠云里雾里。
<br/>　　嘟，压寨夫人第四阶段触发：帮苏樱夺回家产。
<br/>　　程大雷一怔，这时候又有新任务来了。
<br/>　　“小姐，不好啦，官兵把院子围住了。”小蝶此时突然闯进屋子。
<br/>　　程大雷走出屋子，只见院门被狠狠撞开，五六十全副装备的官兵闯进院子。他们都穿着全副盔甲，手里拿着长矛。
<br/>　　腾腾腾！
<br/>　　墙上也挤满了人，拉开弓箭瞄准程大雷。
<br/>　　视线拉高，以院子为中心的几条街上，都挤满了官兵。
<br/>　　“把路围住，马车，调马车过来，这几条街都要围住，不要放一只苍蝇出去。”
<br/>　　程大雷看着这一幕，心道：抓我需要这么大阵仗么，这特么足有上千人啊。
<br/>　　“我……”程大雷举起手：“投降。”
