　　所谓丹师宝卷，便是炼丹师通过炼丹师盟会的考核之后，所颁发的一种资格凭证。
<br/>　　丹师宝卷，是由极为罕见的灵铁打造，炼丹师的等级越高，所使用的灵铁就越是稀有，而且宝卷之中，还包含炼丹师盟会长老的精神印记，以及炼丹师的个人信息，所以很难伪造，或者盗用。
<br/>　　那金甲侍卫说完之后，又加了一句，
<br/>　　“如果你真的是四品炼丹师，按照规定，可以带三十名随从！”
<br/>　　众人听到这话，顿时大喜，纷纷朝叶尘望来。
<br/>　　毕竟在他们想来，叶尘之前确实炼制过四品丹药，而且还是四品上等灵丹，所以至少也是四品炼丹师，甚至还要更高！
<br/>　　只要叶尘拿出自己的丹师宝卷，便能带着他们所有人进入云丹城，可以省下一大笔的开销。
<br/>　　感受到众人炙热的目光，叶尘也顿时有些尴尬了，
<br/>　　他踏入修真界，才不过短短数月，而且一直不是忙着修炼，就是忙着打架、赶路，哪里有时间去弄那东西？
<br/>　　“那个，我虽然的确是四品炼丹师，不过还一直尚未参加炼丹师盟会的考核……”
<br/>　　“这……”
<br/>　　叶尘这话一出，众人顿时尽皆愕然，
<br/>　　而那金甲侍卫，也彻底没有了方才的恭敬，眼眸中透着一抹鄙夷，冷冷道：
<br/>　　“你们是在拿本将寻开心吗？没有经过炼丹师盟会的认可，便不是炼丹师，想要进城，就必须缴纳通行费！”
<br/>　　这金甲侍卫前恭后倨的态度，让叶尘十分不爽。
<br/>　　不过，他也清楚，
<br/>　　这云丹城可不比那黑岩城，硬闯肯定是不行的，而且他和云丹宗渊源颇深，只能按照他们的规矩来了。
<br/>　　“灵城主，要不我先进城，你们在外面等着我，待我补办了丹师宝卷，再出来接你们，如何？”
<br/>　　灵山自然不敢有任何意见，
<br/>　　“一切全凭叶先生做主！”
<br/>　　在来云丹城之前，叶尘特意嘱咐过众人，让他们还是称呼自己为叶先生，以免引人注意，听着也更加顺口一些。
<br/>　　很快，
<br/>　　灵山从乾坤镯中，取出了一万灵石，交给了那名金甲侍卫。
<br/>　　叶尘手上灵草、法宝倒是有不少，灵石却没有几块，所以也只能让灵山来付了。
<br/>　　“你们就在这里等我，我会尽快回来的！”
<br/>　　再次嘱咐了一句之后，叶尘直接迈步朝城中走去。
<br/>　　……
<br/>　　云丹城不愧是六级星域的中心，城内的布局，恢弘大气，单是一条主干道，就长达几千里，犹如一颗参天大树，分出成千上万道支路，支路之下，还有支路，绵延不绝！
<br/>　　若是普通人进入这座巨城之中，想要将整座城池走上一遍，也至少需要好几年的光景。
<br/>　　即便是以叶尘如今的神识，也只能覆盖一小部分。
<br/>　　好在，这炼丹师盟会，乃是云丹城极为重要的一个地方，几乎人尽皆知，叶尘随便一打听，很快便找到了地方。
<br/>　　这是一座透着古朴气息的宫殿，伫立在主干道一侧，周围空无一物，倒也十分醒目。
<br/>　　门前是一块巨大的石碑，上面刻着‘炼丹师盟会’五个金光闪闪的大字。
<br/>　　当叶尘赶的时候，门前聚集着很多人，一问之下，方才知道，云丹城乃是在整个修真界，都很有名的丹药之城，而且又是云丹宗的大本营，这里几乎人人都会炼丹！
<br/>　　而这炼丹师盟会，又是考核炼丹师的唯一机构，因而每天都有成百上千的人，前来这里参加考核，或者为了成为真正的炼丹师，或者提升自己丹师品阶。
<br/>　　问清楚了缘由之后，叶尘也没有多想，于是便跟着众人，随波逐流进入宫殿之中。
<br/>　　宫殿内部，布局精美华丽，而且十分宽阔，犹如一个巨大的广场，容纳十万人都绰绰有余！
<br/>　　宫殿内来的人很多，叶尘粗略估计，大约有三、四万人！
<br/>　　当然了，这些人里头，并非所有人都是来参加丹师考核的，还有不少只是来在一旁观摩的。
<br/>　　叶尘是第一次来，很多东西都不懂，于是漫步在众人之间，听着众人的议论声，从中提取一些重要的信息。
<br/>　　“哎，你们听说了吗？云丹宗的那位蝉衣小姐，今日也要来参加考核呢！”
<br/>　　“废话！这事早就在云丹城传开了，否则你以为今日为何如此热闹？这些人里面，有一半都是冲着蝉衣小姐来的！”
<br/>　　“不错！据说这位蝉衣小姐，乃是云丹宗宗主的亲传弟子，号称是云丹宗百年一遇的丹道天才，二十三岁那年，便已经四品炼丹师了！”
<br/>　　“如今才短短五年时间，她居然就要来参加五品丹师的考核了，如果蝉衣小姐一旦成功，那她将会是丹云宗近千年以来，最年轻的丹道宗师！”
<br/>　　……
<br/>　　听到众人的议论声，叶尘也不由得眉头微微一挑，
<br/>　　“他们说的，莫非就是后来成为云丹宗宗主的蝉衣仙子吗？”
<br/>　　记得前世，他得到云丹仙子的传承之后，也曾关注过云丹宗的情况。
<br/>　　而那时，云丹宗的宗主，便是一位叫做蝉衣仙子的女人，叶尘还和她见过几次面，交流过一些炼丹方面的心得感悟。
<br/>　　不过，按照前世的时间线，那已经是四、五百年之后的事情了。
<br/>　　彼时，叶尘已经在整个修真界声名鹊起，而蝉衣仙子也是威震一方的大人物。
<br/>　　此时的蝉衣仙子，还不足三十岁，自然远远没有达到那种高度。
<br/>　　就在叶尘正回忆着前世的事情，不仅暗暗感慨之时，忽然入口的方向，传来一阵阵躁动之声。
<br/>　　继而，便听到有人高呼道：
<br/>　　“快看！是蝉衣小姐来了！”
<br/>　　唰唰唰！
<br/>　　声音落下，所有人纷纷朝门口方向望去，就连叶尘也不由得转身观望，
<br/>　　只见入口处，
<br/>　　一名身穿白色素衣的美貌女子，从外面缓步走了进来，
<br/>　　这女子看上去也就二十来岁，但是却有着与其年龄颇为不符的淡然和沉静，一头齐腰长的秀发，只用一根紫色的细绳束着，除此之外，全身上下，一件饰品也无，给人一种，十分朴素，却又无比高贵的气息……
<br/>　　“还真是她！”
<br/>　　叶尘看到这女子，不由得低声喃喃了一句。
