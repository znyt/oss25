　　一道狂傲无比的声音，突然从天传来，这一句话的语气中，简直是丝毫没有将沧澜逆，这尊生死境高阶的大人物，放在眼中。
<br/>　　这一道狂傲无比的声音，正是楚风眠。
<br/>　　其他人的死活，楚风眠不管，但是白萱，对楚风眠可是有着不小的恩情，楚风眠当然不会坐视不理。
<br/>　　“楚风眠？”
<br/>　　白萱的脸色，都是一下子化为了惊喜，这声音，正是她无比熟悉的，楚风眠的声音。
<br/>　　本来楚风眠被不死魔尊追杀，白萱的心中都以为楚风眠肯定是死定了。
<br/>　　没想到楚风眠居然是真的活了下来，从不死魔尊的手中，都能活下来。
<br/>　　不过下一刻，白萱的心中却也突然焦急无比，楚风眠如今居然是跟沧澜逆对上了。
<br/>　　沧澜逆的性格，本就是狂傲嚣张到极致的人，如今一旦冲突，绝无善了可能。
<br/>　　尤其是沧澜逆，更是一尊生死境高阶的强者，在这颜古龙帝洞府中，近乎无敌。
<br/>　　楚风眠跟沧澜逆对上，只怕是凶多吉少。
<br/>　　不止是白萱。
<br/>　　听到这一句话的声音，在场无数武者的脸色都是变的极为精彩起来，立刻变的热闹起来。
<br/>　　沧澜逆是谁？沧澜世家的长老！
<br/>　　沧澜世家，可是在北域武道世家的第一人，武道世家，本就是千万年传承下来的，底蕴厚的惊人。
<br/>　　就算是北邙学院，轻易都不会跟沧澜世家起冲突。
<br/>　　如今这突然传出来的声音中，居然是直接在嘲笑沧澜逆。
<br/>　　无数众人也想不到，如今居然会突然出来人来搅局。
<br/>　　沧澜逆的脸色，在听到这句话后，瞬间阴沉到了极致，一双眼睛中的杀意，仿佛是要杀人。
<br/>　　以他的地位，还从来没有人敢这样对他开口的，居然说他不配，他给了面子，却被人一脚踢了回来，简直是对他无边的耻辱。
<br/>　　“阁下的话，未免太过狂傲了吧，既然你不给我沧澜逆这个面子，那我就直接将你抓出来！神神秘秘的，背地里捣鬼！滚出来！”
<br/>　　沧澜逆怒喝一声，凝聚灵力，猛然化为了一道巨大无比的手臂，对着空中抓了过去。
<br/>　　正是要将楚风眠，从这虚空中，直接抓出来。
<br/>　　“轰！”
<br/>　　但是这一抓，深处到空间之中，却是一无所获，反而是楚风眠的声音，再次响起。
<br/>　　“沧澜逆，就你这点小小实力，也敢嚣张打劫？你们沧澜家的人都是强盗出身的？沧澜霸那个小子在里面打劫已经死了，现在你也想要找死？”
<br/>　　楚风眠大笑连连的说道。
<br/>　　他的身形，一下子出现，站在了白萱的面前，手心一动，一股灵力猛然注入到白萱的身躯中。
<br/>　　这一股灵力中，蕴含有着几分战龙血脉的力量，虽然无法改变白萱的身躯，却是足以将她的伤势，直接治好。
<br/>　　一瞬之间，白萱身上的伤势完全恢复好了，身上的力量，甚至都是比起刚刚，还要强大许多。
<br/>　　“这是？”
<br/>　　白萱脸色都是惊讶无比。
<br/>　　她的伤势，不算严重，却也不是轻伤，想要彻底恢复，至少也是需要一两个月的时间才行。
<br/>　　可现在在楚风眠的手中，却是瞬间就足以将她恢复。
<br/>　　令她更加惊讶的是，楚风眠身上的气息，比起上一次见到楚风眠的时候，更加的深不可测了。
<br/>　　甚至是就连白萱，都有些看不出，现在楚风眠的实力，到底是达到了哪一步。
<br/>　　“霸儿！是你杀的？”
<br/>　　听到楚风眠的话，沧澜逆的表情，双眼猛然豆大。
<br/>　　沧澜霸，可是他们沧澜家的天才，更是沧澜逆的儿子。
<br/>　　沧澜霸陨落的消息，沧澜逆自然知晓，他在颜古龙帝洞府中疯狂寻找，想要找出杀了沧澜霸的凶手。
<br/>　　如今听到楚风眠的话，他心中的怒火，一下子爆发出来，脸色变得通红，一身武道，全部展现出来。
<br/>　　“拿命来！我要以你的头颅，祭奠霸儿！”
<br/>　　说话之间，沧澜逆一身武道，已经彻底凝聚，一身灵力，围绕在他的身边，令他看起来如同一颗太阳一般的耀眼。
<br/>　　这强悍的极致的武道，都足以压制着在场大部分人，根本无法出手。
<br/>　　他大手一挥，整个虚空都在颤动，一掌落下，这一掌，简直是蕴含武道之玄妙，将他沧澜世家的武道，淋漓尽致的彻底展现了出来。
<br/>　　“好恐怖的武道！这就是万年武道世家，沧澜世家的武道吗？”
<br/>　　“沧澜逆实力已经达到生死境高阶，一身武道，更是恐怖到了极致，这样的人物，简直已经无敌！”
<br/>　　“这小子居然只是一个神力境九重的人，居然敢跟沧澜逆挑衅，简直是在自己找死！”
<br/>　　一尊尊武者，看到楚风眠的境界后，都是摇了摇头。
<br/>　　他们本以为，楚风眠至少也是一尊生死境高阶的人物，才敢跟沧澜逆如此抗衡。
<br/>　　可眼前的楚风眠，仅仅是神力境九重的境界，这样的人物，就算是天资在妖孽，战胜生死境四重，五重的强者，已经是极限。
<br/>　　有实力张狂，可以说是狂妄，没有实力的张狂，只不过是在自寻死路罢了。
<br/>　　生死境六重，已经是踏入生死境高阶，这可已经是截然不同的实力了，绝不是神力境武者，可以撼动的。
<br/>　　“小心，沧澜世家的武道以力为主，不能硬抗。”
<br/>　　白萱出声提醒道。
<br/>　　她看得出来，楚风眠在颜古龙帝的洞府中，已经是有着更多的奇遇，实力再进一步。
<br/>　　但是沧澜逆，毕竟可是生死境高阶的人物，这样的人物，绝非是神力境武者可以撼动的。
<br/>　　让北邙学院的种子弟子前来，都没有几人可以做得到，除非是天榜前十的人物，才有可能。
<br/>　　但是这种人物，都是传说，在神力境九重苦修了数百年数千年的人物，才能做到这一步。
<br/>　　楚风眠看起来只是刚刚达到神力境九重，还未积累，又怎么可能做到这一步。
<br/>　　白萱相信楚风眠的实力，但是看到沧澜霸出手，她也只能是叹息一声，似乎已经看到了结局。
