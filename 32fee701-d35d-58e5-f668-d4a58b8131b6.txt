　　第636章
<br/>　　黎明天将亮未亮，草原上一片血腥味。尸体倒在地上，帅旗东倒西歪，血从尸体中流出，将土壤浸成红色。
<br/>　　百里胜骑在马上，目光打量着这一幕。败兵聚在他周围，人人表情肃穆，眼中除了愤怒便只剩下愤怒。
<br/>　　“侯爷，目前咱们还有五千人。”孙鸣之递过来一瓢水：“侯爷，先喝些水。”
<br/>　　水灌入喉咙中，百里胜才发现嗓子像是火燎一般疼。昨夜一场血战，狼山关三万将士便只剩下眼前这五千人。其他人要么是死在战场之上，要么是变成溃兵逃亡。
<br/>　　虽然突围成功，但这上万人的死亡，实在是一场惨败。
<br/>　　也幸亏卢俊义及时出现，不然的话，怕是会全军覆没的结局，连这五千人也保不住。
<br/>　　百里胜已经与卢俊义聊过，据他自己所说，他祖籍是冀州人，曾经受过百里家的恩德，所以从冀州不远千里赶来投靠，没想到碰见这种事。
<br/>　　关于他的来历，百里胜总觉得有些奇怪，但也想不通奇怪在那里。不过眼下生死存亡的关头，他也没时间深究。
<br/>　　“侯爷，咱们下一步去哪里？”
<br/>　　身边围着的几十人同时问。
<br/>　　百里胜从一张张脸上扫过，他们大部分身上都有伤，能从昨夜的血战中突围而出，实在是一场幸运。
<br/>　　他抬起头，目光看向东方，手中铁鞭一指，道：“我们往东走，拿下凉州城。”
<br/>　　“吼！”
<br/>　　五千从血战中活下来的儿郎齐声怒吼。
<br/>　　宋伯康的确小看了百里胜，同时也过分高估了自己。战场上瞬息万变，最终结果没有出来前，谁敢说自己是铁定的胜利者。而一个人临死前能爆发出怎样的潜力，确实也是无法通过数据估算出来的。
<br/>　　宋伯康就是因为没算到这一点，才令百里胜突围而出。五千人虽然不多，但如果想要搞事的话，还是能搞出许多事情来的。
<br/>　　而他也算到了，百里胜突围之后，第一个肯定是找凉州城麻烦。自己率领大军出征，凉州城由陆洛坐镇，而陆洛领军打仗的本事实在是有些糟糕。百里胜几乎是不可能打下凉州城的，但凡事都有万一。
<br/>　　于是宋伯康也顾不得赵子龙趁火打劫的事，一面下令各营各城阻击百里胜，一面率领大军往回赶。
<br/>　　双方以凉州为战场，展开一场追逐战，都在赶时间。
<br/>　　百里胜胸怀一腔怒意，领着士兵向凉州城进军，路上的粮草便从附近的村镇抢劫。过程中自然不可能文质彬彬，温文尔雅，杀人放火的事情屡有发生。
<br/>　　“侯爷，前面十里处就是蛤蟆城了，咱们要不要做什么？”孙鸣之问道。
<br/>　　从高处远远望过去，能依稀看到蛤蟆城的城郭。在百里胜心里第一恨的是宋伯康，第二恨的就是程大雷。他竟然敢和宋伯康联手对付自己，完全不顾忌兄弟情谊。当然，自己与他似乎也没有情谊。
<br/>　　难道你忘了我在长安的送衣之情，同桌饮酒之谊……如果有可能的话，百里胜很想杀进蛤蟆城，来一场轰轰烈烈的屠城。
<br/>　　但他之所以没有这么做，并非是因为心中还念着那丁点交情，实在是……蛤蟆城不好大。
<br/>　　兵多将广，粮草堆积如山，自己这五千人如果想要打蛤蟆城，无异于螳臂当车，蚍蜉撼树。
<br/>　　于是百里胜只能暂压雷霆之怒，强按虎狼之威，暂时饶过程大雷，继续向凉州城进军。
<br/>　　“报！”一名小校报来：“前面发现一支人马，看身上穿着似乎是蛤蟆城的人马。”
<br/>　　“什么，难道他们也是来阻拦我等的。”
<br/>　　“呃……看着不想，对方只有三五百人，像是路过。”
<br/>　　百里胜皱起眉头：“小心提防，对方未出手之前，咱们万万不可先动手。”
<br/>　　“是。”
<br/>　　史文恭今天是往朔方城送粮，目前的朔方城还不能保证自给自足，暂时需要蛤蟆城接济。他将粮食送到朔方城，选择原路返回。
<br/>　　“报，将军，前面好像发现了狼山关的人。”
<br/>　　百里胜率领的残兵败将如同疯魔一般，在凉州搞出很多事情，这件事情，史文恭自然也知道了。
<br/>　　现在宋伯康布下天罗地网阻击他，自己也在拼命的往凉州城赶。但却是没想到，史文恭竟会在此地碰上他们。
<br/>　　史文恭自从来到蛤蟆城后，一战没有打过，只能做些运粮的差事。对他来讲，这日子自然是有些无聊的，不过程大雷已经吩咐下来，不可与百里胜交锋，他毕竟也不敢违扛军令。
<br/>　　“莫要惊慌，继续行军，敌不动我不动。”
<br/>　　运粮队继续向前，正好与百里胜的队伍遇上，双方相距大概有一百步，却像是完全看不见对方似的。
<br/>　　但其实谁都是提心吊胆，百里胜不想节外生枝，史文恭见对方人数远超过自己，也不敢动手。
<br/>　　两支队伍擦肩而过，心脏都在胸口提着。无意中，史文恭的眼神从对方中扫过，目光落在卢俊义身上。
<br/>　　恰恰，卢俊义此刻的目光也正看着他，双方目光交汇，隐隐都有种熟悉的感觉，心中同时冒出一个想法。
<br/>　　这条汉子我是在那里见过的。
<br/>　　双方交错而过，在目光看不见对方背影后，同时松了口气。
<br/>　　过了蛤蟆城，其实距离凉州城就不远了。百里胜下令急行军，费了半日光景就赶到凉州城下。此刻，宋伯康的大军还未返回，坐镇凉州城的是陆洛。
<br/>　　百里胜一刻也没有停，立刻下令攻城。士兵悍不畏死，一味猛攻，都想起被围城的屈辱与弟兄们惨死的愤怒。
<br/>　　如此攻城，的确给凉州城带来巨大压力。陆洛又不是很有统兵之能，一时间手忙脚乱，城门险些被攻破。
<br/>　　以百里胜的愤怒，攻下凉州城后必然是一场血腥的杀戮，凉州的局势便会因此改变。
<br/>　　但凉州城城高墙厚，他想一时半刻打下来也是痴心妄想。就在双方混战之时，宋伯康领着大军杀回来了。百里胜腹背受敌，在丢下一批同伴的尸体后，他率领残部狼狈逃窜。
<br/>　　下一个目标，便是长安城。
