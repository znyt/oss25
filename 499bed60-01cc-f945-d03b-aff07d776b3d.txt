        这是一张被折成方形的纸……
<br/>
<br/>        好奇心作祟，童筱颖将这张纸打开来，发现里面竟然是一张类似于内存卡一样的小东西。
<br/>
<br/>        就是不知道这东西有什么用处？
<br/>
<br/>        要不……
<br/>
<br/>        把它交给大哥。
<br/>
<br/>        童筱颖心想着，把这东西重新用纸包好，塞入了口袋中。等着晚饭的时候，再给大哥傅夜沉带去。
<br/>
<br/>        古堡里也一直很宁静，直到午后，童筱颖被一道刺耳的尖叫声惊醒。
<br/>
<br/>        听那声音，像是雅珊的……
<br/>
<br/>        因为担心雅珊的安危，童筱颖没多想，就立即从床上下来，拉开房门跑了出去。
<br/>
<br/>        “把雅珊放了，我放你和林若琴走。”
<br/>
<br/>        童筱颖从阁楼里下来，才走到楼梯拐角，长廊里便传来了十爷的声音。
<br/>
<br/>        “十爷，给我们备车。”何明旭疾言厉色地呵斥。
<br/>
<br/>        童筱颖不得不加快步伐，循声而去，刚走出拐角，就看到何明旭一手从后环住季雅珊的肩头，将她束缚在身前，一手拿着注射器，正抵着季雅珊的脖子。
<br/>
<br/>        何明旭的身旁，林若琴手里也有注射器，但她紧张得连那只拿注射器的手都在颤抖。
<br/>
<br/>        所以，这就是起初，何明旭来找她的原因吗？
<br/>
<br/>        面对死亡，他是被逼急？
<br/>
<br/>        童筱颖正替季雅珊感到担忧的时候，十爷正背对着她，又开口说道：“你把雅珊放了，我给你当人质。”
<br/>
<br/>        “十爷您这是说笑呢？”何明旭不以为然。
<br/>
<br/>        因为十爷背对着自己，童筱颖不知道十爷是作何心态，但她能看出，十爷也不敢向何明旭太逼近。
<br/>
<br/>        而被束缚住的季雅珊，她的脸色十分苍白，似乎在害怕何明旭手中的注射器。
<br/>
<br/>        就在这个时候，林若琴看到了童筱颖，忽然间扯着嗓子喊道：“童筱颖，如果你能离开这里，替我给我老爸捎个口信。就说，我是被……”
<br/>
<br/>        “砰——”
<br/>
<br/>        耳畔响起了枪声。
<br/>
<br/>        林若琴的话还未说完，两眼眼珠子都快要瞪出来了，下一秒便倒在了血泊中。
<br/>
<br/>        何明旭吓得圈住季雅珊转过身去，只见八爷手里拿着枪，正对着他的脑门。
<br/>
<br/>        童筱颖被这一幕怔住，屏气凝神地捂住了嘴巴。
<br/>
<br/>        “把雅珊放了，我还可以留你个全尸。”八爷狂傲道。
<br/>
<br/>        何明旭却放眼望向了童筱颖，翕动着嘴唇。
<br/>
<br/>        他没有发出声音，但是通过他嘴唇上的动作。
<br/>
<br/>        童筱颖读出了，他在对她说：筱颖，对不起，求你帮我照顾好我爸爸妈妈。
<br/>
<br/>        何明旭唇上的动作做完，他似乎是豁出去了，直接拿着注射器往季雅珊的脖子上扎去。
<br/>
<br/>        八爷眼疾手快地又迅速开了一枪。
<br/>
<br/>        很明显，何明旭在一心求死，压根就没有想过要伤害季雅珊。
<br/>
<br/>        又是一声枪响后，何明旭双膝跪地，趁着最后一丝气息，爬到了林若琴的身边。
<br/>
<br/>        他将林若琴抱在了怀里，染了血的大手，轻轻地捂住了林若琴的肚子。
<br/>
<br/>        林若琴……怀孕呢？
<br/>
<br/>        童筱颖惊怔地转过身去背靠着墙壁，浑身起了鸡皮疙瘩，心里堵得发慌，两眼中瞬间布满了血丝。
<br/>
<br/>        “这两个人真是麻烦死了！我待会派人过来，把他俩的尸体处理一下。”八爷不耐烦地说道。
<br/>
<br/>        季雅珊也吓得半死不活，支支吾吾地说道：“这、这两支注射器里的东西也得一并销毁。不然，我们都得死。”
<br/>
<br/>        “这两蠢货，给了他们这么长的时间，都没研究出解药的辅助药来。归根究底，到底是学业不精。当时，我就说过，这两大学都还未毕业，能搞什么研究啊？真的是浪费我古堡里的粮食。”八爷一脸不屑地说道。
<br/>
<br/>        季雅珊随之回到了十爷身边，十爷安慰性地问道：“你没事吧？”
<br/>
<br/>        “嗯，我没事。”季雅珊回答道，顿了顿后，又接着说道，“我去看看筱颖，她一定被刚刚这情形给吓到了！”
<br/>
<br/>        “她也在？”十爷诧异。
<br/>
<br/>        季雅珊点了下头，随之从十爷身旁经过，向楼道口那边走去。
<br/>
<br/>        童筱颖听到了脚步声，双腿软得无法支撑起身子，便背靠着墙壁，缓缓地跪坐在了地上。
<br/>
<br/>        季雅珊走过来，看到她的时候，她已经坐在地上了，于是也跟着蹲下身来：“筱颖，刚刚一定吓到你了吧？没事了，你讨厌的那两个人都已经死了。”
<br/>
<br/>        “我是讨厌他们，但我没想过要他们死！”童筱颖抬眸看着季雅珊，冷冷地说道。
<br/>
<br/>        季雅珊不以为然地回答道：“我早就跟你说过，他俩迟早是要死的。没关系，你是第一次来古堡，第一次见人被杀，以后慢慢会习惯的。”
