　　东极之地，十日横空，烈日炎炎，火光冲天。妖族十大金乌飞翔天际，骄阳横空，焚山煮海，炙烤天地。
<br/>　　随着金乌临近之处，一处处火山爆发，土地龟裂，山摇地动。更有无尽群山丛林，轰然燃起大火，此火并非普通火焰，而是太阳真火，威力强大，极难扑灭，刚一燃起，便向着四方快速蔓延而去。
<br/>　　火焰所过之处，寸草不留，生灵尽灭。洪荒无数生灵惨遭灭亡，其中不仅有巫族，还有妖族与人族。
<br/>　　十大金乌从极东之地而来，最先遭难的便是东海之中的无数水族。
<br/>　　“大哥，要不要下去洗个澡，大日浴东海，想想就美极了。”一只金乌突然说道。
<br/>　　“滚！”
<br/>　　还没等大金乌回答，便有一道威严的声音传了出来，震退几人。紧接着，四海翻滚，海浪千重，一道沧桑的龙吟之音，轰鸣天地。
<br/>　　“昂！昂！昂！……”
<br/>　　“尔等畜生，给我滚出东海，我龙族虽然镇守四海，可却也不是任由你天庭帝俊、太一欺辱的。看在帝俊、太一的面子上，吾不杀你，速速给我滚出四海！”
<br/>　　“大胆，何人敢口出狂言，给本太子出来！吼！”
<br/>　　一众金乌闻言，瞬间便愤怒了，他们在天庭之时，只有欺负别人的份，哪有人敢呵斥他们，不想今日却被别人欺辱了，一个个面色怒红，其中一个更是狂叫道。
<br/>　　“哼！不知死活的畜生，给我滚！”
<br/>　　海藏深处，再次传出一道龙吟，充满威严霸道之气，紧接着又说道：
<br/>　　“不准尔等靠近四海，再不滚，就别走了。哼！”
<br/>　　这话中带着一股无上霸气，浩荡龙威，惊天动地，掀起万丈海浪，苍穹色变，直扑十大金乌。
<br/>　　大海咆哮，无情的骇浪携着翻天覆地之威，滚滚而来，向着十大金乌镇压下去。
<br/>　　在这股气势下，苍穹颤抖，众生恐惧，无数大神通者纷纷骇然起身，看着东海海藏方向，露出惊骇之色：
<br/>　　“龙族不是落寞了吗，怎么还有准圣后期的高手？此龙竟然如此强悍，烛龙……远古龙族之修？……”
<br/>　　“你……你竟敢如此对本太子说话，父皇不会放过你的，你等着！”
<br/>　　十大金乌也被这道威严的气势惊住了，纷纷大骇，放下狠话，连忙飞走。
<br/>　　“嗡！”
<br/>　　十大金乌走后，大海再次恢复平静，波涛滚滚，变得平静下来，仿佛是一头洪荒巨兽潜伏起来，一旦爆发，便会有着不可思议的恐怖力量。
<br/>　　昆仑山，元始三清，接引、准提五圣汇聚。
<br/>　　元始高坐云台，面色淡漠，看着十大金乌闹洪荒，淡淡道：
<br/>　　“巫、妖二族横行霸道，霍乱洪荒，不谙世事，大兴杀伐，造成洪荒无尽生灵遭受战乱困苦，不为该天地之主。人族爱好和平，积极进取，当为天地主角。各位以为如何？”
<br/>　　其余四圣静静思量一番，觉得有理，巫、妖两族霸道至极，掌管洪荒天地，不利于他们行事。当然，最重要的是，在他们成圣后，便知道了，人族为日后的天定的天地主角。
<br/>　　“善！”
<br/>　　老子面无表情，自成圣后，他越发的淡然无为。他为人教教主，人族为天地主角，他获利最大，他没有理由不同意。
<br/>　　“可！”
<br/>　　通天对此不敢兴趣，看着手中的宝剑，眼皮都不抬一下。
<br/>　　“善哉！”
<br/>　　见此，西方两位教主也自无不可。巫、妖两族大战，虽说是东方之事，他们越乱越好，这与他们西方没有关系。但，他们西方若要兴起，若是巫、妖两族一直霸占洪荒，那他们也绝对没有机会兴起，所以，人族为主角他们虽然不爽，但也没有太多的意见。
<br/>　　元始微微一笑，对此毫不在意，此事没有出乎他的意料，紧接着，他淡淡道：
<br/>　　“既然如此，那吾等便遮掩天机，让这十只金乌来揭开这巫妖决战的序幕吧。”
<br/>　　“善！”
<br/>　　语罢，五圣动手，法则涌动，时空扭曲，天机隐退。他们直接蒙蔽天机，让洪荒众大神通者们不能演算未来，不能预知后事。最重要的是，让天界帝俊与太一等妖族高手不知道洪荒地界之事。
<br/>　　十大金乌祸乱大日，焚山毁林，一路肆掠，让许多部落损失惨重，哭嚎声遍野。
<br/>　　“妖族小二，毁我部落，还敢如此猖狂，看我夸父来灭你！”
<br/>　　这日，一声大吼，惊天动地，声若雷霆，在虚空炸响。大地颤抖，苍穹色变，一股滔天煞气，冲天而起，爆发而出，直冲十大金乌而来。
<br/>　　“何人胆敢口出狂言，给本太子出来！”
<br/>　　十大金乌大声怒斥，一个个气焰嚣张，张口一吐，便是十道火焰神龙奔腾而出，带着狰狞之色，向着前方轰去。
<br/>　　“妖族小二，焚我族人，毁我族地，今日吾便灭了尔等，看尔等还敢不敢嚣张！吼！”
<br/>　　夸父一声大喝，惊天动地，紧接着，大地颤抖，苍穹撼动，在金乌的前方，一道恐怖的气势爆发，一个像大山一般的巨人，一步跨了过来。
<br/>　　这大巫气势滔天，煞气惊人，面容粗狂，他神色暴怒，双目通红，手中桃木杖一挥，带动无尽煞气充斥而来，苍穹色变，万山齐动，一股惊人的气势席卷而来。
<br/>　　洪荒中众多大神通者们只看到，那巨人夸父手中桃木杖挥舞，杀气弥漫，惊天动地，十大金乌一个个都娇生惯养，只有大罗金仙初级战力，哪里敌得过巅峰大巫夸父的凶威，没过多久，便纷纷受伤，口中更是惨叫连连。
<br/>　　“啊，不！不好！快走！”
<br/>　　金乌不敌，想要逃走，但夸父哪能让他们逃走呢，他夸父部落都快要被这十只金乌毁了，不报此仇，他如何甘心？
<br/>　　“孽畜休走，拿命来！”
<br/>　　就这样，一追一逃，一场长跑大战就在无数大神通者的瞩目之下，开始了。
<br/>　　十大金乌本身能够御使太阳真火，更能接引九天之上的太阳火精下来，他们虽然敌不过大巫夸父，但他们却能给他制造麻烦。无数太阳真火被金乌引到夸父身上，之后他们仗着金乌的本命神通，飞得极快，这让夸父只能在后面苦苦追逐。幸好夸父身为巨人，体型高大，使出法天象地神通，更是达到数万丈，一步迈出，便是数万里之遥，勉强能够追上。
<br/>　　就这样，夸父追着十个大日，越跑越远。
<br/>　　“金乌不死，夸父不灭，如何能够使得两族大战起来？”
<br/>　　昆仑山，元始等圣人冷笑一声，元始手捏印诀，时空扭曲，因果牵连，没入虚空，直直向着追逐十日的夸父飞去。
<br/>　　无声无息之中，夸父只感觉自己在奔跑之中，身上的力量在逐渐流逝。这种情况刚开始还不明显，但越到后面，这种流失就越明显。
<br/>　　如此，七七四十九天之后，夸父感觉自己累了，就停下来喝水，他大口一张，黄河与渭水两条河就被他侵吞起来。
<br/>　　滚滚河水如同两条水龙一般，对着他的口中钻去，气势惊人。不到一时三刻，两条河便被他喝光了。
<br/>　　此刻，喝了两条大河，对于他来说，一点感觉都没有，他还感觉累，想要河水。于是，他向着北方大泽跑去。
<br/>　　“他不行了，诸位兄弟，杀死他！嘎嘎！”
<br/>　　这时候，十大金乌反应过来，一个个目露凶光，向着夸父杀去。
<br/>　　无尽太阳真火被接引下来，焚烧着大巫夸父的身躯。若是之前，夸父还能抵挡，但如今他的力量流失的差不多了，他哪还能挡得住这炙热的太阳真火？
<br/>　　“啊！我不甘心！就算我死，也定不会让尔等好过！”
<br/>　　许久之后，夸父终于忍受不住被炙烤的煎熬，他的身体开始毁灭，他用尽全力，将手中的桃木杖一挥。木杖闪烁，顿时化为一片桃林阵法，将十大金乌困于阵中，而他的身躯，却也在此时轰然倒下，化为一座巨山。
<br/>　　至此，一代逐日大巫，夸父，身陨。
<br/>　　……
