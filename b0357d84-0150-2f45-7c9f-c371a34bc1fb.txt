
<br/>　　<br/>　　    “哗啦啦……”密林中忽然惊起一阵飞鸟，扑腾着翅膀飞上了半空。<br/>　　<br/>　　    三井小叶谨慎的朝着身后的人摆了下手，示意他们把枪都掏出来，对于那头能生撕马熊的藏獒他们还是比较忌惮的，林子里那一声吼过后，就再也没有什么动静传出来了，仿佛它只是无聊呻吟了下。<br/>　　<br/>　　    “换上装备，准备进入林中，三人一队战术队形，美奈小姐居中……”<br/>　　<br/>　　    当下，三井家族的人马上进入戒备战斗状态，头前三人手里拿着把五十公分左右的开山刀，呈品字形，后面两翼各再有一队，最后还有三人压阵，个个全副武装警惕十足。<br/>　　<br/>　　    “嗡，嗡”三井小叶和三井良河一人操纵个小型的的无人机，底下悬挂着摄像头连接着他们手中的视频屏幕，无人机上的镜头能够传回直径大概十米左右的画面，这种设备能够让他们提前遇见到周围的危险，避免碰上任何突发情况。<br/>　　<br/>　　    三井美奈很无奈的摇了摇头，她仍旧可惜着日记后半段被撕掉的内容，那里面记载的可能就是有关在这片未知区域的一些细节了，可惜最关键的这一段不在了。<br/>　　<br/>　　    两家无人机被操纵着灵活的飞向了林中，三井小叶和三井良河看着画面，然后示意他们开始朝着茂盛的原始森林里前进，这个战术队形加上精良的装备，让他们行进的非常顺利，很快边进入了丛林的深处里。<br/>　　<br/>　　    小林昙一说道：“前面有流水的动静，先往那个方向走，特别是注意脚下和树枝，我们带的抗病毒血清有限，如果碰上有毒的毒虫，有些可能没办法医治，所以先小心谨慎为主”<br/>　　<br/>　　    这里很明显似乎很久以来都没有人在踏足过了，地面上铺着厚厚的树叶都已经发黄了，踩上去后感觉是松软的，但一脚下去枝叶就全都碎成了几片，枯枝一踩就断，这都已经被风化的不知有多少年了。<br/>　　<br/>　　    小林昙一在三井美奈身边，小声的说道：“藏民说，这里住着他们供奉的神明，这一点让我有些想不通，既然这里有神明的话，为什么当初马丁鲍曼还能把那些珍宝运送过来，然后又再次离去了，这神明是干什么用的？”<br/>　　<br/>　　    三井美奈说道：“也可能是那些藏民的信仰，就像我们信奉天照一样，那是骨子里一种根深蒂固的念头吧”<br/>　　<br/>　　    “不见得，藏区这种地方太神秘了，你们是不了解，我在雪域高原这些年，碰到过天多让人无法置信的事了”小林昙一幽幽的叹了口气，说道：“我来到这里的第三年时，曾经跟随登山队去攀登冈仁波齐，在那座海拔六千多米的山峰上，亲眼看见了一个穿着僧袍的喇嘛，赤着脚露着胳膊，浑身上下只有一件单薄的袍子，要知道当时的温度已经零下三十多度了我们穿着厚厚的登山服才能在刺骨的寒风中勉强御寒，而那个僧人就坐在一处山石上闭着眼睛，仿佛被冻僵了一般，当时我们一队人从他身边经过时，队伍里负责搬运的夏尔巴人就跪了下去，那僧人睁开眼睛后就说了一句话，要我们不要从冈仁波齐的南麓靠右方向登山，当时这条线路正是我们预计的登山线，不知为何他却一语就道了出来，这僧人说完以后就又闭上了眼睛，但是队伍里的夏尔巴人却说什么都不肯再走原来的路线了，他们说这是智者的指引，绝对不允许违逆”<br/>　　<br/>　　    三井美奈惊愣的顿了下，问道：“然后呢？”<br/>　　<br/>　　    “夏尔巴人被称为全世界最好的登山家，他们经验丰富，熟知高海拔山峰的地形，也是绝佳的向导，他们说不允许走冈仁波齐的南麓后，我们商议了下也就只能改变路线了”小林昙一忽然惊叹的吐了口气，说道：“就在我们改变路线之后，冈仁波齐南麓发生了近年来最大的一次雪崩，那种大雪坍塌的气势能够掩埋一切山峰下的东西，如果我们恰好走了那条线的话，恐怕尸体都没有再见天日的时候了”<br/>　　<br/>　　    三井美奈顿时不可置信的说道：“是那个僧人的提示？”<br/>　　<br/>　　    小林昙一点头说道：“后来我们打听了下，夏尔巴人说他是冈仁波齐的神明，是大智者，但是却没有人知道他来自哪里，为何在冈仁波齐上，又为什么赤着脚穿着单薄的以上能够生存下去，甚至都没有人知道他的食物来自于哪里……”<br/>　　<br/>　　    冈仁波齐是藏区四大圣山之一，也被誉为地球的中轴线，从古至今内陆人可能对冈仁波齐没有什么听闻，但在藏区，印du，尼bo尔等地，这一座神山几乎等同于南天门后面的天庭了，据说整座山峰上下，一共居住了苯教的三百六十位神灵。<br/>　　<br/>　　    藏区一直都有个传说，那就是当你登上冈仁波齐朝圣归来之后，将相当于你修了三世的功德。<br/>　　<br/>　　    三井美奈和小林昙一正在交谈的时候，树林里的两架无人机正在飞行的时候，突然就从半空中“啪”的一下砸在了地上，三井小叶顿时一愣，连忙走过去捡起无人机后查看着，三井良河皱眉说道：“失灵了，电池的电量还有百分之九十多，但是无线指示灯灭了”<br/>　　<br/>　　    “有电磁干扰？”三井小叶仰着脑袋看了片刻，说道：“除非林中有很强的磁场，试下对讲机看看”<br/>　　<br/>　　    无线对讲和耳麦接通以后，里面传出一阵阵的电流嗞啦声，明显也属于失灵的状态了，讨论了片刻也没有什么结论，他们只得再次朝着林中前进。<br/>　　<br/>　　    “你们看那……”走在队伍最前面的三井小叶突然惊呼了一声，不可置信的指着树林前面，惊愕的回头说道：“看见了么，那里似乎有一座宫殿？”
