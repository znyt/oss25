　　此刻，双方大战打响。
<br/>　　不，或许不能说双方。因为，一方是如狼似虎地近千武者，一方，则只有寥寥四人。
<br/>　　但，就是这般不对等的战斗，战斗情况却是以‘胶着’而进行着。
<br/>　　对，胶着，而并没有任何劣势或者优势产生。
<br/>　　凌羽，拦下了韩苍。
<br/>　　钟无忧，一人战风花月以及大批武者。
<br/>　　白冰雪，站于萧逸身前10米外，双手一股股极寒罡气，覆盖了方圆数十米范围。
<br/>　　范围虽不大，却威力极强。
<br/>　　任何一个踏入这个范围的武者，尽皆顷刻被冻结成冰雕。
<br/>　　叶铭，则牢牢站于萧逸身旁，以防意外。
<br/>　　金光屏障内，萧逸皱着眉。
<br/>　　他万万没想到，这洞府遗迹内的禁制会是金光禁制。
<br/>　　这让他现在很是被动。
<br/>　　半晌，他摇了摇头，盘膝坐下，开始参悟这块武道石碑。
<br/>　　金光屏障，对他而言，事实上很是脆弱。
<br/>　　他随手一剑就能破去。
<br/>　　若是钟无忧等人不敌，或者情况有变，他会立刻破去金光屏障，出手对敌。
<br/>　　虽说金光屏障破去，他将彻底失去参悟武道石碑的机会。
<br/>　　更失去了获得暴雪山庄重宝的机会。
<br/>　　但，他还不至于太过在乎。
<br/>　　在萧逸看来，暴雪山庄的所谓传承与重宝，远远比不上裂天剑魔前辈的传承和馈赠。
<br/>　　能得，便得。
<br/>　　不能得，也不会太过遗憾。
<br/>　　大不了等以后突破极境，实力强了，再来此地，强行破掉大禁制，再去里头重宝。
<br/>　　心头有了想法。
<br/>　　萧逸一边开始快速参悟这块武道石碑。
<br/>　　一边分出一丝心神，注视着钟无忧等人的战斗。
<br/>　　凌羽和韩苍这边。
<br/>　　韩苍，本是天元一重修为，第一轮冰海之缘后，此刻已是天元三重修为。
<br/>　　本身第一轮前十排名中，二至六名的分数相差本就不大。
<br/>　　这几人所获得的吞灵冰鲸，以及吞灵冰鱼的数量，本就差不多。
<br/>　　但，修为境界越是高，所需的突破力量，也是越大。
<br/>　　故，韩苍仅仅跳了两重修为。
<br/>　　而凌羽等人，则跳了四重修为，从地元八重，突破到天元二重。
<br/>　　此刻，韩苍已手段尽出，却根本奈何不了凌羽。
<br/>　　韩苍的秘法，虽厉害。
<br/>　　当初在极寒冰海时，曾以天元一重修为，爆发出媲美巅峰强者的战力。
<br/>　　但，那里是极寒冰海。
<br/>　　冰海之下，蕴含着大量极寒精华。
<br/>　　以及极其浓郁的冰属性气息。
<br/>　　这才让得他的秘法发挥出惊人的威力。
<br/>　　可这里，是洞府遗迹中的宫殿。
<br/>　　虽同样雪花飘舞，但增幅极其有限。
<br/>　　天元三重修为下，顶多发挥了媲美天元八重的战力。
<br/>　　反观凌羽，天元二重修为，在这宫殿之内，战力不增反减弱些许。
<br/>　　可他，仍旧能力战不败。
<br/>　　这时，便看出剑道武者的强悍了，特别是剑宗弟子。
<br/>　　凌羽乃是裂天剑宗剑堂首席，其战力，绝非表面上的修为那么简单。
<br/>　　当初在东边六郡时，不过地元六重修为，已能无视多重修为差距，轻松击败五位半步天元的联手。
<br/>　　现今天元二重修为，单单现今爆发的战力，绝不在天元五重之下。
<br/>　　“星影剑阵。”此时，凌羽大喝一声。
<br/>　　漫天星光涌动，自成无数把星光之剑。
<br/>　　星光之剑，自动运转，竟成一玄妙大阵。
<br/>　　将他自身，以及韩苍，彻底困于大阵之中。
<br/>　　韩苍手中一股冰炎打向剑阵。
<br/>　　冰炎，苍白、冰冷、瘆人。
<br/>　　但却瞬间被无数把星光之剑绞成粉碎。
<br/>　　“好玄妙的大阵。”韩苍狠辣的目光中，露出了些许惊讶。
<br/>　　“每一把星光之剑，威力在天元五重左右。”
<br/>　　“无数把加起来，竟爆发出不下天元八重的战力。”
<br/>　　韩苍忽然冷笑一声，“威力虽强，但这是以你体内元力快速消耗为代价吧。”
<br/>　　“这个剑阵，你维持不了多久的。”
<br/>　　凌羽冷漠着脸，道，“那你呢，你的秘法又能维持多久？”
<br/>　　事实上，韩苍拥有秘法，凌羽同样有。
<br/>　　星影剑阵，本就是裂天剑宗内一门极强的秘法，位列天阶低级。
<br/>　　另一边，钟无忧的战斗，毫无压力。
<br/>　　一颗参天之数，陡然而起。
<br/>　　无数藤蔓，如魔乱舞。
<br/>　　风花月以及数百极寒之地武者，只有勉强的招架之力。
<br/>　　这一千多武者中，除了韩苍有天元境修为外，最强的便是风花月这个半步天元修为了。
<br/>　　其余的，都在地元九重修为以下。
<br/>　　白冰雪那边，也不必多说。
<br/>　　天元二重修为的她，对一群地元九重修为以下武者，毫无压力。
<br/>　　战斗，就这般‘胶着’着。
<br/>　　金光屏障内，萧逸微微一笑。
<br/>　　看来，他能安然将这块武道石碑参悟殆尽了。
<br/>　　同时，他现在更加确定了自己之前的猜测。
<br/>　　这块武道石碑的主人，应该属于暴雪山庄庄主。
<br/>　　但石碑上的武道知识层次，以及难度，完全比不上裂天剑魔前辈的剑道知识。
<br/>　　以他现今修为境界，以及武道知识水平。
<br/>　　顶多一个时辰，必能参悟殆尽。
<br/>　　最重要的是，这块石碑上的武道知识，竟然是剑道知识。
<br/>　　很显然，当年的暴雪山庄庄主，是一位剑道强者。
<br/>　　以萧逸如今剑道水平，身负裂天剑宗两位传奇人物的剑道知识。
<br/>　　外加历代最强剑主的剑道知识。
<br/>　　如此剑道水平下，参悟这块武道石碑上的剑道知识，自是难度不大。
<br/>　　时间，渐渐过去。
<br/>　　大半个时辰后。
<br/>　　金光屏障内，萧逸猛地睁开了眼睛，双眼精光爆射。
<br/>　　“还剩三成。”萧逸自语一声。
<br/>　　整块武道石碑，已参悟七成，还剩三成。
<br/>　　要不了多久，就能完成参悟。
<br/>　　另一边，凌羽、钟无忧等人，也发现了这边的情况。
<br/>　　萧逸身上的那道金光屏障，光芒愈发明亮。
<br/>　　这是武道石碑即将被参悟殆尽，触发下一个禁制的征兆。
<br/>　　“该死。”韩苍咬牙切齿，眼中怨毒，仿佛一柄利剑，直直刺向萧逸。
<br/>　　“哼。”不远处，钟无忧冷哼一声。
<br/>　　“一群孙子，等萧逸完成参悟，老子慢慢和你们算账。”
<br/>　　“哦？是吗？”恰在此时，一道冰冷之言，从暗处响起。
<br/>　　两道身影，猛然从人群中飞跃而出。
<br/>　　身上，杀意四溢。
<br/>　　两把利剑，带着浓郁血芒，直刺萧逸而去。
<br/>　　“不好？”钟无忧等人，大惊失色。
<br/>　　“血无伤、血焰，你们敢？”白冰雪冷喝一声。
<br/>　　不错，忽然出手的两人，竟是血无伤与血焰。
<br/>　　白冰雪想阻拦，但却猛地被数百武者围攻，无法施以援手。
<br/>　　叶铭一直站于萧逸身旁，冷声道，“想伤我萧逸师弟，先过我这一关吧。”
<br/>　　“不自量力，滚。”血无伤人未至，已远远一掌轰出。
<br/>　　叶铭直接被轰退。
<br/>　　随后，两人直取萧逸而去。
<br/>　　恰在此时。
<br/>　　两道身影，同样从人群中飞跃而出。
<br/>　　一把玄铁重刀，重重劈下，横亘于萧逸身前数米外。
<br/>　　一道飘然身影，灵动至极，又飘逸至极，像极了扶风之柳，拦在了金光屏障之前。
<br/>　　来人，正是林劲与柳嫣然。
