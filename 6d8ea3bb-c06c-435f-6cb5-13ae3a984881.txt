　　心魔之王听到楚风眠的命令之后，便是化为了一道遁光，离开了这先祖殿之中，前去了剑源界，去镇守剑源界。
<br/>　　心魔之王早已经是被楚风眠所降伏，现在也是因为楚风眠相助，他才可以凝聚意识，对楚风眠忠心耿耿，根本不担心背叛的风险。
<br/>　　在解决了这仙人之魂后，楚风眠的目光缓缓的看向了那林天殿主，眼神之中露出了几分冷笑。
<br/>　　他早就猜到了这林天殿主肯定不老实，肯定会来对付楚风眠，没想到这五行天内，居然还隐藏着在仙人之魂这样的东西。
<br/>　　这一缕残魂，突然出手，纵然是半步天人境强者都有着很大的可能来不及反应，可能会死在这一缕残魂的手中。
<br/>　　不过楚风眠现在的实力，可是要比起寻常的半步天人境武者还要强横。
<br/>　　这一缕仙人残魂，被楚风眠挥手这哪有，更是化为了心魔之王的补品，为心魔之王增添的力量。
<br/>　　那林天殿主，看到楚风眠出手降伏了那仙人之魂的一刻，已经是面如死灰，急忙的想要向着外面逃去。
<br/>　　可楚风眠挥手之间，同时也封锁了整个先祖殿，这林天殿主，是无路可逃了。
<br/>　　“放过我，放过我，我愿意为你做事，认你为主……”
<br/>　　林天殿主这时才彻底的慌张了，急急忙忙的对着楚风眠喊道，想要臣服楚风眠。
<br/>　　可楚风眠，却不会给这林天殿主第二次机会了。
<br/>　　“死吧！”
<br/>　　楚风眠冰冷的开口道，举手之间一掌轰击过去。
<br/>　　“不！”
<br/>　　那林天殿主还想要挣扎，可在楚风眠的掌心面前，他根本便是都无从反抗，身形猛然炸裂开来。
<br/>　　只见楚风眠的手指一点，从这林天殿主破碎的身躯之中，抓出了一缕金光来，正是这林天殿主的记忆。
<br/>　　林天殿主，身为五行天的高层之一，宝库的真正所在，此人应该知道。
<br/>　　楚风眠翻阅了一番，便是将五行天之中的种种秘密，全部知晓，宝库的真正所在，楚风眠也已经是知道了。
<br/>　　五行天的真正宝库，根本不再这先祖殿之中，这林天殿主只是想要借助先祖殿之中的的那仙人之魂，来对付楚风眠。
<br/>　　可是林天殿主却是太过小看了楚风眠的实力，最终这仙人之魂都成为了楚风眠的囊中之物。
<br/>　　楚风眠的身形一闪，离开了先祖殿，前往了五行天偏僻的一个角落之中。
<br/>　　在这角落中，有着一座看起来残破无比的宫殿，一眼看去，极为平凡。
<br/>　　这里就是五行天的宝库所在。
<br/>　　只怕是不知道秘密的人，根本想不到，五行天的宝库，居然会藏在这样一座残破无比的宫殿之中。
<br/>　　不过当楚风眠走进这残破宫殿之中的一刻，突然之间，周围无数的阵法禁制，却是突然之间爆发了出来。
<br/>　　这些阵法禁制，一直是隐藏了起来，只有人踏入到了这宫殿之中的一刻，才会苏醒开启。

<br/>　　这一道阵法之中，蕴含纯粹的五行之力，几乎是足以跟五行天的护山大阵相提并论。
<br/>　　寻常半步天人境的强者，都未必可以破解了这五行乾坤大阵。
<br/>　　不过楚风眠却只是观察了一眼，便是凝聚灵力，凝聚在了手心之中，突然之间一掌轰击而下。
<br/>　　轰隆！
<br/>　　楚风眠的一掌轰击而出，龙吟响彻，一股股灵力，都是化为了一条条远古战龙的模样。
<br/>　　而楚风眠则是位于这万龙中心，乃是万龙之祖，一声令下，群龙咆哮，无尽龙力，凝聚在掌心之中轰然落下。
<br/>　　砰！
<br/>　　剧烈的灵力碰撞。
<br/>　　在楚风眠的一掌之下，这五行乾坤大阵，被一下子打出了一个缺口，一掌被轰碎。
<br/>　　楚风眠向前一步，再度踏入其中，便是看见在这宫殿深处，琳琅满目的无数宝物。
<br/>　　灵石，灵药，几乎都堆积成山，大量的丹药，装在一个个葫芦之中。
<br/>　　这一个个葫芦几乎都盛放数千上万枚丹药。
<br/>　　这样的葫芦，都有大大小小数千个。
<br/>　　无数灵器，也摆放在了其中，堆积如山。
<br/>　　饶是楚风眠，都被眼前的景象所震惊了。
<br/>　　不过不管是这些灵石，灵药，还是灵器，丹药的品级，却都不是很高，虽然数量众多，可对楚风眠却是用处不大。
<br/>　　毕竟如果是真正那些对半步天人境强者都有着极大效果的圣药，灵器，早就死被五行天的那些老古董瓜分了，不可能安静的放在这宝库之中。
<br/>　　这宝库之中积攒下来的，多半都是对古帝有用的圣药，灵器等，对楚风眠无用。
<br/>　　不过楚风眠刚刚建立起剑道门来，剑道门的宝库倒是空虚，短时间以楚风眠自己的积攒还可以撑住，可长时间却是不行。
<br/>　　这五行天宝库，正好足以去填充剑道门。

<br/>　　楚风眠拿出了一枚空戒，大手一挥，这宝库之中的无数宝物，纷纷被楚风眠收入到了空戒之中。
