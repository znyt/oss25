　　这九曲双煞阵，是利用后天八卦布置的一个简单的奇门阵法，对叶少阳来说，真的只是雕虫小技，也没怎么在意。
<br/>　　下山进入义庄的路上，翠云抓紧了叶少阳的胳膊。
<br/>　　“怎么，害怕了？”叶少阳问。
<br/>　　“嗯，有点，这里不是停放死人的地方吗。”翠云哆哆嗦嗦地说道。
<br/>　　“跟我在一起不用怕死人。”叶少阳突然想到一句很装逼的话，说道，“死人比较怕我。”
<br/>　　从九曲双煞阵进入义庄，已经有两个道士打扮的人在尽头接他们。
<br/>　　张道长介绍了一下，这两个都是义庄的尸工，叶少阳一听就明白了，所谓摇工，就是给赶尸匠打下手的。赶尸的时候，赶尸匠要在后面押着尸体，前面要有个开路的，负责摇铃。
<br/>　　摇铃也不是随便摇的，要配合赶尸匠念咒节奏，与尸体身上的灵符发生作用，三者缺一不可，带路也很有讲究，因此尸工一般也都是道士，只是实力低微，不够当赶尸匠，才给赶尸匠打下手。
<br/>　　那两个尸工跟张道长寒暄了几句，得知他要见青冥道长，于是在前面带路，穿过村里小路，来到正中间一座相对高大的院落门外，张道长也不用人通报，直接上前拍门。
<br/>　　门开后，里面走出一个六十来岁的老道，长得有点贼眉鼠眼的，张道长立刻上前热情地打招呼，为双方介绍，叶少阳才知道这个就是青冥道长，也是这个义庄的负责人。
<br/>　　“这一位是我刚交的好朋友，我的小老弟，叶少阳，茅山天师。”
<br/>　　“茅山天师！”
<br/>　　青冥道长一下怔住，上下打量叶少阳，神色有些狐疑。
<br/>　　这也在张道长的意料之中，他最初听叶少阳自我介绍的时候，表现的比青冥道长还要强烈。当下上前拍了拍他的肩膀，让他进屋再谈。
<br/>　　一行人进屋之后，青冥道长吩咐人上茶。张道长迫不及待地把整件事情讲了一遍，青冥道长听完之后，虽然啊震惊，但看着叶少阳的眼神仍然是有点怀疑，迟疑半晌，说道：“不是我不同意，张道兄介绍来的朋友，我肯定信得过的，只是那具尸体非同小可，万一出了岔子，后果不堪设想……”
<br/>　　虽然张道长把叶少阳夸的天花乱坠，但之前叶少阳作法的时候，他毕竟是不在现场，光是听别人的讲述，肯定难以完全相信。
<br/>　　叶少阳想到这一点，提出先看一看那具女尸。这要求当然不过分，青冥道长亲自带他们去义庄。
<br/>　　在青冥道长居住的这栋宅子的后面，有一长排木头修建的简易的房舍，窗户很多，门上贴着好几张灵符，叶少阳知道这里一定就是义庄了。
<br/>　　上面的那些窗户，是为了通风，因为但凡尸体多的地方，尸气就会浓郁。如果通风不好，尸气难以散去，时间长了会容易尸变，所以古往今来，义庄都修建得十分宽敞，窗户多，风一吹可以带走里面的尸气，然后溶解在空气里。
<br/>　　青冥道长过去推开门，请叶少阳等人进去。
<br/>　　叶少阳见翠云脸色紧张，于是让她在外头等着，自己跟青冥道长和张道长一起进去，一看，偌大的房间空荡荡的，地上铺着一层草席，上面躺着六七具尸体。在房间正中间，摆着一只香炉，点着驱尸香，用来驱除房间里的尸气。
<br/>　　叶少阳刚要往里走，青冥道长轻轻拉住他，笑道：“叶天师道法高深，可能找到，那一具女尸在哪里？”
<br/>　　考我来了？
<br/>　　叶少阳笑了笑，也没作声，转头看去，这些尸体都直挺挺地躺着，身上盖着黄布，连男女都分不清。当下从背包里摸出阴阳盘，以罡气激活，拨动了几下，指针立刻指向了一个方向。
<br/>　　叶少阳走到一具尸体前面，毫不迟疑地掀开了尸体身上的黄布，下方的尸体身材玲珑，关键是有胸，一看就知道是女的，脸上贴着一张灵符。
<br/>　　叶少阳转头冲跟上来的青冥道长笑了笑，道：“是不是她？”
<br/>　　青冥道长一脸的吃惊，看看张道长，又看看叶少阳，皱眉问道：“我用定尸符定住她了，尸气并不外泄，你是怎么查验到是她的？”
<br/>　　“这有什么，任何灵符，都不可能将气息完全封印住，总会有所泄露，只是多与少的差别。只要有泄露，就一定会被检测到。”
<br/>　　青冥道长道：“这么微弱的尸气，你怎么检测地到？”
<br/>　　“对茅山法术来说，没有检测不到的气息。”叶少阳淡定地回答道。
<br/>　　青冥道长还在发呆，张道长笑着上前拍他的肩膀，说道：“我就说吧，我这位叶老弟，可是正宗的茅山天师。虽然因为受伤实力大减，但法术还是高深的。”
<br/>　　青冥道长缓缓点头，冲叶少阳抱拳，微微一笑道：“叶天师请看看这尸体。”
<br/>　　叶少阳蹲在女尸面前，掀开女尸面上的灵符，一眼看去，顿时一惊：之前有灵符挡着，还看不出什么，现在解开灵符，叶少阳才发现，这个女尸真的是……美得不正常。
<br/>　　说不正常，是因为但凡是人，生前不管再好看，在死亡之后，也一定会变得难看，肌肉僵硬是一方面，主要还是灵魂出窍，魂力逐渐散去，这就是为什么尸体看上去总是冷冰冰的缺乏灵气的缘故。
<br/>　　而眼前这具尸体，面色栩栩如生，跟活人一般无二，看上去就像一个人在沉睡中，下一秒就会醒来似的。
<br/>　　“不是煞吗，怎么会这样！”张道长也是第一次亲眼见到这具女尸，惊声说道。
<br/>　　青冥道长叹了口气道：“我上次跟你说的时候，也不知道是这样，以为只是普通的形尸化煞，这几天愈发不正常了，叶天师正好看看，这是什么邪物。”
<br/>　　叶少阳一只手按在女尸的额头上，感觉到女尸的皮肤仍然有弹性，只是没有温度。把拇指按在女尸的额头上，用罡气感知了一下，女尸体内全是煞气，这种煞气，是尸气与怨气的混合，形成了一种更加邪性的气息。
