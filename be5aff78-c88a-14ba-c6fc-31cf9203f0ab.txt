　　第390章
<br/>　　看莫将难脸上的表情，明显是被噎了一下。你身在西北，竟然没听过万马堂，竟然没听过『莫将难』这三个字。
<br/>　　“很有名么？”秦蛮是个老实人，不懂就要问，现在他真的不知道，所以很认真的询问。
<br/>　　莫将难倒抽一口凉气，这个黑脸汉子明显不老实呐。你让我怎么说，难道说我自己很有名气。
<br/>　　“万马堂是西北绿林之首，西北山寨水寨，江湖堂口都要听我们堂主的话。今天我们堂主已来了琴川，还不快快让程大雷出来迎接。”莫将难的手下道。
<br/>　　“西北绿林之首？”秦蛮表情更加困惑：“我一直以为我们大当家是西北绿林之首呢。”
<br/>　　“你……”别说莫将难，就连他的两个随从也被秦蛮的话噎得够呛。
<br/>　　“不管你们是什么人，到了琴川关就要受这里的规矩。拒不接受检查，又打伤卫兵，按照琴川关的规矩……”
<br/>　　秦蛮顿了顿，他也忘了山寨关于这一项是如何规定的。
<br/>　　“缴纳罚金，赶出琴川关。”城门队长赶紧补充道：“若有情节恶劣者，当在大牢监禁。”
<br/>　　城门队长见秦蛮出现，立刻凑上去，心里巴望着秦蛮可以替他出头。
<br/>　　“秦队长，我看他们情节就挺严重的，干脆直接抓起来吧。”
<br/>　　今日负责警戒城门的，恰好是秦蛮的蛮字队。秦蛮虽然外表看上去老师，但内心着实有些护犊子。
<br/>　　“言之有理。”秦蛮目视莫将难，道：“还不下马？”
<br/>　　莫将难冷笑一声：“想要我下马，先要看看你有没有这个资格？”
<br/>　　秦蛮眉头拧起，忽然向前跨出一步，推山倒的架门，侧肘砸向马头。
<br/>　　“给俺下马。”
<br/>　　轰隆隆！
<br/>　　骏马如大山一般崩塌，马上莫将难一惊，身子跃起，亏得他反应及时，才没摔在地上，在众人面前落了威风。
<br/>　　抬眼看向秦蛮，不由得对其刮目相看。程大雷手下还是有高手的，能一招推倒战马，这需要怎样的力气。
<br/>　　莫将难的两个随从也从马上下来，站在莫将难身后。双方互相对峙，目光盯着对方的肩膀膝盖。
<br/>　　能在短暂瞬间，从马上跃下，秦蛮也知道此人并不简单。虽然万马堂这名字自己未曾听过，可想也知道，对方来头不小。
<br/>　　之前已有人去通知程大雷，此刻他恰好过来，就看到城门剑拔弩张的情形。
<br/>　　程大雷挥挥手，让手下人撤了兵刃。
<br/>　　“你们是什么人，为何打伤我的人？”
<br/>　　莫将难刚才紧张得不行，见到程大雷出现，方才松口气。他理了理身上儒生袍子，道：“你就是程大雷？”
<br/>　　程大雷眉头皱起，这人傲慢无礼，究竟什么来头。
<br/>　　秦蛮走过来，低声道：“他说他是万马堂堂主，西北绿林道以他为首。”
<br/>　　“万马堂？什么来路？”程大雷一头雾水：“西北绿林道的首领，不应该是我么？”
<br/>　　“我也是这么认为的，可他非说是他。”
<br/>　　“这个人莫非脑袋有问题。”
<br/>　　“嗯，我看着也像。”
<br/>　　程大雷和秦蛮凑在一起嘀嘀咕咕，这话听在莫将难耳里，他脸上青一阵红一阵。程大雷说些什么他其实听不明白，但傻子也知道，二人这一唱一和的，是根本没将自己放在眼里。
<br/>　　“呔，你们说够了没有。”莫将难突然喝了一声。
<br/>　　程大雷收起脸上笑容，也不再捉弄他，板着脸道：“我不管你们是从什么地方来的，无端扰我琴川关所为何事？如果说不出个子午卯丑，怕你们好来，却是不好走。”
<br/>　　莫将难将双手负在身后，道：“难道不请我们去里面说话？”
<br/>　　“在这里说话也就够了。”程大雷：“你要说什么？”
<br/>　　莫将难脸上肌肉跳动了两下，强压下心头火气，道：“长安城的事你可听说了？”
<br/>　　“长安城发生什么事？”程大雷眨眨眼睛。
<br/>　　“量你孤陋寡闻，也不曾听到消息。”莫将难道：“林天将在幽州兴起义军，要讨伐昏君，现在已经打到长安城下。林天将号召天下绿林道，共诛昏君，天下热血男儿，当共举大事。”
<br/>　　“林天将？”程大雷摇摇头，觉得事情变得越来越扯淡了：“那么这件事和我有什么关系？”
<br/>　　“你琴川关也是天下绿林道之一。那林天将是千年不遇的英雄，此刻举起大旗，天下绿林皆愿意投靠在他的麾下。你琴川关要人有人，要粮有粮，这正是你的好事。如果能在林天将麾下得到重用，日后封王拜将，不在话下。”
<br/>　　程大雷和秦蛮对视一眼，皆感到匪夷所思。
<br/>　　“你是要我出兵？”
<br/>　　莫将难微微点头：“我们西北绿林道以我为首，点起兵马后，马上就要出发前往长安。”
<br/>　　程大雷颇有些无语，钟伟虎才来不久，要自己进京勤王，如今绿林道又过来让自己共举义旗。
<br/>　　现在自己究竟什么身份，是官是匪，是匪是官？
<br/>　　事情变得越来越好玩了呢。
<br/>　　程大雷看着莫将难，忽然叹了口气：“你们回去吧，今日的事情我不怪你，以后，好自为之。”
<br/>　　莫将难皱起眉头：“你不出兵？”
<br/>　　程大雷忍住动手抽他的冲动，这个人大概脑筋有问题，也不能和他太计较。
<br/>　　“你速速回去吧，我琴川关自有我琴川关的行事准则，长安城的事和我无关。”
<br/>　　“我便知道，你不过是个沽名钓誉的懦夫。”莫将难不屑道：“天下绿林道共举义旗，你想做缩头乌龟，不怕成为天下人的公敌。”
<br/>　　程大雷真不知道这万马堂什么来历，平日也不曾听谁说起过。他感到奇怪的事，究竟是怎样的地方，才能培养出这种等级的奇葩。
<br/>　　他已不愿和这种人过多计较，挥了挥手道：“秦蛮，送客。”
<br/>　　眼见程大雷转身要走，言语对自己如此不屑，莫将难立刻动了杀机。口中喝道：“狗贼，留步！”
<br/>　　一步横冲过来，刀已经出鞘，向程大雷背心斩去。
<br/>　　流星逐月般的一刀，果然快到极致，感觉已是空气中一道闪电。
<br/>　　程大雷猛地回头，两指夹住刀柄，不动如山。
<br/>
