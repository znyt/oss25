　　楚风眠跟寒月漓在密室中修炼，血脉交融的时刻。
<br/>　　武胜国度已经是发生了翻天覆地的变化，三大宗门，武胜学院，彻底被降伏。
<br/>　　在秦皇军团跟破灭军团的压迫之下，三大宗门，武胜学院的所有弟子，尽数臣服，再也不敢反抗。
<br/>　　“少主。”
<br/>　　弑剑君在外已经等候许久，看到楚风眠走出来，恭敬的跪下开口道。
<br/>　　“三大宗门跟武胜学院的弟子，已经被全部降伏，贬为奴隶。”
<br/>　　“不错，你做的很好。”
<br/>　　楚风眠看着弑剑君，心神一动，一滴金色血液便是从他的指尖凝聚出来，对着弑剑君飞了过去。
<br/>　　“这是给你的赏赐。”
<br/>　　楚风眠向来是赏罚分明的人，这弑剑君一直以来便是忠心耿耿，楚风眠自然不会少了他的好处。
<br/>　　“多谢少主。”
<br/>　　接过那一滴金色血液，弑剑君也不推辞，他曾经便是炼化过一滴远古战龙精血，自然清楚这远古战龙精血的好处。
<br/>　　他连连行礼，便是接过金色血液，一下子吞噬了下去，顿时弑剑君身上的血脉，再次得到了提升，实力又进一步，达到了神力境八重的地步。
<br/>　　凭借楚风眠赐予他的血脉之力，他的实力足以跟真正神力境九重的圣者抗衡。
<br/>　　这样一来，弑剑君踏入生死境，渡过生死大劫，只怕都不再遥远了。
<br/>　　“还有就是，那三大宗门跟武胜学院的几尊巅峰圣者，想要见少主一面交谈。”
<br/>　　弑剑君炼化了金色血液后，便是再次抬起头对着楚风眠开口道。
<br/>　　“哦？带我去看看。”
<br/>　　楚风眠轻声开口道。
<br/>　　倒是并不意外。
<br/>　　这三大宗门，武胜学院中还是有些老家伙的，这些巅峰圣者虽然对楚风眠造成不了什么威胁。
<br/>　　但是楚风眠可不会相信他们会选择乖乖臣服的，于是让他们闹起来造成麻烦，不如楚风眠亲自去彻底解决了他们，也好没有后顾之忧。
<br/>　　处理掉这些人，楚风眠便是也就可以跟寒月漓一同前往北邙学院了。
<br/>　　弑剑君带着楚风眠，顿时来到了一处虚空深处，在此，一共四尊老者，已经是等待许久。
<br/>　　这四人，皆是巅峰圣者，其中二人便是曾经楚风眠见过的。
<br/>　　一个是武胜学院的创始长老，水墨圣者，另一个便是寒雪阁的老祖宗，寒凝圣者。
<br/>　　而另外两尊巅峰圣者，楚风眠虽然没有见过，但是两人一风一火，显然便是烛炎门跟风神宗的老祖宗。
<br/>　　“诸位要见楚某，是为何事啊？”
<br/>　　楚风眠一步踏入到虚空之中，扫视了一眼那四人，平静的开口道。
<br/>　　“楚风眠，你不用在这里装蒜，你如今势大，我们武胜学院，三大宗门的门下弟子，可以全部臣服于你！但是我们四人，可都是巅峰圣者，绝不可能会臣服任何人！”
<br/>　　一道声音，突然传来，那水墨圣者眼神闪动，看着楚风眠，猛然开口道。
<br/>　　“不错，我们都是一方老祖宗的存在！想要让我们臣服于你，门都没有，我们最多可以辅佐你，但是你不可能管束的了我们！”
<br/>　　另一尊巅峰圣者，那风神宗的老祖宗也是开口了，他眼神闪烁，看着楚风眠，言语中露出几分杀意道。
<br/>　　“小子，今日你若是不答应我们的要求，我们要是闹起来，就凭你手下的小猫两三只，还不可能抵挡得住我们！”
<br/>　　“这么说来，你们是想要造反？”
<br/>　　楚风眠眼神平静，扫视了一圈那四人，缓缓的开口道。
<br/>　　“造反？我们只是给你一个忠告罢了，你现在让我们臣服，门都没有，你要是识时务，我们还可以明面上顺服了你，帮你管理一些宗门事物。”
<br/>　　烛炎门老祖宗也是面露凶狠的开口道。
<br/>　　“但是你若是不识时务的话，那可就别怪我们下手无情了！”
<br/>　　那一直没有开口的寒雪阁老祖，寒凝圣者，也是不由的点了点头。
<br/>　　她虽然没有开口，但是如今这幅样子，却是同意这四人说的话。
<br/>　　“不识时务？手下无情？今日我倒是要看看，你们四个废物，有什么本事敢来威胁楚某！”
<br/>　　楚风眠冷笑一声，猛然大手一挥，一股庞大灵力便是轰然凝聚出来，化为了一道巨大无比的手掌，直接压迫过去。
<br/>　　“该死的小子！你太狂妄了！我们都是巅峰圣者，一方老祖，怎么会怕了你！”
<br/>　　水墨圣者看着楚风眠，怒声喝道。
<br/>　　他凝聚灵力，突然出手，在这虚空之中，无数水墨，却是突然出现。
<br/>　　这水墨，化为了一道道凶猛野兽，猛然向着楚风眠扑了过去。
<br/>　　那另外三尊巅峰圣者，也是同时出手，一道道灵力，纷纷打出，向着楚风眠轰杀了过来。
<br/>　　四人联合起来，妄想要抗衡楚风眠。
<br/>　　“不自量力！”
<br/>　　楚风眠冷哼一声，眼神中尽是不屑，他的心神一动，手心中力量凝聚的更加强横。
<br/>　　一掌落下，这四人联合起来，打出的无数武技，被尽数摧毁，镇压。
<br/>　　他们四人联手的实力，还远远不如那赵无敌，如今还妄想跟楚风眠抗衡。
<br/>　　被楚风眠无情镇压。
<br/>　　巨大的灵力，压迫这他们四人的身躯，令他们四人，全部跪倒在地上。
<br/>　　“我连半步生死境的武者都能灭杀，就凭你们几个老家伙，也想要跟我抗衡，来威胁我？”
<br/>　　楚风眠冷眼看向那四人，手心一动，便是一股黑色灵力凝聚出来。
<br/>　　这一股黑色灵力，顿时化为了四道诸天生死印，一下子刻入到了他们四人的内心深处中。
<br/>　　“你休想……！”
<br/>　　水墨圣者奋力的抵抗者，话音未落，他的表情却是瞬间变了，变的恭敬无比。
<br/>　　在这诸天生死印下，他们的心思都会直接改变，全身心的臣服楚风眠，无法反抗。
<br/>　　“我们愿意臣服少主。”
<br/>　　那四人，一齐齐声说道，表情便是恭敬无比。
<br/>　　看着楚风眠如此轻易的降伏了这四尊巅峰圣者，弑剑君的表情都是一愣。
<br/>　　这等实力，简直已经是超出他的想象了。
<br/>　　“这四人，便是交给你了，我要前往北邙学院，这以后的无数事物，便是全部由你负责。”
<br/>　　降伏了那四人，楚风眠便是直接对着弑剑君说了一句，没等弑剑君回答，楚风眠的身形便是已经消失在了虚空之中。
