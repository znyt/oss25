　　果然还是这个问题啊！
<br/>　　李承乾心中尴尬，露出一丝苦笑说道：“父皇，他们其实都是很好的帮手，您看，其实小泰可以主抓教育工作，他的性格比较追求极致，由他主管教育再好不过；小恪可以主管钱财，他很有这方面的天赋；小佑不用说，那小子就是个将军的命，一见打仗不要命；小愔和小治我还没想好，这两个小子太皮，得揍！”
<br/>　　“得揍……”老头子咂了咂嘴，不知道在想些什么，估计是在为那两个小子的命运感慨吧。
<br/>　　但不得不说，老李对李承乾的这个安排还是很满意的，至少这小子有地方安排他的几个兄弟，而且，他最后那一句‘得揍’说的很好，的确是兄长对弟弟最最常用的一种手段。
<br/>　　不老实，不老实就揍你丫的，打到服为止，这比什么圈禁，流放之类的处理方式要强上许多，也是李二对李承乾比较放心的原因之一。
<br/>　　想到这里，老李对李承乾摆了摆手，让他离开，没有再提皇位的事情。
<br/>　　……
<br/>　　其实这种事情是真是假真的很无所谓，李二现在已经是骑虎难下。
<br/>　　李承乾当局者迷，但是一群旁观的吃瓜群众却清楚的很。
<br/>　　以他现在在朝中的影响力，除了老李继续当这个皇帝，否则根本没有人是他的对手，也就是说等老李没了的那一天，这大唐江山到最后还是会落在李承乾的手中。
<br/>　　而既然这样，李二就根本没有必要枉作小人，把李承乾给废掉，再另立继承人，因为那样只会害了他选出来的那个继承者，同时让大唐陷入一场乱局。
<br/>　　而老李继续当皇帝的话……，老实说，他也是要面子的人，既然下面的人都已经开始反对他继续当皇帝了，老李自然也不会在那个位置上继续强留。
<br/>　　所以李二这一次还真的就是给李承乾好好上了一课，让他随时随地都不要对任何人放松警惕，这是一个皇帝要做到的最基本的一条。
<br/>　　哪怕李承乾心中再不情愿，他也必须这样做，否则的话，大唐江山易主也并不是什么不可能的事情。
<br/>　　毕竟这年头儿人心隔肚皮，保不齐就会出现什么‘胸怀天下’的人，如果不小心提防，若真是出了事情到时候连哭都找不着调儿。
<br/>　　……
<br/>　　李承乾回到东宫的时候，发现自己的几个老婆都十分紧张的在等着自己，显然她们几个也得到了宫里的消息，知道事情出了变故。
<br/>　　可是他能说什么，说老头子摆了自己一道？这话说出去被打板子是一定的。
<br/>　　所以最后他只能苦笑着每个老婆都抱抱，然后叹了口气说道：“这次事情已经定下了，李淳风已经去选日子了，你们不必担心。”
<br/>　　“太子哥哥，要不然这太子咱也不当了吧，我们陪着你去浪迹天涯好了。”程琳这丫头冲动一些，心中想着什么就说什么，一点也不顾周围的环境。
<br/>　　苏玫见她说的有些过份，连忙拉了她一下，小声说道：“小四，不要说了！”
<br/>　　皇宫之中人多口杂，类似程琳这种抱怨如果落在有心人的耳中，到时候向李二或者长皇皇后一说，估计被关进宗正寺也不是什么大事，甚至还有可能牵连李承乾。
<br/>　　所以苏玫才会拉住程小四，生怕她口无遮拦之下再说出什么大不敬的话。
<br/>　　而就在几女各怀心事，面色沉重的时候，小武却是轻轻抿了抿嘴没有说什么，显然这丫头心里也有着自己的算计。
<br/>　　李承乾把她的神情看在眼中，心里不由咯噔一下，当下咳了一声说道：“这件事情到此为止，以后谁都不要再提，否则若出什么乱子，自己解决！”
<br/>　　“是！”见他定了调子，几女齐齐答道，包括小武在内。
<br/>　　……
<br/>　　不说李承乾这边如何安慰几个大小老婆，却说另一边李二回到含凉殿，见到紧张等着消息的长孙皇后，脸上这才露出一抹笑容：“观音婢是在担心么？”
<br/>　　“陛下，妾身如何能不担心，你又不是不知道乾儿那孩子……”
<br/>　　“没事的，朕这不是回来了么。”李二轻轻拉起长孙皇后的手，放在自己的手心里拍了拍，有些感慨的说道：“今后这大唐便是他的了，朕算是彻底可以歇歇喽。”
<br/>　　“陛下……”
<br/>　　“什么也不要说了，朕很清楚自己在干什么，只是苦了你，还要跟着朕受委屈。”
<br/>　　“陛下，妾身有什么委屈的，只是怕您受不了。”长孙皇后摇摇头。
<br/>　　她做了十七年的皇后，努力平衡着后宫的一切，使大唐的后宫从未出现过任何争宠之事，同时她也要关心自己的几个孩子，尤其是那个不让人省心的大儿子。
<br/>　　现在如果老李真的不做皇帝了，对她来说其实也是一种解脱，至少以后不用再担心这担心哪。
<br/>　　可是话说回来，这么多年的皇后当下来，眼下突然成了皇太后，从国母的位置上退下来，长孙皇后多少还是有些失落的，而正是这份失落，让长孙很能体会老头子的心情。
<br/>　　……
<br/>　　是夜，李二书房，一个一身漆黑的人影站在老李的身后。
<br/>　　“事情都安排好了么？”李二看着窗外，负手而立。
<br/>　　“回陛下，已经准备好了，方士七十二人，童男童女各三百，已经送往泉州。”人影的声音有些沙哑，听上去没有任何感情。
<br/>　　“没有人知道吧？”
<br/>　　“是的。”
<br/>　　“很好，让你的人小心切，此事不容有失。”
<br/>　　“喏，臣遵旨！”
<br/>　　“去吧！”
<br/>　　尽管李二已经让他离开，但黑影只是微微晃了一下却并没有动，沙哑的声音再次响起：“陛下……”
<br/>　　“什么都不要说，朕很清楚自己在干什么。”李二突然一抬手，止住黑影的话：“你只要做好自己的事情，其他的事自有别人安排。”
<br/>　　“喏！”黑影似乎有些犹豫，但最后还是抱了抱拳，身影闪动间在房间里消失，寂静的房间中，响起李二的喃喃自语：“世界大的很，能人异士无数，朕就不信，找不到那长生之法！”
