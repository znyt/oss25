　　　　虚府，虽然名义上是一座府邸。
<br/>　　但是实际上，却是占据了斗玄城的一方角落，庭院之大，一眼望去，千万宫殿。
<br/>　　这虚家，是斗玄国度最大的家族之一，有着数十万虚家弟子。
<br/>　　楚风眠一行人，跟随虚堇，进入到了虚家之中。
<br/>　　来到了一座宫殿之中，在这宫殿中，便是座着一名中年男子。
<br/>　　这中年男子，看起来只有三十岁出头的样子，一脸威严，高高在上，乃是一尊巅峰圣者。
<br/>　　不过楚风眠察觉的出，此人的实力，却是比起那夺花尊者，都毫不逊色。
<br/>　　虽然是巅峰圣者，但是却还是在积累，随时可以踏入到生死境的人物，实力已经不亚于寻常生死境的武者了。
<br/>　　“父亲。”
<br/>　　那虚堇，看到那中年男子，便是开口道。
<br/>　　一语便是说出了如今这中年男子的身份，他便是虚堇的父亲，如今这虚家的家主。
<br/>　　“听说你遇到了地魔门人的截杀，那夺花尊者，居然敢截杀你！若是他不死，老夫也要去地魔门，给你讨个公道！”
<br/>　　那虚家家主，冷哼一声道，言语中便是极为气愤。
<br/>　　他们虚家，虽然势力远不如地魔门，但是也不是也可以随时拿捏的。
<br/>　　“你们三人，救了小女，说吧，要什么好处，只要不是太过分，我们虚家都可以满足你们。”
<br/>　　虚家家主转过来，打量了楚风眠一行一眼，便是直接开口道。
<br/>　　这虚家家主的语气，便是极为不屑，楚风眠既然是跟着虚堇回来了，肯定是为了索要好处来了。
<br/>　　他们虚家虽然不算家大业大，但是也算是有些底蕴，于是便是直接打算给楚风眠一点好处，便是将楚风眠直接打发走了便是。
<br/>　　“好处就不用了，只不过虚小姐答应过的，北邙学院考核资格的名额，给我们，我们就直接离开便是。”
<br/>　　楚风眠平静的开口道。
<br/>　　他也懒得跟这虚堇纠缠什么，如今他来就是为了名额来的。
<br/>　　只要是得到了北邙学院考核资格的名额，楚风眠一行人直接离去便是，不需要任何的停留。
<br/>　　“北邙学院考核资格的名额？”
<br/>　　虚家家主听到楚风眠的话，不由的眉头一皱，刚刚开口，想要说些什么。
<br/>　　虚堇却是在一旁，插话一声道。
<br/>　　“父亲，这几位救了小女一命，便是小女的恩人，不招待一下，还是有失礼数吧。”
<br/>　　“我们千里迢迢回来，至少也要摆宴接风吧。”
<br/>　　虚堇说完，便是转过头对着楚风眠说道。
<br/>　　“楚公子可以现在府上住下，这北邙学院考核资格名额的事，小女既然答应了，便是不用着急，等到晚上，便是设宴，为各位接风。”
<br/>　　“华瑞，你带这三位贵客先住下吧。”
<br/>　　虚堇对着旁边的一尊站着的管家，开口道。
<br/>　　这名叫做华瑞的管家，便是带着楚风眠，直接离开。
<br/>　　一直等到楚风眠一行人离开，这虚家家主，脸色才露出几分怀疑，看向虚堇开口道。
<br/>　　“瑾儿，这一次北邙学院考核的资格，我们已经分发下去了，选出了其中的弟子，你怎么会答应他这一点。”
<br/>　　“更何况，这几人看起来实力平平无奇，根本也不值得拉拢，就算是他们救了你，给他们点好处就把他们打发走了便是。”
<br/>　　虚家家主开口道，他一眼便是看出来，楚风眠三人的境界。
<br/>　　就连巅峰圣者都不是，神力境七重的境界，虽然不差，但是在虚家也是可以找出一大把来的。
<br/>　　根本不值得拉拢。
<br/>　　要知道一个北邙学院考核的资格，足以拉拢一尊圣者巅峰的人物的，价值并不低。
<br/>　　“父亲，你觉得我真的会将北邙学院的考核资格交给他们？”
<br/>　　虚堇的脸上，露出几分冷笑道。
<br/>　　“我不过是算计他们，引诱他们过来罢了，夺取他们手中的须弥空间罢了。”
<br/>　　她的话音，恢复了之前的阴冷，一言一句之中，皆是算计，狠毒。
<br/>　　“他们三人，不知道是什么好运气，居然是杀了夺花尊者，将夺花尊者手中的须弥空间拿走了。”
<br/>　　“什么？须弥空间？”
<br/>　　虚家家主听到这须弥空间的一刻，脸色也是骤然惊变道。
<br/>　　“这不是地魔门中的无上宝物吗？据说是由夺天老祖保管的，怎么会在夺花尊者的手中，而且如今居然是落入到了那几人手中。”
<br/>　　“那还耽误什么，快快将这须弥空间抢来便是，须弥空间这等宝物，我们虚家要是可以得到，上供给北邙学院，只怕是会有着无穷无尽的好处。”
<br/>　　虚家家主的脸色都是变的贪婪无比，须弥空间，这等宝物，人人都想要得到。
<br/>　　就算是他们得到，自己无用，若是上供给北邙学院，也足以换取无穷无尽的宝藏，来支持他们虚家壮大。
<br/>　　“没有这么简单，那三人虽然境界不高，但是那尊女子，却是实力极强，我跟她交过手，一招便是必败无疑。”
<br/>　　虚堇缓缓的开口道。
<br/>　　“硬抢，只怕是两败俱伤。”
<br/>　　“那女子的实力，居然足以一招击败你？她可是神力境七重的境界啊，你的实力，巅峰圣者都可以击败，怎么会有人可以一招击败你。”
<br/>　　听到虚堇的话，虚家家主也是大吃一惊道。
<br/>　　虚堇的实力他自然是清楚的，寒月漓可以一招击败虚堇，的确有些令人不可思议。
<br/>　　“而且这些人手中，有着大笔的财富，我亲眼见证那男子拿出了百亿九鼎丹来，他身上的财富，应该是远远不止，我怀疑他甚至可能得到了什么大奇遇，远古宝藏，才会有着如此财富。”
<br/>　　虚堇缓缓的开口说道。
<br/>　　“如今我们还不好直接跟他们为敌，可以先试探一二，今日晚宴，父亲便是可以去请来这斗玄城中其他两大家族的人来，借助他们的力量来试探一下。”
<br/>　　“若是这几人没有实力，我们直接杀了便是，若是有着实力，资质极高，我们也借此机会，拉拢他们。”
<br/>　　虚堇开口道，一切的算计，便是天衣无缝。
<br/>　　在她的算计中，一切结果都是她逞心如意。
<br/>　　“好，就由你说的去做。”
<br/>　　虚家家主眼神闪烁，便是直接开口，身形消失。
