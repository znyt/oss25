　　三年之后，昆仑山中。
<br/>　　三清盘立与虚空之中，不动如山，给人一种威压天地的感觉，就好似一道大山，横立于此。元始依旧原样，他看着老子与通天，淡淡道：
<br/>　　“开始了？”
<br/>　　“嗯。”两人点点头。
<br/>　　“谁先来？”元始眉毛一翘，看着两人问道。
<br/>　　“还是贫道先来吧。”
<br/>　　老子看了通天一眼，见通天眼中闪过跃跃欲试的神情，顿时说道。大兄已经在他之前成圣了，他若是再让通天这个弟弟给超越了，那还怎么玩？
<br/>　　“也罢，谁先来都一样。”元始微微点头道。
<br/>　　老子不再多言，他闭上双眼，养精蓄锐，调养好状态，是身心放空，渐渐到了一种无欲无求的状态，空而不空，与天合一，天地交融。
<br/>　　渐渐地，他的身影越发的淡然了，在他的身上散发着一股若有若无的波动，他此刻就好似一块无情的寒冰，全身上下没有一点人的意味。
<br/>　　呼吸之间，便有无穷的天地灵气倒卷而来，九天之上，星辰之力顺势而来，呈现出五彩缤纷的周天星力，落入老子身上。
<br/>　　在他的身上，渐渐出现两条黑白双色的阴阳鱼，刚开始的时候，它们还是只若有若无，随着时间的流逝，它们渐渐变得宛若实质一般。而且最开始时，它们也只是在老子的头顶游动，但渐渐地，它们却变得越来越大，直至遮蔽了这片天地。
<br/>　　霎时间，整片天地全部消失不见，只剩下黑、白两色，阴阳流转，一动一静，阴极生阳，阳极生阴，阴阳互化，造化相生。
<br/>　　阴阳渐渐演化出天地人三才，之后是地水火风四象奔腾，再之后金木水火土五行相生，幻化出各种各样的异象，之后，六合，七星、八卦、九宫，一一演化，惊天动地。
<br/>　　最后，这些异象有再次归一，返回之前阴阳太极状态，渐渐没入老子身体之中。一切好似变得平静下来，就仿佛之前那幕并没有出现过。
<br/>　　元始与通天在一旁看着，一人成圣的过程，这可并不多见啊，这样难得的机会，可要好好把握。若是他能够从其中学到经验方法，那才是最好啊。
<br/>　　虚空中，老子盘坐在那里，却显得愈发的虚无缥缈，神识不可测，只可用肉眼去看。他全身上下散发着一股淡然无为的气息。
<br/>　　轰！
<br/>　　蓦然间，从虚空之中，走出三道身影，一道穿着白色道袍，头顶十一品净世白莲，与老子有七八分相似，他面带微笑，正是老子的善尸。
<br/>　　“贫道太上老君。”
<br/>　　另一道身穿黑色道袍，手拿灰色石印，一股强烈的生死之意流转其内，他怒目而视，正是老子的恶尸。
<br/>　　“贫道混元上帝。”
<br/>　　最后一道身穿素色道袍，手持太极图，脚踏金桥，他面色无情，淡漠无比，正是老子的执尸。
<br/>　　“贫道道德天尊。”
<br/>　　三道身影，正是老子的善、恶、执三尸，看他们一身修为，已经全部在准圣后期了。
<br/>　　之后，在元始与通天的注视下，善尸太上老君微微一笑，来到太清老子身边，渐渐变得虚幻，融入到他的身子之中，之后，化为一道白色清流，直奔太清老子头顶的三花，速度快到不可思议。它一进入其中一朵白花中，那白花顿时便向着四面开放，变得更加神韵，其上道韵非凡。
<br/>　　一道缩小了不知多少倍的太上老君的幻影端坐在那盛开的白色花朵之中，他闭目诵经，好似在传颂大道妙法。
<br/>　　之后，他的恶尸混元上帝一样，如之前太上老君那般，融入到老子身体之中，化为一道黑色清流，没入太清老子头顶的黑色花朵之中。同样，在黑色清流进入到黑色花朵后，那花朵一样开放得妖异无比，一个同样缩小无数倍的略显虚幻的混元上帝端坐在其上，闭目诵经。
<br/>　　最后，太清老子的执尸道德天尊一样如此。三花齐放，三道虚幻的声音在其中传颂出各种大道经文，演化出各种惊天异象。
<br/>　　轰！
<br/>　　太清老子身上的气息在迅速攀升着，准圣后期，准圣圆满……
<br/>　　他一直来到准圣圆满境界，而且那道强大的气息还在飞速提升着，一点也不见停下来的意思。
<br/>　　他头顶的三花渐渐开始枯萎，开始结出三颗微小的道果，围绕在他头顶的庆云之间不断旋转，演绎出各种道韵。
<br/>　　最后，那三颗微小的道果开始融合，它们融合在一起，形成一颗闪烁着黑、白、金三色光彩的道果，而且这道果也比之前的道果要大一倍。
<br/>　　“混元一体！”
<br/>　　老子大喝一声，他头顶的那颗道果渐渐散发出耀眼无比的光彩，之后，光彩消失，只留下一颗散发着微弱光辉的道果。它在那庆云之中缓缓旋转。
<br/>　　但随着时间的流逝，在那道果之间渐渐出现一些神纹，道纹，仙纹。各种各样的道文都有，最后，那整颗道果都布满的神纹。道果开始变成黑白两色，一半黑一半白，好似两条首尾相连的阴阳鱼抱团而居。
<br/>　　它开始旋转，并且越转越快，到最后用甚至看不清它的速度，只觉得一阵模糊的光团在眼前闪过。
<br/>　　一股属于阴阳的大道道韵，开始弥散开来。
<br/>　　这时，一条闪烁着耀眼紫金光芒的紫气迅速飞入，并飞快没入那道果之中。
<br/>　　轰！
<br/>　　太清老子全身气势大盛，他好似突破的准圣圆满，来到了一个全新的境界，他的神魂飞出自己的身体，向着九天云霄飞去，渐渐地，越飞越高，穿过层层虚空，不知过了多久，他来到一个特殊的空间之中，他看着前方不远处的一团光源，就仿佛是看到了比太阳还要耀眼的存在。
<br/>　　此时，那光源之中，飞出一股无色清流，飞速没入老子的神魂之中，老子觉得自己的神魂力量在迅速被改变，它被改变成了真正散发着无尽圣威的圣魂，而且其中还有浓郁的天道之力。
<br/>　　不知过来多久，老子睁开眼睛。
<br/>　　“吾，太清，今日成圣！”
<br/>　　“洪荒之中，吾为太上道主！”
<br/>　　轰！
<br/>　　一股强烈无比的气势以昆仑山为中心，向着洪荒大地扩散而去，并迅速充斥着整片洪荒宇宙。一时之间，洪荒准圣以下者，全部跪伏在地。
<br/>　　紧接着，在众人还没有醒悟过来之际，天降金花，地涌金莲，天花乱坠，道音袅袅，瑞气飞舞，霞光道道。
<br/>　　圣威惊动九重天，紫气纵横三万里！各种异象惊天动地。
<br/>　　这种异象已经不是第一次出现了，但它每次出现，都能带给人们莫名的震撼，洪荒众生在此刻都知道，天地间要再次增加一尊万劫不灭的圣人了。
<br/>　　洪荒众生对着昆仑山方向叩拜下去，高呼：
<br/>　　“圣人无量！”
<br/>(本章完)
