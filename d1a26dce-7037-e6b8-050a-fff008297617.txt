　　第363章
<br/>　　程大雷向西而去，荒野只剩下孤单单两个人，夜里看不见光，远处不知什么野兽在吼，发出阵阵怪叫。
<br/>　　苏樱和樊梨花相对而立，互相都是有些尴尬的。樊梨花手中举着火把，脸颊火光明灭，却不知该说什么好。
<br/>　　“樊家姑娘。”终究是苏樱情商略高些，微微道了个万福：“麻烦你了。”
<br/>　　樊梨花顿了片刻，最后吐出两个字：“走吧。”
<br/>　　从这里到琴川关还有不远的距离，来的时候都未骑马，所以回去只能选择步行。
<br/>　　樊梨花右手袖着短刀，左手举着火把，走在苏樱后面。如此，若有意外情况，她可以第一时间察觉。
<br/>　　一路沉默，谁也没有说话，只要脚步踩过枯草时发出的噼啪声。
<br/>　　苏樱自然察觉到程大雷和樊梨花之间的异样，但她并不准备说什么。虽然如今的女人都大度，但苏樱还没大度到要把其他女人推到程大雷身边的程度。
<br/>　　她小步赶路，身体又虚弱，自然走得不是很快。樊梨花心里还是有几分着急的，程大雷想要找到那个闯入琴川关的恶人，樊梨花何尝不是。
<br/>　　等将苏樱送回琴川关，她打算立刻原路返回，去追程大雷。
<br/>　　只要发现对方，自己就有把握杀掉对方，现在困难只是在找不到对方而已。
<br/>　　她袖里藏着刀，磨得锋利，三步之内，有把握切断任何一人的喉咙。
<br/>　　说话和做事，樊梨花的确不擅长，她并不是不知道这一点。她真正擅长的，是杀人。
<br/>　　一路返回琴川关，立刻将苏樱交给城门守卫，樊梨花要了一匹马，立刻跑了出来。
<br/>　　樊梨花来得很快，即便在夜幕中她也保持着极快的速度。这荒原没有什么障碍物，极适合起马奔驰。
<br/>　　樊梨花的目的是先找到程大雷，程大雷离开时拿着火把，在这荒野上如一盏孤灯，想要找到他也并不困难。
<br/>　　等二人相遇后，再一起去寻找那魔头，当然，如果是让自己单独遇见就更好了。
<br/>　　前方忽然出现一道黑影，只是手里没有火把，但就那样生生的挡在路上。
<br/>　　大概是火把燃尽熄灭了，樊梨花轻松口气，勒马在其面前停下。
<br/>　　“我已经把人给你送回去了，放心，她一点事都出不了，你等在这里干嘛，找到什么线索没有？”
<br/>　　黑影抬起头，露出一口惨白的牙：
<br/>　　“我在等你呀。”
<br/>　　……
<br/>　　程大雷按照苏樱所指的方向去寻找那恶人，他自然也明白，茫茫荒野，风可以随便埋没一个人的痕迹，想要找到一个人谈何容易。
<br/>　　但程大雷隐隐有种感觉，自己在今夜就会遇到对方。大概是同类之间的感应，钟伟虎不都说了么，自己也是凉州一害。
<br/>　　上山虎碰着下山虎，出海龙碰着入云龙，两兽相争，必有一死，说不得，对方是个可以和自己匹敌的对手。也不知是自己快剑割破对方喉咙，还是对方撕破自己的胸膛，取出热气腾腾的心脏。
<br/>　　想到这里，程大雷竟然隐隐兴奋起来，胸腔里的血已开始变得滚烫。
<br/>　　……
<br/>　　黑影立在路当中，身上裹着一件黑斗篷，刚才距离太远，樊梨花有些看不清楚，但等开口时已经确认对方不是程大雷。
<br/>　　而当对方抬起头，从斗篷中露出一张惨白的脸时，凭直觉樊梨花已经判断出对方的身份。
<br/>　　那张宛若从坟墓中爬出来的脸，身上刺鼻的血腥味，最关键是对方一双眼睛。
<br/>　　草原上有太多野兽，灰狼、豹子、豺狗……樊梨花在很小的时候，就能和这些野兽捉对，对野兽的眼睛并不陌生。
<br/>　　其实野兽和人并无多少不同，灰狼在捕食黄羊时并不会激动，就像人在杀鸡时也谈不上如何残忍。
<br/>　　面前这人的眼睛，和杀鸡之前并没有太大不同。
<br/>　　没有这样的眼神，大概也不可能杀人如杀狗，以一己之力，屠尸过百。
<br/>　　在确认对方身份后，樊梨花笑了。
<br/>　　“你在等我么，恰好，我也在找你。”
<br/>　　声音落地，身体便从马上扑了过去，右手反握刀柄，去扎对方喉咙。
<br/>　　樊梨花当初能够挑落秦蛮、张肥、高飞豹，这其中自然有大家看他是个女流之辈不好下手的意思，但同样也可以证明她的本事。
<br/>　　出手就是要害，出手就要杀人。
<br/>　　这是电光火石的一刀，刀斩出去，樊梨花已从马上站在马下。自己并未带自己的绣绒刀，还是马下步战方便些。
<br/>　　可在一刹那，樊梨花就愣住，她的刀就像陷入铁板中一样，动弹不得。
<br/>　　黑影用手掌握住了她的刀锋，血汩汩流出，可对方就像失去痛觉一般，毫无察觉，脸上还挂着笑容。
<br/>　　“你的脸好漂亮，好女孩是不该玩刀的。”
<br/>　　一股寒意从樊梨花心底泛起，她瞬间就想和这怪人保持距离，哪怕是弃刀。
<br/>　　可她已经没有离开的机会，怪人握住了她的手腕，樊梨花感觉骨头都要被捏碎了。血淋淋的抓住自己的手，像毒蛇缠绕住身体。
<br/>　　瞬间，她就感觉身体悬空，自己被怪人拎了起来。
<br/>　　不应该是拎了起来，而是直接抡了起来，身体以圆周旋转，眼前天昏地暗，还未回过神来，身体已失重凌空，咚地一声，重重跌在十步外。
<br/>　　樊梨花咬着牙，避免自己喊出声来。可血已经浸湿她的牙床，五脏六腑仿佛被肋骨扎穿，全身骨骼像是已经散架。第一时间，樊梨花的右腿小腿骨已经被摔断。
<br/>　　只是一招，她就完全失去了战斗力。
<br/>　　这人好大的力气，同样拥有出色的速度，刚才的一瞬间，樊梨花根本来不及做出任何反应。
<br/>　　怪人一步步向樊梨花过来，苍白脸上表情麻木，可无形中却带给樊梨花巨大的恐怖。
<br/>　　她挣扎着从地上爬起来，用一只腿撑地，踉跄向后腿。
<br/>　　怪人的脚步不紧不慢，不断迫近樊梨花，可也没有着急靠近她。
<br/>　　现在樊梨花像落入他手中的雀，他只等着对方筋疲力尽时，才会想折断对方的喉咙。
<br/>　　“你这张脸很漂亮，可你不该把他弄脏。”
<br/>　　怪人一字一顿，樊梨花冷汗如雨。
