
<br/>　　<br/>　　    院子里，胡胡跟老者聊了半个多小时就告辞了，临走之前他跟对方说道：“其实，前段时间本来有个挺好的机会，南昆仑的崔玄策往外透出了个消息，说是一个叫王惊蛰的人，身上可能会有连山和归藏，后来偶然间我也碰上了这个人，当时常子敬和齐坤跟对方对了几卦，我们就已经推断出了，他确实应该懂得连山和归藏，玄门本来做好准备，五脉共同出手，从他身上逼出这两书来，然后由玄门五脉一起研究，本来打算的挺好，可谁知天有不测啊，中元鬼门开的时候这个王惊蛰进了阴曹地府没能再出来，到如今差不多被困能有两个多月了，大好的机会就这么错过了”<br/>　　<br/>　　    老者看了他一眼，皱眉说道：“真有这回事？那两书失传都多少年了，突然凭空冒出来，这个可能性有多大？”<br/>　　<br/>　　    “王惊蛰是赊刀人……”<br/>　　<br/>　　    老者顿时一愣，张了张嘴一脸的惊讶，片刻后说道：“这么说的话，可能性确实很大了，赊刀人近几十年不太出世，这个年代的人了解的都不太多，但从以往的传言上来讲，这帮卖菜刀的通风水走阴阳，铁口直断，倒是很有可能懂得连山和归藏，不然他们哪来的那么大能耐？”<br/>　　<br/>　　    “不过，说什么现在都晚了”胡胡感叹着说道。<br/>　　<br/>　　    没过多久，胡胡跟老者边走边聊走出了院子，跟对方告别后，他上了停在路边的车。<br/>　　<br/>　　    离胡胡不远，王惊蛰，菜刀文和林汶骐眼睛都盯着他的背影，林汶骐手指摩挲着一把锃亮的尖刀，皱眉说道：“多好的机会，我直接过去捂着他嘴捅上两刀然后掉头就走，这不就完事了么，还非得要等什么机会啊？”<br/>　　<br/>　　    “你现在要是过去就把人给捅死了，那无疑是把沈君给泡了，因为他没办法和上面的领导交代”王惊蛰拍了拍他的肩膀说道：“还有，咱们也不是职业杀，不能这么干，咱们要碰这个胡胡啊，得需要有正当的理由才行，至少不能给别人怀疑的机会，是和沈君有关系的，总而言之就两个字……找茬！”<br/>　　<br/>　　    胡胡上了车，丝毫没意识到，背地里有几双眼睛在审视着他。<br/>　　<br/>　　    林汶骐有点不甘的说道：“真还要在等等啊，这么好的机会错过了就白瞎了”<br/>　　<br/>　　    “等吧，机会挺好，但时机不合适”<br/>　　<br/>　　    “这么束手束脚的真不是我风格，哎”林汶骐摇头叹了口气，明显觉得事情这么办有点不顺心，他最崇尚的就是手起刀落，快意恩仇，而不是还得整出个什么作战方案来。<br/>　　<br/>　　    胡胡上了车后，一手把着方向盘，踩着油门，一手拿出电话拨了出去：“喂？你们在哪呢，我出来了”<br/>　　<br/>　　    “正谈着事呢，马上就要完了”<br/>　　<br/>　　    胡胡说道：“找个地方见一面，咱们再商量下吧，我这边也刚出来”<br/>　　<br/>　　    “好，茶楼见……”<br/>　　<br/>　　    胡胡开车离开了老城区，随后菜刀文开着车就跟上了，怕离的太近被发现，他们这辆车就一直落后几个车位，控制在能看见对方的后备箱就行了。<br/>　　<br/>　　    半个多小时后，胡胡的车来到一条商业街附近，停在一处车位里后就下了车，然后进了一家茶馆里。<br/>　　<br/>　　    “他应该是来见什么人的，不然自己一个人，喝的什么茶啊？”小草看着胡胡进了那间茶馆说道。<br/>　　<br/>　　    王惊蛰抻着脖子说道：“那正好了……”<br/>　　<br/>　　    胡胡进了茶馆，来到一个包间里，他进来的时候里面已经坐了两个人，一个是齐峥坤，另外一个死命字脉的常子敬。<br/>　　<br/>　　    “跟他谈的怎么样了，茶还没上来，先喝点水吧”齐峥坤端起茶壶问道。<br/>　　<br/>　　    “别给我倒了，在那就喝了一肚子的水，咱们就赶紧说点干货的事情吧”胡胡摆了摆手，脸色不太好看的说道：“老人家么，年纪大了明显顾忌比较多，前怕狼后怕虎的也正常，他这个年纪了就是想混几年然后慢慢等着进棺材就行了，肯定不想往自己的身上揽事了，明哲保身吧”<br/>　　<br/>　　    常子敬说道：“这么一来的话，少了他的支持，怎们的进度可就挺难了啊，这帮老家伙别看都退了，但门下子弟众多，没有他们登高一呼的话，玄门五脉想要拧在一起，就不那么容易了”<br/>　　<br/>　　    “如果，花钱买通呢？”齐峥坤接着问道。<br/>　　<br/>　　    “给多少啊？几百万，一千万还是上亿啊？根本就不是钱的事……”胡胡烦躁的叹了口气，问道：“不说我这里了，以后我在研究他吧，你们两个那边谈的如何了？”<br/>　　<br/>　　    隔壁的包间里，林汶骐和菜刀文把门开了一条缝隙，眼神撇着门外，过了片刻之后一个侍应生端着茶盘就走了过来，到了隔壁的房间外，刚要敲门，菜刀文就把自己包间的门给开开了。<br/>　　<br/>　　    “唉，你过来一下”菜刀文冲着侍应生招了下手。<br/>　　<br/>　　    “您好，先生？”<br/>　　<br/>　　    “我们要的茶呢……”<br/>　　<br/>　　    “哦，您的要稍等一会了，我们后面正在准备，马上就给您送过来了”<br/>　　<br/>　　    菜刀文指了指他手里的茶盘，说道：“这是给谁的茶啊？”<br/>　　<br/>　　    “您旁边房间里客人点的”<br/>　　<br/>　　    菜刀文斜了着眼睛说道：“明明是我们先来的，为啥先给别人啊，你们茶馆就是这么做生意的啊？”<br/>　　<br/>　　    侍应生连忙解释道：“先生，您来的要晚一点，隔壁房间的客人早就过来了”<br/>　　<br/>　　    “别扯，我们进来的时候旁边还没有人呢，就是我们先来的，你把茶盘放下给我们，那个屋子里的等着下次再送吧”<br/>　　<br/>　　    侍应生说道：“先生，真的是他们先来的，比您早到了十几分钟呢，您就是多等一会也要不了多久的，最多十五分钟我们就能把水沏好了”<br/>　　<br/>　　    “一分钟我都等不了，我是个言出必行的人……”<br/>　　<br/>　　    这边，菜刀文跟侍应生争执了几句，隔壁房间胡胡他们明显也是等了半天还没人送茶水，就打开门叫了两声侍应生。<br/>　　<br/>　　    侍应生明显有点急了，连连点头说道：“先生您稍微等下，我把茶叶给隔壁送过去，然后马上下楼把您的给端过来，稍等，几分钟就行了”<br/>　　<br/>　　    侍应生说完掉头就走，林汶骐无语的说道：“找茬找成这样，也是够埋汰的了”<br/>　　<br/>　　    菜刀文呲着牙笑了：“管他呢，怎么无赖，怎么来呗”
