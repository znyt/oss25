　　清理门户！
<br/>　　此时战术中队每一个人的心情就跟肖天行在刚得知消息时的一样，纠结，矛盾，还有……痛心！
<br/>　　慕容飞还有不到一年的时间就要从战术中队退役，他为龙隐部队为国家奉献了自己的青春，可是他为什么会在这个时候，去做这样一件糊涂事。
<br/>　　“无常从佟千雪那里得知是普提查的手下导致她流产，十分后悔当初放走普提查，要为佟千雪和他死去的孩子讨回一个公道。”
<br/>　　赵天宇跟慕容飞保持了一样的说法，说道：“于是他找到了一个代号‘司机’的境外军火贩子，打算从他手中购买军火，私自出境去暗杀普提查。‘司机’是境外某军火贩卖集团在亚洲的代理人，因为一次境外的军火交易遭到追杀，不得不逃入我国境内，国安方面盯着这个家伙已经很长时间了。”
<br/>　　“根据国安方面提供的‘司机’的口供，无常本来已经跟‘司机’谈妥了价格，可是在交易的时候‘司机’临时要求无常把自己也带出境，无常不答应，双方在争执中，无常开枪把‘司机’打成重伤。现在情况你们都清楚了吗？”
<br/>　　“清楚了……”
<br/>　　战术中队队员声音低沉的回答。
<br/>　　“全都焉了吧唧的，没吃饱饭啊！？”赵天宇瞪着战术中队，大声说道：“谁要觉得自己无法执行这次行动的，现在就站出来，别出去以后给我丢人，我立刻换人！我就不信整个龙隐部队除了你们就没人能执行这次行动了！”
<br/>　　“报告！”罗昊站直身体，大声回答道：“我们可以执行这次行动！”
<br/>　　“可以就最好。”赵天宇点了点头，说道：“时间紧迫，你们的装备已经被送上了直升机，暗影在直升机上等你们，你们现在就出发跟暗影进行汇合，具体行动细节会由暗影跟你们细说，出发！”
<br/>　　尽管心中有诸多不愿，可最终战术中队还是踏上了前往东海市的直升机。正如赵天宇所说的那般，即便他们不愿意执行，也会有其他人去完成这次行动。如果他们能劝阻慕容飞回头的话，他或许还有回头之路。
<br/>　　一架米-171运输直升机在一架武装直升机的护卫下，直奔东海市而去。
<br/>　　“暗影，这次怎么是你亲自带队？”罗昊对着云岳喊道。
<br/>　　“我带队难道有什么问题吗？”云岳对着罗昊反问。
<br/>　　罗昊摇了摇头，回答道：“我只是觉得有些好奇而已。”
<br/>　　云岳扭头看了眼直升机外，随后把一部平板电脑递给罗昊，说道：“上面就是这次‘行动’的细节。记住只准看，不准说！”
<br/>　　罗昊有些疑惑地从云岳手中接过了平板电脑，翻看起平板电脑上的“行动简报”。罗昊脸上的表情也是从最开始的错愕，逐渐变得震惊，他抬头看了眼坐在自己对面的云岳，随即轻轻摇了摇头，他现在算是想明白为什么自己老婆会有那么高的智商了，有云岳这样的老爸，想变笨也是件很困难的事情！
<br/>　　把平板电脑上的“行动简报”看完，罗昊面无表情的把平板电脑递给坐在自己身边的齐凡。
<br/>　　齐凡接过平板电脑，也是跟着阅读起来，只是看了还不到两页，齐凡便是充满震惊的看向云岳，正想要对齐凡开口询问，云岳便是说道：“记住，只能看不能问，看完就给下一个人！”
<br/>　　齐凡点了点头，仔细地把平板电脑上的“追魂”行动细节看完，然后把平板电脑递给下一个人。
<br/>　　每个人看到平板电脑上的“追魂”行动细节，全都是跟罗昊和齐凡一样的表情，但是很快就又把表现在脸上的震惊和诧异掩盖下去，把“追魂”行动计划书的每个细节都仔细地记下。
<br/>　　距离东海市还有一半路程，所有战术中队队员已经全都看完了“追魂”行动的计划书，平板电脑也已经回到了云岳的手中。
<br/>　　“行动简报都已经清楚了？”云岳脸上带着微笑看着直升机机舱中的战术中队队员，问道。
<br/>　　“清楚！”
<br/>　　战术中队队员脸上此刻的表情已经是变得无比轻松，总算是从之前纠结和矛盾中摆脱了出来。
<br/>　　“那我们的武器装备？”
<br/>　　陈应天突然想到一件事情，立刻拔出自己战术背心中的弹匣。
<br/>　　“你们战术背心中的弹匣，除了最左侧那个弹匣是实弹，其他弹匣中全都是空包弹。”云岳对所有战术中队队员叮嘱道：“记住，不要在行动中拿错弹匣，拿错了弹匣，无常就真的死了！”
<br/>　　“放心，这个怎么会错呢？”陈应天很是自信的说道：“专业的，奥斯卡级别的演员！”
<br/>　　“不过上面那个牺牲品谁来做？”周柏对云岳问道：“虽然是炸点，可是那玩意儿还是很疼的。”
<br/>　　“自己翻看你们的背包。”云岳说道：“谁的背包中有血袋，谁就是牺牲品。”
<br/>　　每个人都开始翻看自己的背包，他们在给菜鸟进行入队考试时，扮演“恐怖分子”的时候，全尝试过炸点，这东西虽然不会造成什么危险，可是被炸一下还真是挺疼的，所以没有人想被当做被慕容飞狙击的目标，然后被来上这么一下字。
<br/>　　“我靠！”
<br/>　　司徒空从自己的背包中拿出一个血袋，一脸郁闷的说道：“怎么又是我啊！？”
<br/>　　“龙猫，恭喜你‘中奖’了！”
<br/>　　坐在司徒空身边的周柏，一脸幸灾乐祸的说道：“恭喜！恭喜！”
<br/>　　“恭喜你个鸡蛋饼啊恭喜！”司徒空翻了个白眼，把血袋塞进自己的迷彩服中，说道：“上次考核那群菜鸟啊，你们身上都一个炸点，我身上五个，为什么这些‘好事’总是轮到我？”
<br/>　　“能者多劳嘛。”罗昊对司徒空说道：“别郁闷了，既然中奖了，就享受你的奖励吧。”
<br/>　　“你们这些家伙，真是好兄弟啊！”司徒空把迷彩服的拉链拉好，撇了撇嘴说道。
<br/>　　云岳抬手看了眼手表，说道：“还有一个小时，所有人做好准备！”
