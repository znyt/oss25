　　出了裂天剑派所属森林，便是四季城。
<br/>　　萧逸一路飞奔，直接去了百武城，并去了一趟猎妖殿。
<br/>　　三级以上的妖兽，一般在猎妖殿都会有资料，记录其属性、危险程度，乃至在哪里出没。
<br/>　　这些资料都是免费的，任何人都可以去查阅。
<br/>　　查阅完毕，萧逸大概了解这些妖兽的情况后，刚准备离开。
<br/>　　却见任务处的任务栏上，有一条白家发布的寻人任务，寻的，正是易霄。
<br/>　　萧逸眼睛一亮，想来是白家有六宝玲珑参的消息了。
<br/>　　离开猎妖殿，萧逸寻了个无人的角落，换了一身劲装，戴上幽魂面具，才再次回到猎妖殿，并将这条任务接下。
<br/>　　查阅资料，无需出示令牌，任何人都可以查阅，所以萧逸以自己的身份去看就可以了。
<br/>　　但接任务，查看任务信息，就必须出示令牌了，他只能恢复到易霄的身份。
<br/>　　猎妖殿的令牌，记录着武者的气息，任何人都只能拥有一块。
<br/>　　萧逸是无法申请第二块的。
<br/>　　言归正传。
<br/>　　接过任务，萧逸看了眼任务的信息，白家的留言是，让他在百武城内等待。
<br/>　　任务处的工作人员，这时也道，“易霄，这个寻人任务，你已自己完成，但没有奖励点；稍后，会有猎妖殿人员去白家通知任务已结束”
<br/>　　“嗯。”萧逸点点头，而后离开猎妖殿，随意找了间客栈住了下来。
<br/>　　.....
<br/>　　半天后。
<br/>　　白家主、白管家，以及十数个白家武者出现在客栈中。
<br/>　　“白家主。”萧逸拱了拱手。
<br/>　　“呵呵，易霄小兄弟不必多礼。”白家主笑着说道。
<br/>　　“白家主。”萧逸道，“易某还有要事在身，就直说了，六宝玲珑参可有什么消息？”
<br/>　　“没有。”白家主摇摇头。
<br/>　　“没有？”萧逸一愣，那这白家主发任务叫自己来做什么？
<br/>　　“哈哈。”白家主忽然大笑道，“六宝玲珑参没有消息，可是....”
<br/>　　白家主顿了顿，道，“八宝玲珑参，却有消息。”
<br/>　　“八宝玲珑参？”萧逸一惊。
<br/>　　八宝玲珑参，和六宝玲珑参是一样的东西，但效果更大，品阶更高，也更珍贵稀罕。
<br/>　　八宝玲珑参，已经可以列入五品天材地宝。
<br/>　　“在哪？”萧逸问道。
<br/>　　“别急。”白家主笑道，“两天后，百武城将举行一场盛大的拍卖会，由本土的两个商业家族，以及盛宝商行联合举行。”
<br/>　　“八宝玲珑参，正是此次的拍卖品之一。”
<br/>　　“你这两个月，销声匿迹。我还以为你去了其他城池呢，本来打算这两天你再不出现，我就先买下来的。”
<br/>　　“现在你出现了，正好，可以一起去拍卖会。”
<br/>　　这次的拍卖会，物品阶品之高，有些东西连白家都需要。
<br/>　　“拍卖会。”萧逸皱了皱眉头。
<br/>　　白家主看出萧逸所想，道，“我知道你和北山慕容家的事，百武城盛宝商行的几个执事，也在通缉你。”
<br/>　　“不过，这里毕竟不是郡都。盛宝商行的势力，远不如我们白家，你可以放心。”
<br/>　　“要不然，若你不方便的话，我用白家的名义替你买下来，你不去也可以，在客栈等我们回来即可。”
<br/>　　萧逸摇摇头，笑道，“谢白家主好意了，易某也想去见识见识，便一并去吧。”
<br/>　　.....
<br/>　　两天后，盛宝商行。
<br/>　　萧逸跟着白家一行人，坐到了早已准备好的贵宾席上。
<br/>　　这次的拍卖会，盛况空前，一般的人根本没资格来。
<br/>　　来的，都是些百武城附近十数城有名的武者。或某家族的家主，或某城城主，或出名的猎妖师、独行武者等等。
<br/>　　而贵宾席，只有五个，分别属于五个大势力。
<br/>　　百武城顾家，赫然也在此列。
<br/>　　数个时辰后，宾客纷至沓来，直至将拍卖会完全坐满，拍卖才算开始。
<br/>　　拍卖会，有两层。
<br/>　　一层，是普通客人，位子相连，足有数百个。
<br/>　　二层，则是贵宾席，呈圆形，五个贵宾席分别环圈分布。
<br/>　　而中间，则是镂空。
<br/>　　贵宾席的人，可直接以居高临下的视角，一览下面的拍卖物。
<br/>　　“各位宾客。”
<br/>　　这时，一个拍卖师在一层出现，走上拍卖台，高声说着。
<br/>　　武者依靠体内真气，足可将声音清晰传遍整个拍卖会。
<br/>　　“首先，我代表拍卖会，感谢你们的到来。其次，鄙人今日能担任拍卖师，为诸位服务，乃鄙人的荣幸。”
<br/>　　“客气客气。”一层中，一众人连忙说道，“能参加如此盛大的拍卖会，我等高兴还来不及呢。”
<br/>　　这次联手举行的另外两个商业家族，本身在百武城内也是势力颇大，丝毫不亚于顾家。
<br/>　　而这拍卖师，乃是其中一个商业家族中的执事，故客人们都显得很是客气。
<br/>　　“好了，鄙人也不多说废话了。”拍卖师道，“诸位最关心的，必然是拍卖的物品。此次拍卖会，正式开始。”
<br/>　　“有请第一件拍卖品。”
<br/>　　拍卖师说罢，四个壮汉拿着一物出现，并将其放到展览台。
<br/>　　而后四人分别站到四边，警惕地防备着。
<br/>　　这四人，皆是洞玄三重武者；有两个属于盛宝商行，另外两个来自个本土的商业家族。
<br/>　　这四人，负责保护拍卖品的安全。
<br/>　　“此乃亚灵器，三土傀儡球。使用后，可释放三只土属性傀儡，擅长防御，实力为先天九重。”
<br/>　　拍卖师介绍完毕。
<br/>　　一层的宾客们已经双眼放光。
<br/>　　“三土傀儡球，起拍价，八万两，每次加价，不得低于五千两。”
<br/>　　拍卖师说完，宾客们，已经纷纷出价。
<br/>　　“八万五千两。”
<br/>　　“九万两。”
<br/>　　“....”
<br/>　　上方的贵宾席，则没有任何动作。
<br/>　　“易霄小兄弟不拍一下吗，这东西不错的，特别是你这种长期在外的猎妖师，必要时可救自己一命呢。”白家主说道。
<br/>　　萧逸摇摇头。
<br/>　　傀儡球的强弱，一般看炼药师的水准。
<br/>　　这颗三土傀儡球，论等级，和自己的三火傀儡球一样，但傀儡的实力，却差得远。
<br/>　　若是自己的三火傀儡球拿去拍卖，起拍价起码十二万两。
<br/>　　下方，竞拍激烈。
<br/>　　不多时，被一个大家族的家主以十一万两拍得。
<br/>　　本来这颗三土傀儡球应该不值这个价。
<br/>　　而这个家主，修为乃是洞玄一重，本身用不上。但家族中有一他疼爱的子弟，近期要出外历练。
<br/>　　这东西，对于低级武者来说，必要时可是能救命的。
<br/>　　那么，这位家主用高价买下来倒也正常。
<br/>　　“第二件物品。”拍卖师说罢。
<br/>　　四个壮汉，将一玉瓶放到展览台上。
<br/>　　“瓶内所放，乃是雷火丹，总共十粒。每粒起拍价一万两，每次加价，不得低于一千两。”
<br/>　　雷火丹，三品巅峰丹药；炼制到不难，只是材料极难寻得。
<br/>　　有雷火淬体之效，可让武者肉体更强；加之本身带有庞大药力，武者服之，可在一天内大幅度提升修炼速度。
<br/>　　效果对半步洞玄以下所有武者起效。
<br/>　　各个宾客纷纷出价。
<br/>　　上方贵宾席，仍旧没有人出价。
<br/>　　萧逸也对这东西没有兴趣。
<br/>　　“第三件物品....”
<br/>　　“第四件物品....”
<br/>　　“.....”
<br/>　　一件件物品拿出，都是些珍稀的东西，但萧逸毫无兴趣。
<br/>　　直到第八件物品，萧逸才眼前一亮。
<br/>　　拍卖师介绍道，“这是一本风属性的玄阶巅峰武技，乃是一位不愿意透漏姓名的武者授予我们拍卖。”
<br/>　　“起拍价十万两，每次加价，不得低于一万两。”
<br/>　　“诸位可要把握好机会了，玄阶巅峰武技，已经有许多年没有出现在拍卖会上了。若不是这位武者急需银两，绝不会拿出来卖。”
<br/>　　不得不说，这拍卖师的拍卖手段还是可以的，一下子激起了宾客们的购买欲望。
<br/>　　连上方贵宾席中，也有人在蠢蠢欲动。
<br/>　　“易霄小兄弟，一连八件物品你都没有出价，难道连这玄阶巅峰武技也看不上眼？”白家主问道。
<br/>　　萧逸苦笑道，“不是看不上眼，只是没钱买罢了。”
<br/>　　他的钱，要留着买八宝玲珑参，自然不会随便浪费。
<br/>　　再者，风属性的武技，对他用途不大，而且，他已经有电光火石了。
<br/>　　白家主也没有出价，白家本身就有玄阶巅峰武技，而且也不适合风属性武技。
<br/>　　下方的竞拍，极为激烈，也为这次拍卖引来了第一个高潮。
<br/>　　最终，这本玄阶巅峰武技，以二十五万两，被贵宾席上的一个大势力买下。
<br/>　　“第九件物品....”
<br/>　　“第十件....”
<br/>　　不得不说，这次的拍卖会，各种拍卖品都非凡物，来这一趟，就算不买东西，也增长了不少见识。
<br/>　　“第十八件拍卖品，便是本次拍卖会的压轴宝物之一。”
<br/>　　拍卖师高亢的声音，传遍全场。
<br/>　　随着四个壮汉将物品摆上展览台，萧逸顿时打起了精神。
<br/>　　那，正是八宝玲珑参。
<br/>(本章完)
