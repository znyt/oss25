　　第451章
<br/>　　银眸提住铁锤后，实力暴增。双手握着铁锤，在马上抡开，看她单薄的身板，似乎随时会被铁锤带着飞起来。
<br/>　　自然不用有这方面担心，二三百斤重的铁锤在她手中轻若无物，只听得砰砰声响，凡砸到敌人身上，无不头破血流，鲜血四溅。
<br/>　　短短时间，她杀得人比程大雷还要多。
<br/>　　程大雷骑着黑牛在敌人队伍中冲锋，来去几次，黑牛将敌人的队伍撞开一个缺口，不知多少人，死在黑牛的牛蹄之下。
<br/>　　老兵组有条不紊的调动，似乎并没有因同伴的死而影响士气。渐渐，程大雷和银眸被堵在一个土坡上，老兵组握着长矛发起冲锋。
<br/>　　程大雷只感觉一阵阵心寒，打到现在，老兵组付出了严重的伤亡，但他们仍没有一人后退。依旧是进攻进攻进攻。悍不畏死，向死冲锋，程大雷毫不怀疑，这可能是帝国最精锐的一支队伍。
<br/>　　可惜他们人数太少了，如果老兵组有一万人，他们将是整个天下最强之军，放在战场上，将无往不胜。
<br/>　　他们是真正的战士，无论是敌是友，程大雷都对他们保持尊重。是朋友，程大雷向他们献上敬意，是敌人，程大雷也同样向他们献上敬意，然后，杀光他们。
<br/>　　程大雷和银眸占据了土坡，脚下堆满了尸体。敌人举着长矛冲上来，被程大雷一斧切为两半，敌人握着大刀冲上来，被银眸一锤砸烂。
<br/>　　但他们并没有犹豫或后退，依旧在冲锋，踩着同伴的尸体冲锋。
<br/>　　然后，程大雷就真的把他们都杀光了。
<br/>　　战斗过后，夜已经深了，天空之上升起一轮月亮来，草原的夜，月光像水一样流动。
<br/>　　程大雷脚边堆满了尸体，每具尸体都保持着身体前倾的姿势，如今堆成一座小山大小。
<br/>　　老兵组一百零三人，全部阵亡，无有一人后退。
<br/>　　草原之上，除了程大雷三人外，只有几匹散落的战马，其他，再无活物。
<br/>　　程大雷从一匹战马身上抓过一个酒囊，打开瓶塞，咕咚咚往肚子里灌了一阵。
<br/>　　“大当家……”徐神机到他身边。
<br/>　　程大雷将酒囊递给他，徐神机摇摇头拒绝了。
<br/>　　“大当家，咱们该出发了。”
<br/>　　“是呐，该出发了。”
<br/>　　程大雷又喝了一口酒，将酒囊挂在牛角上，手中的斧头挥了挥。
<br/>　　“走喽！”
<br/>　　三人摸黑出发，借天上的月光探路，向杀虎口方向前进。
<br/>　　此时此刻，杀虎口内打得正是热闹。
<br/>　　重赏之下，必有勇夫。帝国军失去了耐心，在今夜发起总攻，目的就是将起义军的残兵败将一网打尽。针对每个起义军的脑袋，都定下了高额悬赏，基本上，只要有机会得到一个起义军的脑袋，一辈子便可吃喝不愁。如果能侥幸砍下一个头目的脑袋，那么子孙三代都不用担心衣食。
<br/>　　在这种状况下，自然个个奋勇，人人争先。
<br/>　　但令人难以置信的是，在如此状况下，起义军的阵型竟然没有被冲乱。在林少羽的号召下，他们抱团聚在一起，面对敌军的冲锋，悍不畏死，顶住了一波又一波的进攻。
<br/>　　原因无它，因为他们已经没了退路。身后是滔滔大河，鹅毛沉底，而面对起义军的逆贼，帝国军也不会开出降者免死之类的条件。
<br/>　　既然没有退路，那也只能拼死一战，在战死之前，多杀死几个敌人。
<br/>　　他们以河为界，聚拢在义字旗下，外边是层层包围的帝国军。
<br/>　　大旗迎风飘荡，士气高涨，连日来忍饥挨饿，此刻竟也不知疲惫。
<br/>　　林少羽一马当先，立在队伍最前方，右手提着铁枪。
<br/>　　“吾乃林家林少羽，想取我人头的，速速过来送死。”
<br/>　　杀威淋漓，以一人之勇硬撼千军万马，看着那飘荡的义字旗，帝国军竟犹豫不敢上前。
<br/>　　李行哉缓缓打马，来到两军阵前，目光落在林少羽身上。
<br/>　　“少羽，你已是穷途末路，何必再做这垂死挣扎？”
<br/>　　林少羽看着李行哉，发出一声冷笑，道：“你李家人要报仇，尽管放马过来，我的脑袋就在这里，看你们谁有资格来取。”
<br/>　　李行哉无言以对，任谁都能看出来，在千军万马的围攻之下，林少羽今天已经是在劫难逃，死路一条。然而，如果是单枪匹马挑战，帝国十万大军，就算在林少羽负伤的情况下，谁也不敢说能胜过他。
<br/>　　当初，林家被满门抄斩，林家上下百余口，就剩下林少羽一人。可就是一个林少羽，却踹翻了大半个帝国，让整个帝国陷入乱世中。
<br/>　　而相比之下，李家这一代就显得人才凋零，不管是李行哉还是李乐天，都没有那种力挽狂澜，撑乾坤之将倒的气魄。
<br/>　　李行哉深吸一口气，吐干净心中的愁绪，道：“少羽，路已到尽头，何必再执迷不悟，你可知道，程大雷已经来了，你不死，他必然会闯进来。求仁得仁，战死沙场或许就是你的追求，但你愿意看到程大雷也与你陪葬么？”
<br/>　　“大当家……”林少羽一怔：“他来了。”
<br/>　　提起程大雷的名字，林少羽呼吸一滞，眸子中的煞气被冲淡几分。
<br/>　　“你觉得，他愿意看到你如今这个样子么？”李行哉道：“你已走到绝路，干脆给自己一个痛快，引刀一块，不愧英雄一生。难道你希望自己的人头，被某个无名小卒砍去，借你的脑袋，全他们的名声。”
<br/>　　林少羽立在那里，低头思索，目光闪烁不定。他早已有必死之心，死他并不怕，却是没料到，程大雷竟然会千万里过来搭救自己。仔细想想，这确实也是程大雷的行事风格，倒也不是很难想到。
<br/>　　林冲在其身后道：“少羽，死则死已，临死之前，多杀几人，也落得痛快。让大当家知道，我们没坠他的威名。”
<br/>　　“是呐，若大当家知道我是自戳，定然会看不起我。”林少羽思索程大雷若落在此地该会如何做。以他的性格，是万万不会自杀的。
<br/>　　他忽然昂起头，手握长枪，冲着千军万马大喊。
<br/>　　“林少羽的人头就在这里，那个想要，尽管放马来取。”
<br/>(本章完)
