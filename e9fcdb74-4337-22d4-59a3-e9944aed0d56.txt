　　抱着柔软的娇躯，慕清霄思绪万千，来到青云位面六年，收获良多，修为提升了数个台阶，诞生了领域，还领悟了人剑合一的神通。
<br/>　　可谓是，今时不同往日，位面结束后，还有神秘的商店等着自己，一个位面给他带来的收获，真的难以估算。
<br/>　　而最重要的，便是怀中的佳人了，任何人任何事物都无可替代。
<br/>　　“嗯哼......”
<br/>　　睡梦中的陆雪琪娇哼一声，如同八爪鱼一样抱着慕清霄，小脑袋在他胸膛上拱了拱，并没有苏醒过来，彻夜驰骋，对体力的消耗还是蛮大的。
<br/>　　在慕清霄的怀中，她卸下了以往的冰冷，如同找到家的小女孩一样，睡的极其安详。
<br/>　　半个时辰后，睡眼朦胧，陆雪琪苏醒过来，揉了揉眼睛，忘记自己现在的状况，娇躯全部都暴露在空气中，引得慕清霄小腹再次冒起一股邪火。
<br/>　　“雪琪，早上有利于修炼，我们抓紧时间吧。”
<br/>　　话语刚落，在陆雪琪惊呼中，慕清霄翻身压了上去，又是一场翻云覆雨。
<br/>　　早上九点左右，两人同时走出客房，慕清霄整个人看上去都是神清气爽，陆雪琪脸上则残留着欢娱后的韵味。
<br/>　　来到酒楼，仆役已经准备好了早餐，而且告知，昨夜在此处落脚的客官有不少，但没有见到当初与他一起来此的张小凡等人。
<br/>　　慕清霄一边吃着早餐，道：“既然如此，雪琪，只好等我办完事情以后，在陪你回一趟青云，如何？”
<br/>　　陆雪琪已经是他的妻子，美眸中尽是温柔，道：“都听你的。”
<br/>　　“好。”
<br/>　　既然没有张小凡他们的消息，慕清霄也不打算继续停留，随后请店老板给捎个话，若是张小凡一行人来此，让他们无需担心。
<br/>　　处理琐事，慕清霄直接带着陆雪琪，化作剑光射向狐岐山。
<br/>　　......
<br/>　　......
<br/>　　第二天一早，一道剑光落在狐岐山范围内。
<br/>　　青云以北有一青山绵延不断，此山本是妖狐一族世代繁衍之地，其名为狐岐。
<br/>　　却不知多少年前鬼王宗众人来到此处大建土木，并将这狐岐山作为鬼王宗的总坛所在地，而鬼王宗宗主万人往亦是在此处不断研习尝试四灵血阵。
<br/>　　随着四灵血阵威力的日益加强，居住在狐岐山的妖狐们渐渐不安起来，随即发生的大地震更是让妖狐族人与鬼王宗弟子均是死伤惨重，而如今的狐岐山鬼王宗总坛是青龙与幽姬在之前的遗址上重新修建的。
<br/>　　“此地就是鬼王宗所在的狐岐山吗？”
<br/>　　望着周围青山绿水，鸟语花香的景色，陆雪琪真的很难相信，这里竟然是鬼王宗的总坛。
<br/>　　“按照地图上的标记，如果我们没有走偏，这里肯定就是狐岐山了，而且，你看......”
<br/>　　陆雪琪顺着慕清霄所指的方向看去，正好是狐岐山的山顶，隐隐能看到拔高的建筑物，虽不如青云门玉清殿宏伟，但也是一座不小的宫殿。
<br/>　　顺着山道上的阶梯，慕清霄牵着陆雪琪来到山腰，就在这时，一道低喝声响起：“你们是什么人，竟敢擅闯鬼王宗！”
<br/>　　只见半山腰入口处，两名持剑的黑衣青年从树杆后走了出来，看到陆雪琪的第一眼，眼中闪过一丝贪婪，可是仔细看了两人一眼，连忙收起贪婪的目光。
<br/>　　他们不是蠢货，知道有的人能惹，有的人不能惹，面前的俊男美女一看就知道不是普通人，他们只是鬼王宗的普通弟子，想要活的久，就得放聪明点。
<br/>　　慕清霄自然将他们每一个神色都看在眼中，轻声道：“劳烦通报朱雀圣使，便说故人来此。”
<br/>　　两名黑衣青年相视一眼，微微点头，其中一名青年严肃道：“你在此稍等片刻，我这就禀报圣使大人。”
<br/>　　片刻后，山顶一道紫色飞射而来，紧接着一名身着黑衣，面蒙轻纱的女子飘然而下，看到慕清霄容貌的瞬间，美目剧烈收缩。
<br/>　　“这位前辈怎么到鬼王宗来了。”幽姬心中有些困惑，上前一步，语气恭敬，道：“不知前辈来此，幽姬有失远迎。”
<br/>　　闻言，慕清霄嘴角抽了抽，道：“我的年龄没这么大，你不必叫我前辈，叫我慕清霄，或者慕公子都可以。”
<br/>　　“那好，不知道慕公子来鬼王宗有何贵干？”
<br/>　　幽姬盯着慕清霄，秀眉挑了挑，毕竟这名青年身上的气息实在太危险，哪怕鬼王都没有如此恐怖的气息。
<br/>　　慕清霄并不知道，自己散发出的气质，给幽姬带去成吨的压力，语气平淡，道：“我是来鬼王的，不知道可否带路？”
<br/>　　语气平淡，却毋庸置疑，幽姬知道，慕清霄今天是非见鬼王不可，叹道：“慕公子，请随我来吧。”
<br/>　　待慕清霄与陆雪琪的身影消失，两名黑衣青年都有种劫后余生之感，同时心中震惊无以复加，刚才的青年到底是谁，居然让朱雀圣使如此低声下气。
<br/>　　狐岐山巅，鬼王宗凉亭中，慕清霄与陆雪琪坐在木桌前，品着清香的茶水。
<br/>　　很快，幽姬就带着几道身影，匆匆忙忙的赶了过来。
<br/>　　为首之人身穿黑衣，中年文士，细眉方脸，相貌虽然算不上俊美，但颇为温雅，目光炯炯，文雅中带着不怒自威的气势。
<br/>　　中年文士左侧，是一位白面书生，青袍羽冠，温文尔雅，青袍袖口上绣有一条青龙。
<br/>　　右侧则是一名绿色倩影，此时一双明亮的眼睛正盯着慕清霄与陆雪琪。
<br/>　　毫无疑问，最前面的中年文士就是鬼王宗宗主万人往，而左侧的温文尔雅的白面书生正是四圣使之一的青龙圣使，至于幽姬和碧瑶早就见过数次。
<br/>　　“哈哈，慕公子能来鬼王宗，真是令寒舍蓬荜生辉啊！”
<br/>　　万人往领着三人，满脸迎笑的走了上来，根本连面都没见过，却像多年未年的老朋友。
<br/>　　在幽姬从空桑山回来后，就将途中的经过一一禀报给了鬼王，特别是慕清霄这号危险人物，更是强调了好几遍。
<br/>　　鬼王原本不信，可看到慕清霄的一瞬间，他立刻就信了，坐在他面前的不是一个人，而是一柄恐怖至极的剑！
