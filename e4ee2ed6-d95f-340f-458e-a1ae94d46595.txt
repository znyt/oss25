　　第224章
<br/>　　“怎么是和朕学的。”
<br/>　　“我可看见，前几日父王和刘公公穿着便服，鬼鬼祟祟出去了。”
<br/>　　“什么叫鬼鬼祟祟，朕那是工作……工作的事情，和你说了你也不懂。”
<br/>　　“父王溜出宫干嘛了？”
<br/>　　“那不叫溜，这天下都是朕的，朕想去哪儿还不是去哪儿。”明帝看着自己的女儿，终究是怒不起来，其他王子公主见到自己，总会恭恭敬敬，只有她在自己面前不太守规矩，可这种不守规矩，反而让明帝感受到一丝珍贵的亲情。
<br/>　　“父王，你在看什么呢？”
<br/>　　“是这次武举的文章，我从中看看，可否有能用的人才。”
<br/>　　“那么有么？”
<br/>　　“嘿，要么狗屁不通，要么都是些陈词滥调，勉强称得上优秀的人有，可若说出类拔萃者却一个也无。”提起这件事，明帝就气不打一处来。
<br/>　　他不想再聊这个，目光望着李婉儿，道：“想好了没，有没有要嫁的人？”
<br/>　　“喏，我才不想嫁人呢。”
<br/>　　“总要嫁人的，早晚你会遇到个愿意为他生儿育女的人。”
<br/>　　“可女儿现在还没遇到么，还能再等等。”
<br/>　　“不能再等了。”明帝道：“父王已经替你选了一个，想不想知道是谁？”
<br/>　　“呵，反正不是姓屈的，就是姓柳的，父王要是敢逼我，我就绝食。”
<br/>　　“不是姓屈的，也不是姓柳的。”明帝摇摇头，道：“父王还很真替你选了一个，此人能文能武，就是我说的那种出类拔萃，不拘一格的人才。”
<br/>　　“呵，还有这种人，长得怎样？”
<br/>　　“呃……应该不算丑吧。”明帝像是在回忆什么。
<br/>　　“不姓柳，也不姓屈，姓什么？”
<br/>　　明帝顿了顿：“姓牛怎么样？”
<br/>　　“姓牛……”李婉儿的嘴一下裂开：“父王，你说的该不会是他吧！”
<br/>　　……
<br/>　　程大雷正在家里养伤，事实上，他伤已经好得差不多了，可奈何长安城的达官贵人不知道抽什么疯，宴请的请柬每天都送过来。
<br/>　　程大雷只好装病在床，省的麻烦。
<br/>　　在他卧床的这段时间，也得到各种消息。
<br/>　　武举快要结束了，最后的武状元不是柳轻名，就是曲九江。最终，程大雷也没能改变什么。
<br/>　　而戎族的和谈使团已经进京，他们是有戎族内数个大部落组成的代表，目前已经开始和朝廷进行谈判桌上的交锋。
<br/>　　咚咚咚！
<br/>　　急促的脚步声，一个人影突然闯进门内，程大雷瞥了她一眼，无奈道：“李董，不是我不给你做事，实在是我行动不便，无法出门呐。”
<br/>　　李婉儿目光扫了一眼屋子，突然抄起一个花瓶向程大雷砸来。
<br/>　　“我擦！”
<br/>　　程大雷立刻从床上跳起来，伸手接住花瓶，心道：幸好我有灵犀一指。
<br/>　　李婉儿将一个板凳丢过来。
<br/>　　“灵犀一指。”
<br/>　　李婉儿将一方砚台丢过来。
<br/>　　“一指。”
<br/>　　李婉儿丢过来一个头盔。
<br/>　　“指。”
<br/>　　李婉儿端着一盆水泼过来。
<br/>　　“灵……”
<br/>　　程大雷被浇了一个落汤鸡。
<br/>　　“我特么……”程大雷揉揉脸，怒道：“你耍什么疯！”
<br/>　　“姓程的，你和我父王说什么了？”
<br/>　　“和你父王……我根本没见过他。”
<br/>　　“哼，还出类拔萃，不拘一格，你也撒泡尿照照，你也配。”
<br/>　　程大雷开始解裤子。
<br/>　　“你干嘛！”李婉儿一惊。
<br/>　　“撒泡尿照照。”
<br/>　　“你混蛋！”李婉儿尖叫，然后看到程大雷只是吓唬自己，于是怒道：“我说你对我的事怎么这么上心，原来是监守自盗，癞蛤蟆想吃天鹅肉。”
<br/>　　程大雷有些懵，究竟是为什么，你来我这里又喊又叫的，再者说，我癞蛤蟆想吃天鹅肉怎么了，这是我们蛤蟆寨的宗旨呐。
<br/>　　“发生什么事了？”程大雷问。
<br/>　　李婉儿有些说不出口，她冷哼一声：“十日后，将会选出武状元，那天文臣武将，包括帝国战神也会出现，你的好运气来了，你也要到场，父王要封你一个官。”
<br/>　　其实后面的话，程大雷已经听不太清，他只记住一句话：杨龙停也会到。
<br/>　　杨龙停现在的住所，天天有重兵防守，程大雷派人打探过，根本进不去。
<br/>　　还有十天么，这或许是自己最后的机会。
<br/>　　“程大雷，你莫要忘了，你只是一个山贼，永远都是。”
<br/>　　李婉儿气哼哼的放下一句话，拂袖离去。
<br/>　　程大雷到现在也没明白她发怒的原因，自己是个山贼怎么了，我从没有因为这个职业自卑过，你也不要有职业歧视么。
<br/>　　程大雷这才发现，原来帝国公主生气，和一个普通女人也差不多么。而他现在，也不关心李婉儿为何生气，他想的是十日后的事情。
<br/>　　武科举的结束，文官武将皆在，既封赏武状元，也会给自己一个官职，也会封赏杨龙停。
<br/>　　程大雷对做官没有兴趣，帝国朝廷的勾心斗角早已令他烦不胜烦，相比庙堂之高，程大雷更喜欢江湖之远。
<br/>　　可程大雷对杨龙停很有兴趣。
<br/>　　关键是怎么做。
<br/>　　这时候，阿苦咚咚的跑进来，见左右无人，才紧张道：“大哥，您那两位兄弟走了。”
<br/>　　“走了！”程大雷一惊，他知道阿苦说的是林冲和林少羽，可二人为何会走了。
<br/>　　“难道是被官府抓走了？”
<br/>　　“没有，为防止走漏消息，送吃的都是我一个人去，绝对不会有人发现他们的藏身处。”阿苦道：“今日我去的时候，发现他们已经离开，庙内并没有打斗的痕迹，临走前还在墙上留下四个字。”
<br/>　　“什么？”
<br/>　　“江湖再见。”
<br/>　　“呃……”
<br/>　　程大雷有些无语，他倒是不担心二人走后的安全问题，怕的是二人没走，反而寻机刺杀明帝。
<br/>　　以现在的状况，刺杀明帝几乎和送死差不多。
<br/>　　但愿二人真的走了吧。
<br/>　　“你撒出兄弟去，在城内寻找二人的消息，事情悄悄的做，一旦发现二人，立刻留住，绝不能让二人做傻事。”
<br/>　　“是。”
<br/>　　“还有一件事。”程大雷忽然想到什么：“不是有个戏班子要见我嘛，你们让他们来我这里。”
<br/>　　“大哥是想听戏解闷？”阿苦理解程大雷的心思，他在这里，的确有些无聊。
<br/>　　“听什么戏，我是教他们排一处戏。”程大雷心中大概已有了主意。
