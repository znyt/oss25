　　叶少阳不明白，道：“意味什么？”
<br/>　　“也有很多门派并不服从法术公会的引领，早就在串联抵抗，暗中积累着力量，这一次却让桃花山走在了前头，这些门派都蠢蠢欲动，法术公会不想让这种势头延续下去……”
<br/>　　“杀鸡儆猴？”
<br/>　　“对，杀鸡儆猴。法术公会一定要打击桃花山，将这个神秘人物按下去，也算给这些门派立威。”道渊说到这里，端起茶壶喝起来，这时候包子从叶少阳身后探出脑袋，被他看到。
<br/>　　道渊真人一眼就看到他是邪物，愣了一下道：“这是谁。”
<br/>　　“我新收的妖仆，你先别管这个，你接着说。话说，法术公会总不能把人杀了吧？”
<br/>　　“这当然不会，不过，可以比试，只要当着众多人的面，打败那个神秘人物，一切都解决了。”
<br/>　　毛小方道：“你不是说那人本事很强。”
<br/>　　“是很强。所以到时候别的门派都会看到，这样的强者都没法跟法术公会斗，别人自然也就死了这条心。”道渊真人无奈地耸了耸肩，“说起来，道法比实力更重要，但是现在没人论道了，你想告诉别人道理，首先得用拳头征服他。”
<br/>　　实力为王，这是亘古不变的道理，就算是法术界，也不能做到真的超脱。
<br/>　　毛小方当然也明白这个道理，沉吟道：“星月奴看中的弟子，自然也是很厉害的了，能被这个神秘人物轻松击败，说明这人真的挺可怕的。”
<br/>　　“所以法术公会才会召集弟子一起过去，也是很重视这件事。”道渊真人转头去看叶少阳，说道：“这件事，你怎么看？”
<br/>　　叶少阳手托下巴，说道：“我想去一次桃花山，见一见这个神秘人。我不是这个时代的人，轩辕氏跟法术界的纠葛，我不想掺和，不过你们是我好兄弟，我一定会帮你们的。”
<br/>　　跟这两个比自己大一百岁的人称兄道弟，叶少阳有一种很奇怪的感觉，尤其是道渊真人，可是自己老祖一辈的人……
<br/>　　三人商量之后，开始收拾行李，立刻也就走了。
<br/>　　叶少阳想着，梧桐等人突然被召唤回去，八成也是因为桃花山这件事，这样的话，等自己赶到的时候，说不定还有机会见上一面。
<br/>　　三人刚上路不久，李浩然跟碧清总算找到了镇上的这间客栈：他们早就到了古墓，只是四下里没有找到叶少阳，碧清觉得他们可能会在附近镇上落脚，这才来到镇上打听，耽误了不少时间，找到客栈里去的时候，叶少阳三人已经走了。
<br/>　　碧清找老板打听了他们去的方向，两人一路追过去，本来完全是可以追上的，但是巧的是，叶少阳三人之前走了一段，毛小方才想起来之前说好了下墓去弄些金子，跟道渊真人一说，道渊真人对盗墓没什么兴趣，不过对里面的字画很感兴趣，于是跟他们一起又折回头，前往古墓。
<br/>　　李浩然跟碧清跟他们没走一条路，所以就这么生生错过了，闹了个乌龙，碧清也是万万不会想到，在他们到处找叶少阳不见的时候，叶少阳其实就在自己的老巢里……
<br/>　　见到李浩然，碧清自然也不回古墓去了，反正炼尸缸也在她身上，两人商量一下，决定一起回桃花山，理由是碧清认为叶少阳既然跟那几个法师混在一起，大概也会一起去桃花山，李浩然对此更是有把握，相信以叶少阳那种性格，绝对不会错过这么热闹的事情。
<br/>　　而且李浩然还要回去带领桃花山抵抗法术公会的围攻，于是一起离去。
<br/>　　叶少阳三人先到镇上的铁匠铺，买了一把大锤，然后一起下古墓，来到那个藏宝的密室前，用铁锤砸开了上门的门锁，里面一切都跟通灵佛子花里幻境里的一样。
<br/>　　按照包子的说法，通灵佛子花一旦扎根，可以感知到方圆十里左右之内的一切环境，因此才以现实为蓝本，创造了那个幻境。
<br/>　　包子在前面带路，大伙先找到金子，但是这玩意不太好挟带，而且三人都不是很贪财（叶少阳除外，但是怕这里的东西带不到自己的时代，不想白费力气），因此取的不多，只是一人拿了几个金元宝，然后进入密室的隔间，找到那些古玩玉器，各自挑选了几件。
<br/>　　道渊真人只对书画感兴趣，把这些古墓里的书画一一看过，眼睛都湿润了。他是懂这个的，也只有真正懂，才知道它们的价值。
<br/>　　“我还是选一件吧，这些宝贝……不能让我一个人都霸占了，不然我自己也不安心。”道渊真人犹豫半天，最后选了一幅画带走。
<br/>　　离开古墓的时候，三人走的是一条跟之前不同的路，避过了那口井，根据包子说，那口井的下面的确能通到地下河，里面生存着一群蝾螈，叶少阳他们之前只遇到两只，运气真的算是挺好的了。
<br/>　　从古墓出去之后，三个人，加上美华和包子两个邪物，一起上路，前往杭州桃花山。这中间美华还回自己的洞府去了一下，按照叶少阳的要求，把自己那些手下都给遣散了——像他们这样啸聚在一起，不管是不是邪修，只要形成了一方势力，将来自然都会引起阴司和法术界的注意，不会有什么好下场。美华当然是听从叶少阳的安排，再说能跟在叶少阳身边，她也是心满意足，对自己的洞府并不留恋。
<br/>　　走在路上，大伙又聊了起来。
<br/>　　叶少阳将道渊真人走后发生的这些事情，跟他讲了一遍。道渊真人最后把目光落在包子身上，好奇地打量它，问了他很多关于通灵佛子花生长的问题，包子很懒得回答。
<br/>　　“对了，少阳子，我在桃花山，遇到你们茅山的掌教云秋生了。”道渊真人想起这件事，跟叶少阳说起来。
<br/>　　叶少阳怔了一下道：“云秋生在桃花山干什么？”
<br/>　　“我不是跟你说过吗，茅山是不服被法术公会管束的，我们龙虎山也不服，还有好几个门派，这次桃花山带头闹起来，几大门派也都过去了，想要跟桃花山讨论一番，顺便也见识一下那位神秘人物……”
<br/>
