　　“菩提本无树，明镜亦非台。
<br/>　　本来无一物，何处惹尘埃？”
<br/>　　准提圣人祭出自己的菩提法相，遮天蔽日般的菩提树掩盖了苍穹，无量光辉绽放虚空，任由那浩大的混洞如何侵犯，就是不能接近他分毫。
<br/>　　接着，准提不停，一道金色的手印从虚空之中伸展过来。那金色手印直接一击将万劫魔尊的混洞击碎，消散于天地之中，之后继续向着万劫魔尊抓去。
<br/>　　“咔擦……”
<br/>　　万劫魔尊周边的护体魔域响起了喀嚓声，紧接着他的魔域直接碎裂。
<br/>　　几乎是在那强大的金色手印要抓住万劫魔尊的同时，万劫魔尊手中的魔刀带起滔天的道韵规则，无尽魔气纵横，将手中的魔刀卷起无尽魔纹，看向金色手印。
<br/>　　同时，他的身体也急速后退，圣人的力量，他一时还不想与之硬碰，不然吃亏的终究会是自己。
<br/>　　“砰！”
<br/>　　万劫魔尊周身的道韵被手印抓得支离破碎，同时他手中的魔刀也劈在了那金色手印之上，发出一阵声响，那金色手印被魔纹震得破碎开来。
<br/>　　万劫魔尊没有逃避，周身道韵流转之间，那些轻微的伤势便早已消失的无影无踪。
<br/>　　还没等他反应过来，那磅礴浩瀚的释道规则道韵就碾压过来，那种规则似乎要将他的道韵和他所在的这片空间化为虚无。
<br/>　　万劫魔尊不退反进，跨前一步，周身魔道道韵反轰击了出去。
<br/>　　想象中的道韵规则爆炸并没有出现，万劫魔尊轰出去的道韵就好像是撞击在了一团棉花之上，消失的无影无踪。
<br/>　　准提圣人目光淡然，对于万劫魔尊尊能够破除自己的手段，好似早有所料，他不喜不悲，圣人的心境被放了出来。不知从何处取出七宝妙树，立刻出手，对着万劫魔尊打去。
<br/>　　“嗡！”
<br/>　　七宝妙树，释放出圣人的力量，七彩匹练，蕴含着圣人的威能，直冲万劫魔尊这里，冲击而至，璀璨了天地，照亮了万物。
<br/>　　万劫魔尊毫无畏惧，左眼中闪烁着猩红的光芒，右眼中闪烁着幽暗的光芒，手中魔刀更是高高举起，向着虚空中的七宝妙树劈去。
<br/>　　“屠天魔道！”
<br/>　　无上魔气纵横，无尽魔刀规则扑面而来，化为一柄天地霸刀，横贯虚空，充斥天地，魔气逼人，一刀斩下，斩天灭地，霸道绝伦，不愧为魔中至尊。
<br/>　　“轰隆隆……”
<br/>　　一声超级巨响响起，两者之间的这一招，居然相互僵持起来，距离冲击之处，周边虚空扭曲，许多裂缝乍现，准提圣人脸色怒红，目中带有一丝惊骇，但更多的却是凝重，以及狠辣。眼前此人还没有成圣，便有如此实力，若是让他成圣了，那还了得？必须在他成圣之前杀了他。
<br/>　　“轰！”
<br/>　　准提圣人继续出手，却是不准备给万劫魔尊喘息的机会。他全身震动，全力出手，有佛光普照，术法万千，圣人手段千变万化，更好似带动了洪荒三界的天地之力一般，种种神通，化为毁天灭地般的攻击，直冲万劫魔尊而来。
<br/>　　“哈哈！圣人又如何，本尊今日便要战一战你这圣人！”
<br/>　　万劫魔尊顿时豪情万丈，大笑声豪迈，张狂依旧，整个人蓦然仰天大吼，脚下魔域扩散，魔气冲天，黑芒之中，一道道劫魔出现，面露冷色，疯狂嗜血之光，咆哮连连，疯狂的攻击准提圣人的神通。
<br/>　　“岂有此理？好一个狂妄的万劫魔尊！不过，在圣人面前，一切都是蝼蚁，今日定要叫你灰飞烟灭，否则哪里显得出圣人的威严！”
<br/>　　陡听万劫魔尊的豪情狂言，准提圣人顿时大怒，觉得自己被人小看了，手中七宝妙树连连刷动，打向那无数的劫魔。
<br/>　　“哈哈，本尊今日便要逆战圣人，借此突破。吼！”战斗之中，万劫魔尊一声怒吼，手中魔刀连连挥动，斩向准提。
<br/>　　“你以为，如此就能够胜我，不够。”
<br/>　　准提忍住心中的怒意，头顶菩提法相，脚踏无边佛海，大战万劫魔尊，一切显得游刃有余。
<br/>　　“轰！”“轰！”“轰！”……
<br/>　　几乎刹那之间，两大顶尖强者，便出现在西方大地，苍穹之上大战，相互厮杀起来。准提圣人明显是想要活捉万劫魔尊，而万劫魔尊却是毫无畏惧，出手狠辣，招招歹毒，不愧为魔中至尊。
<br/>　　“不够，还是不够！还差一点。”
<br/>　　万劫魔尊心中怒吼，他觉得自己距离突破混元还差一点，不足够自己突破，他想要利用圣人的压迫，来使得自己完成这最后的蜕变，成就混元。
<br/>　　这一战，惊天动地，一个是西方教准提圣人，身为洪荒六圣之一，实力之强，自然不用多说，毕竟，圣人之下，皆为蝼蚁，这句话可不是说说而已。另一个，则为神秘的劫魔殿之主，万劫魔尊，来历神秘，有可能是远古时期的人物，道行高深，法力无边远超准圣圆满。
<br/>　　双方在西方大地之上，于苍穹之巅，激战而起，生死厮杀之中，你来我往，巨大的力量，彼此之间的对抗，有血光激射，蕴含滔天的毁灭之道，横扫八方，切割苍穹。圣人佛光，也是名副其实，璀璨耀世，一举一动，莫不带着巨大的力量，让万劫魔尊落入下风。
<br/>　　一场大战，万劫魔尊大多时候都被准提圣人压着打，偶尔有几招大招，也是被准提圣人化解了。万劫魔尊受伤颇重，好几次都险些被圣人灭杀，但他仗着自己的魔道神通，险而险之的避了开来。
<br/>　　“他居然，真的做到了？万劫魔尊，对决准提圣人，尽管落入了下风，但依旧很厉害了，好厉害啊！他究竟是什么人物？”这是大神通者们的惊叹。
