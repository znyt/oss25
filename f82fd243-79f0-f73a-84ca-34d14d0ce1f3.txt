　　四季城外，某密林中，萧逸快速飞奔着。
<br/>　　不多时，他的身影忽然变得摇晃，一个趔趄，差点没站稳，停了下来。
<br/>　　“噗。”萧逸猛地吐出一口腥血。
<br/>　　他的伤势，早就重得难以强撑下去。
<br/>　　在附近随意找了座小山，大手一挥，轰出个山洞。
<br/>　　萧逸走进去，做了些掩饰，掩盖住洞口，彻底瘫坐在地，大口喘着粗气。
<br/>　　“洞玄境的全力一拳，竟强悍如斯。”
<br/>　　若非有幽魂面具和真气外衣抵消了一部分力量，他的肉体又足够强悍，恐怕此刻已经死了。
<br/>　　洞玄境和半步洞玄的差距极大，足以轻松做到秒杀。
<br/>　　也就是萧逸这个变态，若是换了别人，早就死了。
<br/>　　饶是如此，他的伤势也极其严重。
<br/>　　可以说，至离开萧家以来，这是伤得最重的一次。
<br/>　　今日的行事，其实很是危险，但他又不得不去做。
<br/>　　自从昨日在白家听到慕容杀和慕容千军的对话后，他就在做着打算和计划。
<br/>　　北山慕容家和暗影楼，每三个月都会在四季城外，进行一次中等规模的交易。
<br/>　　这不是什么秘密，很多人都知道。
<br/>　　但，交易从来没有出过事，也从没人敢打他们两家商物的主意。
<br/>　　今日，萧逸特地埋伏在四季城外，杀了他们的人，烧了他们的商物，目的只有一个。
<br/>　　他既然无法对付慕容杀这个洞玄六重武者，便只能让慕容杀焦头烂额，没有机会去紫云城。
<br/>　　车队中的商物，都是些数量多，又普通的商物，烧了也无所谓。
<br/>　　真正贵重，又值钱的商物，其实都在慕容雄、慕容智，和一些护卫的乾坤袋中。如今都在萧逸手上。
<br/>　　这次，他公然抢夺了这批货物，慕容杀必定要不惜一切代价找到他，找回货物。
<br/>　　在此之前，慕容杀便没有机会去紫云城对付萧家了。
<br/>　　起码，在慕容杀看来，紫云城萧家只是个随手可灭的小家族，只是小事；而这批被抢的货物，却是大事。
<br/>　　虽说如今萧家有猎妖令这张‘保命牌’，但凡事都有意外。
<br/>　　如果能让萧家不用直面对上慕容家的任何一人，那便更好了。
<br/>　　现在，一来有猎妖令做底牌；二来慕容杀没有机会去紫云城萧家。萧家可保无虞。
<br/>　　明日便是裂天剑派开山收弟子之日，萧逸必须做好这一切，才能放心。
<br/>　　.....
<br/>　　萧逸想罢，将今日抢夺的乾坤袋拿了出来，总共八个。
<br/>　　两个分别属于慕容雄和慕容智，其余六个在其它护卫队身上。
<br/>　　将其一一打开后，萧逸眼都直了。
<br/>　　慕容雄的，乃是高级乾坤袋，空间极大，清一色都是天材地宝和丹药，而且尽皆品级不低。
<br/>　　天材地宝全是三品，足有数千。
<br/>　　萧逸认得这些材料，大多是用来炼制先天丹、隐气丹的。
<br/>　　丹药也是三品丹药，都是些在外战斗时需要的疗伤丹药，恢复真气的丹药。
<br/>　　慕容智的，也是高级乾坤袋，清一色妖兽内丹和妖兽精血。
<br/>　　且全都是三级。
<br/>　　三级妖兽内丹，五百颗，三级妖兽精血，五百瓶。
<br/>　　接着便是六个护卫手上的中级乾坤袋。
<br/>　　内头皆是兵器和毒药。
<br/>　　刀叉剑戟鞭棍锤匕，足有千把，且全都不是凡品。
<br/>　　随便一件都比得上自己价值千两的紫云精铁剑。
<br/>　　毒药，也都是剧毒，仅需一点，便足以将先天境以下武者轻松毒杀。
<br/>　　“啧啧。”萧逸轻笑一声，“真不愧是北山慕容家，这批商物的价格加起来，简直是天价。”
<br/>　　还有就是暗影楼，拥有的杀手数量，必然极多。
<br/>　　不过，这批商物里，真正对自己有用的，只有妖兽精血。
<br/>　　足足五百瓶，且全是三级。想自己这三个月来，一边剿杀山匪，一边杀妖兽，也不过杀了百多头三级妖兽罢了。
<br/>　　“本只想搅得慕容杀焦头烂额，没想到收获如此大。”萧逸心里大喜。
<br/>　　毫不犹豫地将五百瓶妖兽精血一一服下。
<br/>　　而后按照修罗战体中的功法运行。
<br/>　　数个时辰后，当所有妖兽精血消耗光，修罗战体第二层轰然突破，且达到小成。
<br/>　　原本被慕容智一拳造成的伤势，也顷刻间痊愈。
<br/>　　如今的肉体力量，足以媲美洞玄四重实力的妖兽。
<br/>　　也就是说，凭着肉体力量，自己已然有洞玄四重的战力。
<br/>　　不过，单单是第二层小成，却已经要消耗如此多的妖兽精血。
<br/>　　若是炼至大成，乃至日后的第三层、第四层，要消耗的妖兽精血，恐怕是个天文数字。
<br/>　　当然，修罗战体如此逆天的功法，付出再大都是值得的。
<br/>　　“没想到临去裂天剑派的最后一天，还能有如此收获。慕容家的杂碎，谢谢你们了。”萧逸冷笑一声。
<br/>　　.....
<br/>　　同一时间，百武城，盛宝商行内，慕容杀和慕容胤正在大发雷霆。
<br/>　　“该死的易霄，坏了本执事的大事，令得盛宝商行无法进驻四季城。”
<br/>　　“今日又抢我们的商物，真是欺人太甚。”
<br/>　　慕容胤愤怒地说着，双拳握得劈啪作响。
<br/>　　慕容杀也是脸色难看，道，“慕容胤执事，若非你昨日处处针对他，今日想必也不会出这档事。”
<br/>　　“与我何干？”慕容胤反驳道。
<br/>　　“哼。”慕容杀道，“天才都是有傲气的，你昨日这般针对他，他不报复我们才怪。”
<br/>　　“此事传出去，恐怕家族那边，首先怪罪的就是我们。”
<br/>　　“我千里迢迢，将这次的商物从家族运送过来百武城，如今却被抢了，你让我如何向家族交代？”
<br/>　　慕容杀认为，这根本就是易霄在报复慕容胤，自己不过是遭了横祸，被牵连了。
<br/>　　他怎么都想不到，最根本的原因，是他想去对付紫云城萧家。
<br/>　　慕容胤冷声道，“慕容杀执事，你觉得我们现在还互相埋怨有用吗？倒不如将那易霄捉回来，剥皮抽筋，夺回商物。”
<br/>　　“嗯。”慕容杀点点头，说道，“无论如何，易霄此举等同向我们北山慕容家挑衅，必须捉他回来，拿回商物。”
<br/>　　慕容胤道，“我现在立刻将百武城这边的力量全部派出去，一定能将易霄捉回来。”
<br/>　　慕容杀立刻说道，“那易霄能将慕容雄和慕容智杀掉，他的紫炎，太过恐怖了，一般的武者根本不是他的对手。”
<br/>　　“还是派人去将他的行踪寻到，我们两人亲自出手吧。”
<br/>　　“我们亲自出手？”慕容胤一愣，随即不屑道，“哼，区区一个毛头小子，也值得我们两人出手？”
<br/>　　慕容杀沉声道，“你现在还这般小看他，吃的亏还不够吗？”
<br/>　　“那批商物，价值之大，对于家族来说不算什么。可对于我们来说，一辈子都还不上。”
<br/>　　“若是夺不回来，不单止家族那边无法交代，暗影楼也不会放过我们。”
<br/>　　慕容胤闻言，脸色变得极其难看，但也只能愤愤地点了点头。
<br/>　　恰在此时，两个年轻人走来。
<br/>　　正是慕容轩和慕容千军。
<br/>　　“慕容杀执事，不知可忙完了？紫云城那边....”慕容千军恭谨地问着。
<br/>　　然而，他的话还未说完，就已经被慕容杀冷冷地打断了。
<br/>　　“本执事现在哪里还有时间管那种闲事。”慕容杀冷声道。
<br/>　　“区区一个紫云城萧家，等我处理好事务，挥手就灭掉了。若商物夺不回来，我自己都大难临头。”
<br/>　　如今慕容杀和慕容胤暴躁而又愤怒，慕容千军他们现在过来，简直成了出气筒。
<br/>　　“可是...”慕容千军想说些什么。
<br/>　　“没有可是。”慕容杀呵斥道，“现在立刻滚回郡都家族，安心修炼。待得这里的事情了结，我会再通知你们。”
<br/>　　慕容千军和慕容轩顿时脸色变黑，他们急着报仇，可是自己只是小人物罢了，在北山慕容家中根本没有权势。
<br/>　　那些执事，想帮他们便帮，不想帮，随手就打发他们回去了。
<br/>　　“是。”两人恭谨地应答一声，虽然内心不甘，却不敢在这个时候忤逆慕容杀。
<br/>　　......
<br/>　　翌日，四季城内，茶楼、酒馆、坊市，四处都在谈论着今日裂天剑派开山收弟子之事。
<br/>　　“裂天剑派今日开山，从北山郡四面八方赶来的年轻武者，都快要将四季城挤满了。”
<br/>　　“呵呵，裂天剑派乃是武道圣地，自然个个都想加入。”
<br/>　　“只不过，听说考核也非常困难，一百个人去参加，也未必有一个能通过呢。”
<br/>　　“.....”
<br/>　　茶馆内，个个都在谈论着。
<br/>　　作为四季城内的居民，他们对于能毗邻裂天剑派还是感到非常自豪的。
<br/>　　不过，忽然，有一些人却聊起了其它事情。
<br/>　　“喂，你们听说了吗？昨日，盛宝商行和暗影楼的交易商物，被人抢了。”
<br/>　　“什么？不可能吧。这些年来，就没见过有人敢打这交易的主意。可没人敢得罪这两大势力阿。”
<br/>　　“什么不可能，我跟你说，抢夺商物的，就是易霄。”
<br/>　　“易霄？那个紫炎易霄？不会吧，他的胆子这么大？”
<br/>　　一桌人边品茶，边聊着八卦。
<br/>　　他们却没发现，在他们旁边茶桌，正有一个戴着蓑笠的年轻人细细听着。
<br/>　　“你们忘记了吗？那慕容胤执事在白家处处针对易霄大人。”
<br/>　　“易霄大人何许人也，武道天才，轻易败无忧公子；四品炼药师，连败慕容胤、方鹤、叶铭。”
<br/>　　“这样的天才，何等的傲气，不去报复慕容胤才怪。”
<br/>　　那人滔滔不绝地说着，听得同桌旁人满脸过瘾。
<br/>　　那人继续道，“现在，整个百武城盛宝商行都在通缉易霄。还有暗影楼的无忧公子，也在到处寻他。”
<br/>　　旁人点点头，道，“也是，那批商物暗影楼也有份，肯定也会找易霄麻烦的。”
<br/>　　那人摇摇头，道，“才不是这个原因，你们忘了吗？无忧公子是个战斗狂，他之前被易霄大人打败了，自然要把场子找回来。”
<br/>　　....
<br/>　　一桌人仍在八卦地谈论着，却没有发现，旁边一桌的年轻人，已经起身离去。
<br/>　　无论在什么地方，像茶楼和酒馆这种龙蛇混杂之地，总是能听到不少八卦和消息。
<br/>　　那年轻人，正是萧逸。
<br/>　　“呵呵，以为我在报复慕容胤吗？也罢，反正都是要找我麻烦的。”萧逸暗暗想着。
<br/>　　“只是，那无忧公子是战斗狂是怎么回事？那家伙到处在寻我？”
<br/>　　萧逸皱起了眉头。
<br/>　　半晌，他无奈地摇摇头，暗道，“算了，都是一群混蛋，先去裂天剑派再说。”
<br/>(本章完)
