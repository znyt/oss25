A大吴校是一个性情中人，是知识分子里少有的那几个喜欢夏邦的，上一次迷.奸案，那个周副校顾及学校声誉，处理的很不及时，夏邦带着人大闹T大，吴校就觉得大快人心。
<br/>
<br/>        就是这几年吴校有点郁郁不得志，A大的学习风气越来越差，已经在帝都医科大学里垫底了，他和周副校当年可是同窗，现在每一次开会，周副校看到他都像是一只得意的公鸡，雄赳赳气扬扬，将他压得死死的，所以吴校才四十多岁的年纪，头顶的头发都快秃没了。
<br/>
<br/>        夏邦开心的大笑，“吴校，算你有眼光，你将我家绾绾收下，我保证以后你会惊喜的。”
<br/>
<br/>        “夏大当家的，我很敬重你，自然也很喜欢令千金，但是校有校规，令千金进A大，我还是要了解一下令千金的学历成绩。”
<br/>
<br/>        三个月前陆司爵带着陆寒霆离开的时候，抹去了夏夕绾在海城的一切记忆，资本的力量太强大了。
<br/>
<br/>        而当年13岁的夏夕绾就是被保送了一等医科学府T大，也就是刚才将三个人拒之门外的那个T大，刷新了整个T大的历史，创下了医科界的最新记录。
<br/>
<br/>        不过，15岁成为双博士后的夏夕绾被选中，参与了一项秘密研究，因此她所有的学历都被T大正校封存成了秘密档案，不能透露出去。
<br/>
<br/>        当年夏夕绾在T大的时候那位周副校长还没有被提拔，还是一位教务主任，她印象不是太深。
<br/>
<br/>        天才少年班和正常入学的T大学子们本来就有壁，又过了五年之久，周副校或许对她有印象，但是已经想不起来了。
<br/>
<br/>        科学院每年都会在这些医科大学里招生，她不愿意夏家人夹在她和T大中间受气为难，既然去不了T大，来A大也是不错的选择。
<br/>
<br/>        夏夕绾这一次来帝都城只希望自己可以顺利拿到妈咪留下来的第二个宝箱。
<br/>
<br/>        “吴校，我以往的学历成绩因为某些原因不能告诉你，不过我这里有海城枢密研究院李文清院长的亲笔介绍信一封。”夏夕绾将介绍信拿了出来。
<br/>
<br/>        吴校跟李文清院长都是一个圈子里的，自然是熟人，他打开介绍信看了一眼，惊奇道，“李院长我还是很熟悉的，他眼光甚高，竟然会为了你写亲笔介绍信。”
<br/>
<br/>        夏夕绾澄亮的翦瞳看着吴校，“吴校，你可以先收下我，最近不是快联考了吗，你可以看一下我的考试成绩再决定要不要将我转正，总之我有信心，吴校如果愿意入股我，绝对不亏。”
<br/>
<br/>        吴校看着女孩眸子里那一闪一闪的碎光，璀璨夺目，从容坦然的让人心安与信服，吴校这一颗郁郁不得志的心突然扑通扑通的跳动起来了，他有一种预感，这个女孩会改写A大垫底的历史！
<br/>
<br/>        他捡到宝了！
<br/>
<br/>        “好，我收下你！”
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾选择了中医专业，上课铃声响了，中医2班是吴校的课，但是班上的同学们都在嬉笑打闹，毫无严肃的学习氛围。
