　　那近乎无敌的荒古大道，在如今众目睽睽之下，被楚风眠一剑破开。
<br/>　　就连荒古大道背后的时空长河，都在楚风眠的剑下，彻底毁灭。
<br/>　　“我的天，这到底是什么实力！此人到底还隐藏这多恐怖的力量！”
<br/>　　“我们北域中，什么时候诞生了如此恐怖的天才，他的实力已经超越了北邙学院的种子弟子了，莫非他就是下一个夜天君？”
<br/>　　“三尊荒古门弟子，吞服荒古神丹缔造而出的荒古大道，就这样被人打破了，怎么可能！”
<br/>　　一道道惊异无比的声音响起，无数目光，看向楚风眠，看着楚风眠，直接一步之间，踏了出去。
<br/>　　楚风眠大手一挥，直接向着那荒古门抓了过去。
<br/>　　“荒古大道，居然破碎了！不可能！我们凝聚出的荒古大道，让生死境高阶的强者来都不一定可以打破，怎么会被这小子一剑斩破！”
<br/>　　那三尊荒古门弟子，都被连连败退，眼神中还都是震惊无比。
<br/>　　如今到现在，似乎都没有反应过来。
<br/>　　荒古大道已经凝聚完成，他们三人应该早就立于不败之地了才对。
<br/>　　可就在他们的眼前，荒古大道，被楚风眠一剑斩破。
<br/>　　荒古大道，可是荒古门的立宗之本，九域之中，无人不知这荒古大道的强大。
<br/>　　如今被一个神力境四重的武者直接斩断，他们的心中，谁又能想到。
<br/>　　“荒古门从万年前，就自大无比，没想到万年之后，非但是自大的毛病没改，这弟子都一届不如一届。”
<br/>　　楚风眠看向那三尊荒古门弟子，冷喝一声道。
<br/>　　“你们三人都算有着一些奇遇，杀了你们，也足以弥补我的损失了！给我过来！”
<br/>　　这三尊荒古门弟子，以近乎全部力量，凝聚而出的荒古大道已经破碎，他们那里还有任何的还手之力。
<br/>　　在楚风眠的面前，被一下子给抓了过来，握在手心中，一个个脸色都是憋得通红。
<br/>　　“你不能杀我们，我们是荒古门的弟子……”
<br/>　　“废话真多！我既然杀了两个，就不差你们三个！”
<br/>　　楚风眠冷喝一声，他手中猛然用力，这三尊荒古门弟子，都要这力量之下，被直接挤爆压扁了一样。
<br/>　　“住手，这三人不能杀！”
<br/>　　一道声音，突然响起，只见一道身影，从旁边猛然窜了出来。
<br/>　　这道身影之长，弥漫这漫天恨意，正是落长恨。
<br/>　　落长恨一手猛然对着楚风眠抓了过来，似乎是想要从楚风眠的手中，救走这荒古门三人。
<br/>　　“哼！”
<br/>　　楚风眠瞥了一眼落长恨一样，手中的力量陡然加大，这三尊荒古门弟子，一声惨叫之后，纷纷破碎在楚风眠的手中。
<br/>　　他们身上的一切奇遇，血气，空戒，都被楚风眠大手一挥，尽数收在手中。
<br/>　　“该死！”
<br/>　　落长恨看着楚风眠，脸色气的通红，他都开口了，楚风眠竟然都丝毫不理会他，当场杀了这荒古门三人。
<br/>　　落长恨指着楚风眠，大声的喝道。
<br/>　　“小子，你知不知道你做了大错事！招惹了大麻烦了！他们三人可是荒古门的人，你现在杀了他们，不止是给你带来麻烦，更是会给学院带来麻烦的，你可知道？”
<br/>　　楚风眠瞥了一眼落长恨，平静的开口道。
<br/>　　“麻烦？有麻烦，楚某自己自会解决，不用你来担心。”
<br/>　　“自己解决？”
<br/>　　落长恨的脸上露出几分不悦之色道。
<br/>　　“这些可都是荒古门的弟子，代表这七大宗门，你白白给学院招惹了如此强敌，已经是犯了大罪孽！如今你居然还不知悔改。“
<br/>　　落长恨话中的语气，简直就像是一个上司，在训斥下属一般，令楚风眠的眉头都微微一皱，声音也冰冷了许多。
<br/>　　“落长恨，楚某如何做事，还不用你来管！”
<br/>　　楚风眠冷哼一声道。
<br/>　　他楚风眠要杀谁，还没人管得着，这落长恨抬出北邙学院来想要压楚风眠，简直可笑。
<br/>　　别说是没有人能够管得住楚风眠，今日这些荒古门弟子，本身就是该死之人。
<br/>　　楚风眠杀了他们，问心无愧。
<br/>　　“不用我管？好！一个区区北邙学院的新人，如今就这样目中无人，你等着回到学院中去，我一定要向着刑法堂状告你的罪孽！看看你是不是真的无法无天！”
<br/>　　落长恨冷哼一声，直接便是转身，拂袖而去。
<br/>　　“想走？”
<br/>　　楚风眠冷哼一声，猛然间大手一挥，直接将落长恨的周围，直接拦住。
<br/>　　“谁允许你走了？”
<br/>　　这落长恨指责楚风眠一顿，就想要离开，还想要将今日的事情捅上刑法堂去，楚风眠怎么会让他这样离开。
<br/>　　刑法堂，可是北邙学院中最为重要庞大的一个堂口。
<br/>　　进了刑法堂的人，就算是没罪，也要被拔下一层皮来。
<br/>　　这落长恨之前就利用长恨之道，来偷袭楚风眠，楚风眠可都记得了，现在还想要随便离开，哪有那种好事。
<br/>　　落长恨看着楚风眠封锁他的面前，脸色微微一变，怒声喝道。
<br/>　　“楚风眠，你还想做什么？我可是北邙学院的种子弟子！你敢来得罪我？”
<br/>　　“我想做什么？落长恨，你以为你出手偷袭我，我会不知道，现在想逃？”
<br/>　　楚风眠看着落长恨，大笑起来道。
<br/>　　“落长恨，你大概不知道我的手段，敢来出手偷袭我的人，都要死！你也一样！”
<br/>　　“北邙学院的种子弟子？我连七大宗门的人都敢杀，你这点身份算什么，给我死吧！”
<br/>　　楚风眠凝聚灵力，化为了战龙之爪，顿时从天而降，一下子轰击下去。
<br/>　　“该死！小子你真的是无法无天了，还敢残害同门！你简直就是一尊魔头！”
<br/>　　落长恨看到楚风眠战龙之爪落下，脸色也是骤变，一身灵力毫不收敛的全然爆发出来。
<br/>　　楚风眠的实力，他之前可是见识过的，要是在楚风眠的面前留手，只怕就再也没有展现下来的可能了。
<br/>　　“长恨刀！”
<br/>　　无尽恨意，在落长恨的手心中凝聚出了长恨刀，顿时猛然一刀，向着空中斩了过去。
<br/>　　妄想以这长恨刀的力量，抗衡战龙之爪。
<br/>　　“可笑！”
<br/>　　楚风眠只是淡然吐出两个字，战龙之爪无情落下，顿时这长恨刀的力量，都被战龙之爪的面前，轻易碾碎。
