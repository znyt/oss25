　　黑耀十方石，天界果，青雀血。
<br/>　　这三件宝物，全部都被楚风眠收走。
<br/>　　“你看，前方！”
<br/>　　寒月漓在一旁，突然出声道。
<br/>　　就在前方的云羽天宫之上，巨大无比的宫门，如今全部打开，数百座宫殿，一眼望去，全部打开。
<br/>　　在其中的每一座宫殿之中，都是堆满了无数宝物，王品灵石，无数灵药，在其中，堆积成山。
<br/>　　在其中的一些宫殿之中，更是有着一些闪耀的拳头大小的岩石。
<br/>　　羽化晶！
<br/>　　这一块块拳头大小的岩石，正是羽化晶，一眼望过去，分布在每一处宫殿之中。
<br/>　　这羽化晶，一共十万，谁可以全部得到，谁便是就可以得到这云羽天宫的传承。
<br/>　　“走！”
<br/>　　楚风眠当机立断之际，猛然一动，御使战龙宝车，直接向着云羽天宫上冲了过去。
<br/>　　这黑耀十方石，天界果，青雀血这三件宝物，对楚风眠来说虽然是有着大用。
<br/>　　若是现在炼化的话，也足以是让楚风眠的提升，提升无数。
<br/>　　不过楚风眠也没有在半路停留，他御使战龙宝车，一直是冲入到了云羽天宫之中。
<br/>　　炼化这三件宝物，至少也要浪费不少的时间，现在这云羽天宫的宝藏传承才是最为重要。
<br/>　　楚风眠的战龙宝车，一路驰骋。
<br/>　　踏出彩虹阶梯的一刻，他的战龙宝车，便是突然消散。
<br/>　　并非是消失，而是失去灵力，直接消失，回到了楚风眠的空戒之中去。
<br/>　　在这云羽天宫之中，似乎是有着一种禁制，来压制着这样的坐骑灵器。
<br/>　　毕竟这里可是云羽天宫，曾经云羽天宫的核心，无比威严，自然不会允许让无数武者，在这其中随意穿行。
<br/>　　楚风眠扫视了周围一拳，便是身躯一动，猛然向着其中一座宫殿急速跑去，催动身形，速度快到极致。
<br/>　　这一座宫殿之中的羽化晶数量最多，正是现在楚风眠所说需要收取的。
<br/>　　楚风眠身形变幻连连，很快便是来到了这一座宫殿之中，从这宫殿中，其中的任何一件宝物上，似乎都刻画有着一种极为特殊的禁制。
<br/>　　就跟在那颜古龙帝洞府之中所见到的一样，似乎是这云羽天宫中的强者所设。
<br/>　　不过七万年的时间，上面禁制的力量已经是弱到了极点，生死境的武者，都足以打破。
<br/>　　楚风眠大手一挥，猛然抓向一枚羽化晶，随着这一抓，这一枚羽化晶上的禁制，瞬间全部破碎。
<br/>　　这一枚羽化晶，便是落入到了楚风眠的手中。
<br/>　　这羽化晶，实际上价值却不算很高，在炼器上稍微有些用处，但是以现在楚风眠的财富来说，并没有太多用处。
<br/>　　不过现在显然，这羽化晶的作用已经是不止于此，现在这些羽化晶，已经是关系这云羽天宫的传承，云羽天宫的宝藏。
<br/>　　谁要是能够得到这云羽天宫的传承，这云羽天宫的一切财富，也都属于他。
<br/>　　这云羽天宫的财富，只怕是比起七大宗门想加都毫不逊色，谁能够得到，真的就是巨富无比了。
<br/>　　这样一枚羽化晶，现在比起在场的无数宝藏，更加重要。
<br/>　　“统统都给我收取！”
<br/>　　楚风眠看着宫殿之中的无数禁制，突然怒喝一声，在他的背后，远古战龙突然显现，一声龙吟之下，楚风眠猛然一拳打出。
<br/>　　这一道拳风，横扫整个宫殿，其中的无数禁制，全部破碎。
<br/>　　随着楚风眠大手一挥，打开空戒，这宫殿之中的所有宝物，如今全部都被楚风眠收入到了空戒之中。
<br/>　　“在场诸位，谁若是先收集到五万枚羽化晶，便是可以得到我云羽天宫的传承。”
<br/>　　就在楚风眠收取掉这一座宫殿之中所有的羽化晶的时候，那器灵老者的声音，却是再度开口了。
<br/>　　先收集到五万枚！
<br/>　　楚风眠目光凝重，他收取这宫殿之中的所有羽化晶，一共才两千枚左右。
<br/>　　这云羽天宫，一共是有着足足一百座宫殿，楚风眠至少要收取其中的一半，才可以得到五万枚。
<br/>　　以楚风眠跟寒月漓两个人的速度，绝对是无法跟七大宗门那些弟子一同收集相比。
<br/>　　更何况，在场的许多武者所在的势力，都是七大宗门的下属，他们所收集的羽化晶，也很有可能是去给七大宗门。
<br/>　　在这种拼人力的时候，楚风眠便是陷入到了绝对的劣势之中。
<br/>　　楚风眠已经是感觉得到，那些七大宗门的弟子，已经是进入到了云羽天宫之中来，也在不断的收集这云羽天宫。
<br/>　　几乎每一座宫殿中，都有着武者的存在。
<br/>　　“跟这些人相比速度，我是绝对的劣势。”
<br/>　　楚风眠心中也清楚，若是这样相比收集的速度，楚风眠绝对比不上七大宗门，数十人一起相比。
<br/>　　就算是楚风眠这样一下子收取一座宫殿，足以直接打破所有的禁制，但是将半数宫殿都给收集一边，也要到天荒地老。
<br/>　　而且那个时候，大半宫殿中的羽化晶，早已经就被收取完了。
<br/>　　“看来，只能是让他们来帮我收集了。”
<br/>　　楚风眠的嘴角露出几分冷笑。
<br/>　　既然是他自己收集不完，那就让其他人来收集好了，楚风眠只需要将其其他武者手中的羽化晶，全部抢走便是。
<br/>　　楚风眠心中一下子便是做出决定，与其是他自己来慢慢收集，还不如直接去抢来的快。
<br/>　　“走，我们去分头猎杀其他武者！”
<br/>　　楚风眠身形一动，猛然之间化为了一道遁光，猛然向着一座宫殿走去。
<br/>　　在他身旁的寒月漓也是一样，化为了一道遁光，向着另外一个方向走去。
<br/>　　以现在寒月漓的实力，遇上大部分的武者，都足以击败。
<br/>　　就算是遇到荒圣君等人，楚风眠也可以及时赶到。
<br/>　　“这云羽天宫，真的是无穷宝藏，这一方宫殿中的宝藏，比起我一生所见还要多。”
<br/>　　在一处宫殿中，两名荒古门弟子，正在收集宝藏。
<br/>　　他们两人纷纷动用灵力，在宫殿中的无数禁制，全部打破，其中一件件宝物跟那些羽化晶，都是落入到了他们的手中去。
<br/>　　“只可惜，这羽化晶我们必须要交给师兄，不然也许我们也有可能得到这云羽天宫的传承。”
<br/>　　一名荒古门弟子，看着面前的羽化晶，不由的感叹了一声道。
