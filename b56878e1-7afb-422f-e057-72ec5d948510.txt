　　第125章
<br/>　　蛤蟆寨，聚义厅。
<br/>　　青牛山五十多家山寨的头目，有一个算一个，今天全部到了。蛤蟆寨的聚义厅，从没像今天这样热闹。这些人程大雷有些认的，有些只是恍过面，有些脸熟而已。
<br/>　　“程当家义薄云天，我们都看在眼里，以后我这条命就是程当家的。”
<br/>　　“程当家让俺往西，我就往西，程当家让俺往东，我就往东。谁不服程当家，就是不服我。”
<br/>　　“程当家你就莫要推辞了，大家以后都跟你混。”
<br/>　　乱糟糟的人群，说的都是同一个意思。青牛山所有山寨全部愿意并入蛤蟆寨，唯程大雷马首是瞻。
<br/>　　可是，如此乱哄哄的人群中，今日理所应当的主角程大雷，却保持着诡异的安静，他坐在太师椅上，如庙里的泥胎塑像。
<br/>　　程大雷其实很理解很大家的心情，在这个时代，大家都是为一口饭活着，而跟着程大雷显然是能够吃饱饭的。
<br/>　　可是，程大雷还有些犹豫。
<br/>　　青牛山所有山贼加起来，约有五千人左右，其中一多半是老弱妇孺，能打的不过两千余人。因为这次旱灾，走了一部分人，剩下的壮劳力就更少了。
<br/>　　养活这五千人，可不是一件容易的事情，莫看程大雷这次弄回来四十万斤的稻米，但真让这些人吃，其实也吃不了几个月。
<br/>　　比如五千人每人每天给你一块钱，那么你一年的收入就是一个天文数字。但同样，如果每天都要给这五千人每人提供一斤口粮，这同样是一笔天文数字。
<br/>　　况且这五千人，也不是每个都是良善之辈，用乌合之众来形容他们，都算是褒奖。现在都蛤蟆寨不过百十人，这些人程大雷有把握完全掌控，可当山寨的人口扩展到五千人，程大雷就没有这个信心。
<br/>　　当然，程大雷想到的都是坏处，人多自然也有人多的好处。
<br/>　　这就像一个创业公司，靠一个项目掘到第一桶金，那么是小富则安，守好自己的一亩三分地，还是开拓项目，有可能亏本，也有可能获的更多利润。
<br/>　　该如何选呢……
<br/>　　周围的声音还在吵嚷，程大雷一时也没有去理睬他们。这次运粮回来，自己的任务算是完成了，刚才被打断，奖励是什么自己还没来的及看呢。
<br/>　　嘟，正义的少年喔，恭喜你完成任务，获的奖励……三级山寨建设图纸。
<br/>　　这是程大雷获的的第二张图纸，上次只是防御图纸，而这次是完整的建设图纸。
<br/>　　程大雷脑海中出现图纸的模样，升级到三级所需的建筑他还没来的及看，只注意到一条：
<br/>　　所需人口：一千。
<br/>　　聚义厅依旧乱哄哄一片，程大雷坐在最高处，第一次认真的打量他们。
<br/>　　自己是不是该往前进一步了？
<br/>　　“好了！”
<br/>　　程大雷忽然起身，说也奇怪，本来噪杂的人群，在程大雷吐出这两个字后，瞬间变的鸦雀无声，所有人的目光都放在程大雷身上。
<br/>　　程大雷长身而起，高声到：“大家的话我都听明白了，人在江湖飘，什么最重要……”
<br/>　　程大雷目光环视过人群，在众人还没来及回过味来时，斩钉截铁道：“义气！”
<br/>　　“而我这个人，最他妈在乎的就是义气。今天大家既然看的起我程某人，那么日后咱们就是一家兄弟，有我一口吃的，就绝对饿不着大家。”
<br/>　　“好！”
<br/>　　欢呼声一片，众人都陷入一种莫名的亢奋中。程大雷揉揉发僵的脸，默默坐了下来。
<br/>　　……
<br/>　　欢宴终会结束，不管它开始时是多么热闹。
<br/>　　事情过去三四天后，程大雷窝在屋里检查那份三级山寨的图纸。
<br/>　　山寨由初级升到二级后，程大雷也同样生了一级，如果没有意外的话，等山寨升到三级，自己的战斗力也能升一级。
<br/>　　经过呼尔勒的事情，程大雷已经意识到，自己目前的实力是不太够用的。拎着斧头吓唬吓唬人还可以，但真如果刀对枪，硬碰硬几乎和送死差不多。
<br/>　　程大雷已经通过秦蛮、赵子龙、小白狼等功夫不错的人了解过，战场厮杀之道，一是气力，二就是招式。二这两样每一样都得夜以继日得勤学苦练。
<br/>　　真要练个十几二十年，程大雷一没有这份恒心，二也没有这个时间。
<br/>　　至于这世界有没有内力轻功，以及飞剑问仙的事情……反正，他们都听过不少传说，见是谁都没见过。
<br/>　　传说中，程大雷还会五雷法呢。
<br/>　　于是，通过升级山寨实力而提升自己的等级就成了唯一的办法。
<br/>　　“大当家，你在里面么？”
<br/>　　高飞虎从屋外挑帘进来，目前各山寨正在融合，所有山寨并入蛤蟆寨。当然，当家的就只有程大雷一个。五十几家山寨合为一家，本来人心就不齐，再搞出各二当家、三当家、四当家之类的，岂非等着山寨里闹出一个个小山头。
<br/>　　程大雷回过神来：“什么事？”
<br/>　　“大当家，最近我一直在想，咱们这么多山寨合在一起，终究是件大事，是不是该请一下江湖上的朋友，让他们给我们做个见证。”
<br/>　　“看不出来，你还是个挺有仪式感的人。瞎搞什么，咱们关起门来过我们的小日子就是。”程大雷否决了他这个提议，又问道：“刚才你说江湖上的朋友，咱们青牛山外，都有什么绿林豪杰？”
<br/>　　乱世出恶匪，这天下的土匪不止程大雷一家，只不过程大雷对周围对情形不太清楚。
<br/>　　“本来，距离青牛山最近的，就是落叶城的杜老大，后来他不是没了么……”
<br/>　　杜老大还是程大雷亲自动的手，当然对方如今是生是死谁也不知道。
<br/>　　“其他呢？”
<br/>　　“其他就多了去了，做独行客的周狗周鼠两兄弟，赛燕子单飞，做水上生意的马家父子……要说最大的，就是伏虎山的黑旋风了。”
<br/>　　“黑旋风……李逵么？”
<br/>　　“姓孙。”高飞虎一头雾水。
<br/>　　“好啦，好啦，大家各扫门前雪，也休管他人瓦上霜。”程大雷道：“他们就不用请了。你这次来的正好，我还有件事要找你商量。”
<br/>　　“什么事？”
<br/>　　“我准备择地重新建一座山寨。”
<br/>　　
<br/>(本章完)
