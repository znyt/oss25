        “明司寒你……”
<br/>
<br/>        “嘟——”
<br/>
<br/>        连星儿的话还未说完，明司寒便直接把电话给挂断了。
<br/>
<br/>        他让她理不清头绪，甚至不知所以然。
<br/>
<br/>        这还是她所认识的明司寒吗？
<br/>
<br/>        连星儿觉得一定是自己产生了幻觉。
<br/>
<br/>        她抚着额头，回到餐馆，将手机还给了张子晋后，跟大家打了声招呼：“你们慢慢吃，我有点不舒服，先回宿舍了。”
<br/>
<br/>        连星儿的怪异举动，虽然让顾千凝他们感到一头雾水，但是好在有郎安跟着，他们都很放心让她去了。
<br/>
<br/>        郎安默默无闻地跟在连星儿身后，看着她瑟瑟发抖的背影，以及晚风吹拂起她的发梢，于是脱下了外套，疾步上前，将外套搭在了连星儿的肩头上。
<br/>
<br/>        连星儿有些无动于衷，六神无主地继续往前走，就连红绿灯也不看，直接往马路中间走。
<br/>
<br/>        郎安见状，直接跑上前去，将连星儿打横抱起，快速冲到了马路对面，躲过了过往的车辆。
<br/>
<br/>        “混蛋！谁叫你抱我的！你放我下来！”连星儿一边推搡着郎安的胸膛，一边摆动着双脚挣扎。
<br/>
<br/>        郎安放下连星儿后，“啪”地一声，偏过头去，干净的左脸颊上瞬间出现了粉色的巴掌印。
<br/>
<br/>        连星儿收了手，给完郎安刚刚那一巴掌后，气急败坏地吼道：“即刻起，请你离我五米远！”
<br/>
<br/>        “……”郎安面无表情，毕恭毕敬地颔首。
<br/>
<br/>        连星儿气得嘴唇发抖，甩手往校门口跑去。
<br/>
<br/>        郎安觉得差不多有五米远的距离时，才一丝不苟地跟了上去。
<br/>
<br/>        连星儿回到宿舍里后，进了卧室的第一件事情就是反锁房门。
<br/>
<br/>        郎安没有过多的话语，进了隔壁房间，直接躺在了床上。
<br/>
<br/>        直到连星儿那边没动静了，他才起身去洗澡。
<br/>
<br/>        第二天，连星儿仍旧没有接到明司寒的电话。
<br/>
<br/>        第三天依旧如此。
<br/>
<br/>        连星儿被明司寒整得有些茶不思饭不想，手里拿着手机，一有空，她便一遍又一遍地拨打过去，依旧无人接听。
<br/>
<br/>        这种状态持续了一个星期后，明司寒答应她每个双休日会回国来陪她，结果，他也食言了。
<br/>
<br/>        只可惜，她不能出国，如果能出国的话……
<br/>
<br/>        连星儿想到这里时，脑子里突然有了个很糟糕的主意。
<br/>
<br/>        本来，平日里都懒得理会郎安的她，把这个糟糕的主意，打到了郎安的身上。
<br/>
<br/>        “郎安，你帮我去M国的HF大学，找明司寒，好不好？”连星儿一改常态，难得一次温柔似水地跟郎安说话。
<br/>
<br/>        郎安看了下客厅壁钟上显示的时间，答非所问：“大小姐，七点半，您晚上还有自习课要上，时间快到了。”
<br/>
<br/>        “你不是想要我的血吗？”连星儿从沙发上跳了下来，直接蹦到郎安的跟前，仰头看着郎安，接着说道，“你帮我找明司寒，问清楚他冷落我的原因，我就把我的血给你！”
<br/>
<br/>        郎安瞥向连星儿，目光飞快地从她身上扫过后，面无表情地说道：“我若是拿了你的血，傅先生会让我死无葬身之地。”
<br/>
<br/>        “那你要怎样才肯帮我这个忙？”连星儿懊恼地抓了抓刚洗过的长发，粉粉的嘴唇，不满地嘟了起来。
<br/>
<br/>        她此刻的样子，就像一个芭比公主，长发飘飘，脸颊白里透红，看上去很是可爱。
<br/>
<br/>        郎安有那么几秒钟看走了神，随即立即颔首欠身，毕恭毕敬地回答：“我不能和大小姐您分开，所以，您的这个忙，我帮不了。”
<br/>
<br/>        连星儿转过身去，拿起抱枕，直接往沙发上一躺，心情糟糕透顶地说道：“从现在开始，我要绝食。”
<br/>
<br/>        郎安微微皱了下眉头，然后走过去，在连星儿身旁坐下，云淡风轻地应道：“既然大小姐想要绝食，那么我也陪大小姐一起绝食。”
<br/>
<br/>        “我跟明司寒闹情绪，又不是跟你闹情绪，你绝什么食啊？”连星儿立马坐起身，极其不解地看着郎安。
<br/>
<br/>        “大小姐要绝食，那位明家小少爷又不知道，但我知道。大小姐，这不是明摆着要绝食给我看的吗？”郎安极其不识趣地往连星儿身上泼冷水，还把话说得格外理直气壮。
<br/>
<br/>        连星儿被郎安这话怼得哑口无言。
<br/>
<br/>        郎安的侧颜轮廓棱角分明，面庞俊俏，眼睛里却透着一丝不易近人的冷气。
<br/>
<br/>        连星儿挫败地看着郎安，因为她发现，她拿他一点办法也没有。
<br/>
<br/>        不就是被明司寒给冷暴力对待了嘛！没什么大不了的！
<br/>
<br/>        连星儿深吸了口气，缓解了自己这糟糕的心情后，一从沙发上起身，郎安也紧跟着立马从沙发上站起身来。
